स वीक्षमाणः तत्रस्थो मार्गमाणः च मैथिलीम्
अवेक्षमाणश्च महीं सर्वां तामन्ववैक्षत 		స వీక్షమాణ: తత్రస్థో - మార్గమాణశ్చ మైథిలీమ్
అవేక్షమాణశ్చ మహీం - సర్వాం తామన్వవైక్షత
స: = హనుమా; వీక్షమాణ: = చూచుచు; తత్రస్థో = అక్కడ కూర్చుండి; మార్గమాణశ్చ = నలు దిక్కులను గమనించుచు; మైథిలీమ్ = సీతామాతను కోసం; అవేక్షమాణశ్చ = పరిశీలన; మహీం సర్వాం = ఆ ప్రదేశమంతా; తామ్ = ఆమె కోసం; అన్వవైక్షత = అన్వేషించడం
హనుమంతుడు ఆ శింశుపావృక్షముపై కూర్చుండి సీతామాతను వెదకుచు, నలు దిక్కులను గమనించుచు, అచటి ప్రదేశమునంతటిని నిదానించి చూచెను – 15.1
सन्तान कलताभिश्च पादपैर् उपशोभिताम्
दिव्यगन्ध रसोपेताम् सर्वतः समलम्कृताम् 		సన్తాన కలతాభిశ్చ - పాదపైరుపశోభితామ్
దివ్యగన్ధ రసోపేతాం - సర్వతః సమలంకృతామ్
సన్తాన కలతాభిశ్చ = అశోకవని తీగలు అల్లుకునియున్న; పాదపై:= చెట్లు; ఉపశోభితామ్ = అద్భుతంగా ; దివ్య = దివ్యమగు ; గన్ధ రసోపేతాం = పరిమళమును; సర్వతః = అంతటా వెదజల్లుచు; సమలంకృతామ్ = అలంకృతమై ఉండెను;
ఆ అశోకవని తీగలు అల్లుకునియున్న కల్పవృక్షములతో నిండి ఉండెను. అవి దివ్యమగు పరిమళమును వెదజల్లుచు ఆ వనమంతటను నిండియుండెను – 15.2
ताम् स नन्दन सम्काशाम् मृगपक्षिभिरावृताम्
हर्म्य प्रासाद सम्बाधाम् कोकिलाकुल निस्स्वनाम् 		తాం స నందన సంకాశాం - మృగ పక్షిభిరావృతామ్
హర్మ్య ప్రాసాద సంబాధాం - కోకిలాకుల నిస్స్వనామ్
తాం= ఆ అశోకవనము; స నందన= ఆ దేవేంద్రుని నందనవనమును (తోట); సంకాశాం= సమానంగా; మృగ పక్షిభి: = జంతువులు మరియు పక్షులతో; ఆవృతామ్= చుట్టూతా; హర్మ్య ప్రాసాద = పెద్ద భవనములతోను ; సంబాధాం = నిండి ఉంది; కోకిలాకుల =కోకిల మధురధ్వనులు; నిస్స్వనామ్= వినబడుచుండగా;
పెక్కు మృగములతోను, పక్షులతోను, ఎక్కువగా ఉన్న భవనములతోను, కోకిల మధురధ్వనులు వినబడుచుండగా, దేవేంద్రుని నందనవనమును పోలియుండెను – 15.3

काञ्चनोत्पल पद्माभिः वापीभिः उपशोभिताम्
बह्वासन कुथोपेताम् बहुभूमि गृहायुताम् 		కా౦చనోత్పల పద్మాభి: - వాపీభిరుప శోభితామ్
బహ్వాసన కుథోపేతాం - బహుభూమి గృహాయుతామ్
కా౦చనోత్పల= బంగారు కలువలతోను; పద్మాభి:= పద్మములతోను; వాపీభి:= దిగుడు బావులతో ; ఉపశోభితామ్= శోభించుచుండెను; బహ్వ ఆసన = పెక్కు ఆసనములతో, కుథోపేతాం = మరియు తివాచీలతో ; బహు భూమి = వేలకొలది ; గృహ = భూగృహములతో; ఆయుతామ్ = నిండియుండెను
అయ్యది బంగారు కలువలతోను, పద్మములతోను కూడిన దిగుడు బావులతో శోభించుచుండెను. రత్నకంబళములతో కూడిన పెక్కు ఆసనములతో, వేలకొలది భూగృహములతో నిండియుండెను – 15.4

सर्वर्तुकुसुमै रम्यां फलवद्भिश्च पादपैः
पुष्पितानामशोकानां श्रिया सूर्योदयप्रभाम् 		సర్వర్తు కుసుమై రమ్యాం - ఫలవద్భిశ్చ పాదపైః
పుష్పితానామ్ అశోకానాం - శ్రియా సూర్యోదయ ప్రభామ్
సర్వ ఋతు= సర్వ ఋతువులయందును; కుసుమై= పూచెడి పూవులతో; రమ్యాం = రమ్యమైన; ఫలవద్భిశ్చ= పండ్లతో కూడిన; పాదపైః= వృక్షములతోను; పుష్పితానామ్ = పుష్పించిన ; అశోకానాం = అశోకవృక్షములు ; శ్రియా = కాంతితో; సూర్యోదయ = ఉదయ సూర్యునికాంతితో; ప్రభామ్=శోభించు;
అది సర్వ ఋతువులయందును పూచెడి పూవులతో, పండ్లతో కూడిన వృక్షములతోను రమ్యమైయుండెను. పుష్పించిన అశోకవృక్షములు ఉదయ సూర్యునికాంతితో శోభించుచుండెను – 15.5

प्रदीप्तामिव तत्रस्थो मारुतिस्समुदैक्षत
निष्पत्रशाखां विहगैः क्रियमाणामिवासकृत् 		ప్రదీప్తామివ తత్రస్థో - మారుతిః సముదైక్షత
నిష్పత్ర శాఖాం విహగైః - క్రియమాణామివాసకృత్
ప్రదీప్తామ్ ఇవ= ప్రకాశవంతమైన; తత్రస్థో= అక్కడ; మారుతిః= హనుమ ; సముదైక్షత= దగ్గరగా పరిశీలించడం; నిష్పత్ర = ఆకులు లేకుండా ; శాఖాం = శాఖలు ; విహగైః = విదిలించుచున్న పక్షులతో ; క్రియమాణామ్ ఇవ = లేకుండనట్లు చేయుటకో ; అసకృత్ = తరచూ;
ఆకులు రెమ్మలు లేకుండనట్లు చేయుటకో అనునట్లు విదిలించుచున్న రెక్కలు గల పక్షులతో కూడిన వృక్షములు గల ఆ వనమున హనుమ పరికించి చూచెను – 15.6

विनिष्पतद्भिः शतशश्चित्रैः पुष्पावतं सकैः
आमूल पुष्प निचितै: अशोकैः शोक नाशनैः 		వినిష్పతద్భిః శతశ: - చిత్రై: పుష్పావతం సకైః
ఆమూల పుష్ప నిచితై: - అశోకైః శోక నాశనైః
వినిష్పతద్భిః= రాలుచున్న; శతశ:= వందలకొలదిగా;చిత్రైః= వివిధవర్ణములు; పుష్పా= పుష్పములచే; వతంసకైః= అద్భుతమైన; ఆమూల = వేళ్ళ వరకు ;పుష్ప = పువ్వులు ;నిచితై: = విస్తరించాయి; అశోకైః =అశోకవృక్షముల ; శోక నాశనైః= శోకమును నశింపజేయుచున్న;
వందలకొలదిగా రాలుచున్న వివిధవర్ణములు గల పుష్పములచే వెలయు మొదళ్ళు కలిగి, శోకమును నశింపజేయుచున్న అశోకవృక్షములను చూచెను – 15.7

पुष्पभार अतिभारैश्च स्पृशद्भि: इव मेदिनीम्
कर्णिकारैः कुसुमितैः किंशुकैश्च सुपुष्पितैः 		పుష్పభార అతిభారైశ్చ - స్పృశద్భిరివ మేదినీమ్
కర్ణికారైః కుసుమితైః - కింశుకైశ్చ సుపుష్పితైః
పుష్ప భార= పుష్ప సమూహాల బరువుతో ; అతిభారైశ్చ= బరువుచే ; స్పృశద్భి:= వంగడం/తాకడం; ఇవ మేదినీమ్= ఆ నేలను; కర్ణికారైః = కర్ణికార చెట్లు (కొండ గోగులతో) ; కుసుమితైః = మొగ్గలతో ; కింశుకైశ్చ= కింశుకా చెట్లు (మోదుగు చెట్లతో);సుపుష్పితైః= పూర్తి మొగ్గలతో
పూవుల యొక్క బరువుచే మిక్కిలి వంగి నేలను త్రాకుచున్నట్లు కొండ గోగులతో పుష్పించిన మోదుగు చెట్లతో అది ప్రకాశించుచుండెను – 15.8

स देशः प्रभया तेषाम् प्रदीप्त इव सर्वतः
पुम्नागाः सप्त पर्णाश्च चम्पकोद्दालका: तथा		 స దేశః ప్రభయా తేషాం - ప్రదీప్త ఇవ సర్వతః
పున్నాగాః సప్తపర్ణాశ్చ - చమ్పకోద్దాలకా: తథా
స దేశః= ఆ ప్రాంతం; ప్రభయా తేషాం=వాటి కాంతిచే; ప్రదీప్త ఇవ= వెలుగుచున్నట్లుండెను; సర్వతః= అన్ని దిశలలో;పున్నాగాః = పున్నాగ/పొన్నచెట్లు ;సప్తపర్ణాశ్చ = ఏడాకుల అరటిచెట్లు ;చమ్పక = సంపెంగలు/చంపక చెట్లు ; ఉద్దాలకా: తథా = ఉద్దాలక/విరిగి చెట్లు;
బాగుగా పుష్పించిన పున్నాగ పొన్నచెట్లు, ఏడాకుల అరటిచెట్లు, సంపెంగలు, ఉద్దాలక/విరిగి చెట్లతో, వాటి కాంతిచే ఆ ప్రదేశమంతయు వెలుగుచున్నట్లుండెను -15.9

विवृद्ध मूला बहवः शोभन्ते स्म सुपुष्पिताः
शातकुम्भ निभाः केचित् केचिद् अग्नि शिखोपमाः 		వివృద్ధ మూలా బహవః - శోభన్తే స్మ సుపుష్పితాః
శాతకుమ్భ నిభాః కేచిత్ - కేచిత్ అగ్ని శిఖోపమాః
వివృద్ధ మూలా= లావుపాటి ముదళ్ళుగల; బహవః= చాలా; శోభన్తే స్మ= శోభించుచుండెను; సుపుష్పితాః=బాగా పుష్పించిన; శాత కుమ్భ నిభాః =బంగారు రంగులో; కేచిత్ = కొన్ని; కేచిత్= కొన్ని; అగ్ని శిఖోపమాః=అగ్ని శిఖలవలె;
లావుపాటి ముదళ్ళుగల బరిగిచెట్లు శోభించుచుండెను. అచటి వృక్షములు కొన్ని బంగారువలెను, కొన్ని అగ్ని శిఖలవలెను కానవచ్చుచుండెను – 15.10

ఈమెకు ఎట్టి రూపము, అవయవములు ఉండెనో, శ్రీరామునకూ అట్టి రూపము అవయవ సౌష్టము ఉన్నవి. నల్లని కనులు కలిగిన ఈమెయే ఆయనకి అన్ని విధముల తగియున్నది – 15.51

अस्या देव्या मनस्तस्मिंस्तस्य चास्यां प्रतिष्ठितम्
तेनेयं स च धर्मात्मा मुहूर्तमपि जीवति 		అస్యా దేవ్యా మనస్తస్మిన్ - తస్య చాస్యాం ప్రతిష్ఠితమ్
తేనేయం స చ ధర్మాత్మా - ముహూర్తమపి జీవతి
అస్యా దేవ్యా: = ఈమె యొక్క; మన: = మనస్సు; తస్మిన్ = శ్రీరామునియందును; తస్య = అతని మనస్సు; అస్యాం చ = ఈమె యందును; ప్రతిష్ఠితమ్ = గట్టిగా లగ్నమై ఉంది; తేన యం = ఆ కారణంగా; ఇయం = ఈమె; స చ ధర్మాత్మా = ధర్మాత్ముడైన శ్రీరాముని; ముహూర్తమ్ = ఒక క్షణం కూడా విడువక ; అపి జీవతి = జీవించియున్నారు
ఈమె యొక్క మనస్సు శ్రీరామునియందును, శ్రీరాముని మనస్సు ఈమె యందును బాగుగా లగ్నమై ఉన్నవి. అందువల్లనే శ్రీరాముని విడిచి ఈమె, ఈమెను విడిచి అతడు ఇంతవరకు జీవించియున్నారు – 15.52

दुष्करम् कृतवान् रामो हीनो यदनया प्रभुः
धारयत्यात्मनो देहम् न शोके नावसीदति 		దుష్కరం కృతవాన్ రామో - హీనో యదనయా ప్రభు:
ధారయత్యాత్మనో దేహం - న శోకే నావసీదతి
దుష్కరం = దుష్కరనమైన; కృతవాన్ =కార్యములు ;రామో = శ్రీరాముడు ; హీనో = వేరు /విడిచి; యదనయా = ఈమెను ; ప్రభు:= ప్రభువైన శ్రీరాముడు; ధారయత్యాత్మనో = ధరించి ఉండుట ; దేహం = ప్రాణములను ; శోకే = శోకంతో; న అవసీదతి = అతను చనిపోలేదు ;
ప్రభువైన శ్రీరాముడు ఈమెను విడిచి జీవించి ఉండుటయు, దు:ఖముచే ప్రాణములను ధరించి ఉండుటయు ప్రశంసనీయము – 15.53

दुष्करम् कुरुते रामो य इमां मत्तकाशिनीम्
सीताम् विना महाबाहु: मुहूर्तमपि जीवति 		దుష్కరం కురుతే రామో - య ఇమాం మత్తకాశినీమ్
సీతాం వినా మహాబాహు: - ముహూర్తమపి జీవతి
దుష్కరం= దుష్కరనమైన/కష్టమైన; కురుతే=కార్యములు ; రామో= శ్రీరాముడు ; య ఇమాం = ఈమె (సీతామాత); మత్తకాశినీమ్ =యవ్వని; సీతాం వినా = సీతామాత లేకుండా; మహాబాహు: =మహాబాహుడు; ముహూర్తమ్ = ఒక్క క్షణం విడిచి; అపి జీవతి = జీవించియున్నాడు ;
ఉత్తమురాలగు ఈ సీతామాతని విడిచి, మహాబాహుడగు శ్రీరాముడింత వరకు బ్రతికియుండుట మిక్కిలి ఆశ్చర్యముగనే ఉన్నది – 15.54

एवं सीतां तदा दृष्ट्वा हृष्टः पवनसम्भवः
जगाम मनसा रामं प्रशशंस च तं प्रभुम् 		ఏవం సీతాం తదా దృష్ట్వా - హృష్టః పవన సంభవః
జగామ మనసా రామం - ప్రశశంస చ తం ప్రభుమ్
ఏవం = ఆ విధంగా; సీతాం = సీతాదేవిని; తదా = అప్పుడు; దృష్ట్వా = చూసిన; హృష్టః = సంతోషించి; పవన సంభవః = హనుమా; జగామ = తలంచుకొని; మనసా = మనస్సునందే; రామం = శ్రీరాముని; ప్రశశంస = ప్రశంసించెను; తం ప్రభుమ్ = ఆ ప్రభువుని
వాయుపుత్రుడగు హనుమంతుడిట్లు సీతాదేవిని చూచినందులకు మిక్కిలి సంతోషించి, శ్రీరాముని తలంచుకొని, అతనిని తన మనస్సునందే ప్రశంసించెను – 15.55
