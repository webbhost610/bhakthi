तस्मिन्नेव ततः काले राजपुत्री त्वनिन्दिता
रूपयौवन सम्पन्नं भूषणोत्तम भूषितम् 		తస్మిన్ ఏవ తతః కాలే - రాజపుత్రీ తు అనిన్దితా
రూప యౌవన సంపన్నం - భూషణోత్తమ భూషితమ్
తతః = దాని తరువాత; తస్మిన్న్ కాలే ఏవ = ఆ సమయంలో; రాజపుత్రీ = రాజపుత్రికయు; అనిన్దితా = ఏ నింద లేకుండా; రూప యౌవన సంపన్నం = రూపయౌవన సంపన్నుడును; భూషణోత్తమ భూషితమ్ = అద్భుతమైన ఆభరణాలతో అలంకరించుకొన్నవాడును;
అంత ఆ సమయమున రాజపుత్రికయు, పరిశుద్ధస్వభావము, గల సీతామాతని చూడడానికి రూపయౌవన సంపన్నుడై, మేలైన నగలు అలంకరించుకొని వచ్చెను- 19.1

ततो दृष्ट्वैव वैदेही रावणम् राक्षसाधिपम्
प्रावेपत वरारोहा प्रवाते कदली यथा 		తతో దృష్ట్వైవ వైదేహీ - రావణం రాక్షసాధిపమ్
ప్రావేపత వరారోహా - ప్రవాతే కదళీ యథా
తత: దృష్ట్వైవ = చూసినప్పుడు; వైదేహీ = సీతాదేవి; రావణం = రావణుని; రాక్షసాధిపమ్ = రాక్షసుల రాజు; ప్రావేపత = కంపించెను; వరారోహా = అద్భుతమైన అవయవాలకు; ప్రవాతే = పెనుగాలిచే మిక్కిలి ఊగులాడు; కదళీ యథా = అరటిచెట్టువలె
రాక్షసులకు రాజు అయిన రావణుని దూరము నుండి చూడగానే, సీతాదేవి పెనుగాలిచే ఊగులాడు అరటిచెట్టువలె మిక్కిలి కంపించెను – 19.2

आच्छाद्योदरमूरुभ्यां बाहुभ्यां च पयोधरौ
उपविष्टा विशालाक्षी रुदन्ती वरवर्णिनी 		ఆచ్ఛాద్యోదరం ఊరుభ్యాం - బాహుభ్యాం చ పయోధరౌ
ఉపవిష్టా విశాలాక్షీ - రుదన్తీ వరవర్ణినీ
ఆచ్ఛాద్య: = కప్పుకొని; ఉదరం = కడుపు; ఊరుభ్యాం = తొడలతో; బాహుభ్యాం చ = చేతులతో; పయోధర = ఛాతీ (స్తనములను); ఉపవిష్టా = కూర్చుని; విశాలాక్షీ = విశాలమగు నేత్రములు కల; రుదన్తీ = ఏడువసాగెను; వరవర్ణినీ = పతివ్రతయు;
పతివ్రతయు, విశాలమగు నేత్రములు కల సీతాదేవి, రావణుని చూచి తోడలతో కడుపును, చేతులతో స్తనములను కప్పుకొని ముడుచుకొని కూర్చుండి ఏడువసాగెను – 19.3

दशग्रीवस्तु वैदेहीम् रक्षितां राक्षसीगणैः
ददर्श दीनाम् दुःखार्तां नावं सन्नामिवार्णवे 		దశగ్రీవస్తు వైదేహీం - రక్షితాం రాక్షసీ గణైః
దదర్శ దీనాం దుఃఖార్తాం - నావం సన్నామ్ ఇవార్ణవే
దశగ్రీవస్తు = రావణ; వైదేహీం = సీతాదేవిని; రక్షితాం = రక్షింపబడి; రాక్షసీ గణైః = రాక్షస స్త్రీల గుంపుచే; దదర్శ = చూచెను; దీనాం = దీనురాలు; దుఃఖార్తాం = దు:ఖపీడితయై; నావం ఇవ = నావవలె; సన్నామ్ = ప్రమాదంలో మునిగిన; అర్ణవే = సముద్రము నందు;
రాక్షస స్త్రీల గుంపుచే రక్షింపబడితు దు:ఖపీడితయై, సముద్రము నందు మునిగిన నావవలె ఉన్న సీతాదేవి రావణుడు చూచెను – 19.4

असंवृतायाम् आसीनाम् धरण्याम् संशितव्रताम्
चिनां प्रपतितां भूमौ शाखामिव वनस्पतेः 		అసంవృతాయామ్ ఆసీనాం - ధరణ్యాం సంశిత వ్రతామ్
ఛిన్నాం ప్రపతితాం భూమౌ - శాఖామివ వనస్పతేః
అసంవృతాయామ్ = అస్తరణ లేని; ఆసీనాం = కూర్చున్న; ధరణ్యాం = నేలపై; సంశిత వ్రతామ్ = పాతివ్రత్య నియమము అవలంబించిన; ఛిన్నాం = నరకుటచే విరిగి; ప్రపతితాం = పడిపోయిన; భుమౌ = భూమిపై; శాఖామ్ ఇవ =కొమ్మవలె; వనస్పతేః = చెట్టు;
పాతివ్రత్య నియమము అవలంబించిన సీతాదేవి, అస్తరణ లేని నేలపై కూర్చున్నదై నరకుటచే భూమిపైబడిన చెట్టుకొమ్మవలె కానవచ్చెను – 19. 5

मलमण्डन चित्राङ्गीम् मण्डनार्हाम् अमण्डिताम्
मृणाली पङ्कदिग्धेव विभाति न विभाति च 		మల మణ్డన చిత్రాంగీం - మణ్డనార్హామ్ అమణ్డితామ్
మృణాలీ పంక దిగ్ధేవ - విభాతి న విభాతి చ
మల మణ్డన చిత్రాంగీం = మురికి పట్టిన శరీరముతో; మణ్డనార్హామ్ = ఆభరణాలతో అలంకరించేందుకు తగిన; అమణ్డితామ్ = ఏ ఆభరణాలు లేకుండా/ధరింపక; మృణాలీ = తామర తూడువలె; పంక దిగ్ధేవ = బురదతో కూడిన; విభాతి = ప్రకాశిస్తూ; న విభాతి చ = ప్రకాశింపకున్నది
మురికి పట్టుట వలన వింతగా తోచు శరీరము కలిగి అలంకారములకు తగియున్నను, వాటిని ధరింపకయే యున్న సీతాదేవి, బురదతో కూడిన తామర తూడువలె ప్రకాశించుచు ప్రకాశింపకున్నదిగా తోచెను – 19.6

समीपम् राजसिम्हस्य रामस्य विदितात्मनः
सङ्कल्प हय सम्युक्तै: यान्तीमिव मनोरथैः 		సమీపం రాజ సింహస్య - రామస్య విదితాత్మనః
సంకల్ప హయ సంయుక్తై: - యాంతీమివ మనోరథైః
సమీపం = దగ్గరకు; రాజ సింహస్య = రాజులలో ఒక సింహం; రామస్య = శ్రీరాముని; విదితాత్మనః = వ్యక్తిత్వంలో తత్వవేత్త; సంకల్ప హయ సంయుక్తై: = గుర్రాలు వంటి ఆలోచనలతో జతకలిసి; యాంతీమ్ = పోవుచున్నట్లు; ఇవ మనోరథైః = మనస్సులోని కోరికలతో;
ఆమె సంకల్పముననెడి గుర్రములతో కూడిన మనస్సులోని కోరికలతో రాజశ్రేష్ఠుండును, తత్వవేత్త అయిన శ్రీరాముని సమీపమునకు పోవుచున్నట్లు కానవచ్చినది – 19.7

शुष्यन्तीम् रुदतीमेकां ध्यान शोक परायणाम्
दुःखस्यान्तम् अपश्यन्तीं रामां राममनुव्रताम् 		శుష్యన్తీం రుదతీమ్ ఏకాం - ధ్యాన శోక పరాయణామ్
దుఃఖస్యాన్తమ్ అపశ్యన్తీం-రామాం రామమనువ్రతామ్
శుష్యన్తీం = మిక్కిలి చిక్కిపోయి; రుదతీమ్ = ఏడ్చుచు; ఏకాం = ఏకాకియై; ధ్యాన శోక పరాయణామ్ = ధ్యానించుచుండెను; దుఃఖస్య = దు:ఖమునకు; అన్తమ్ = తుది /అంతం; అపశ్యన్తీం = చూడలేక; రామాం = శ్రీరాముని; రామమ్ = శ్రీరాముడినే ; అనువ్రతామ్ = ఆలోచనతో;
ఆమె మిక్కిలి చిక్కిపోయి, ఏడ్చుచు ఏకాకియై ఎల్లప్పుడును ఆలోచనతోను, దు:ఖముతోను కూడియున్నదై తన దు:ఖమునకు తుది కానక మనస్సులో శ్రీరాముడినే ధ్యానించుచుండెను – 19.8

वेष्टमानां तथाविष्टां पन्नगेन्द्र वधूमिव
धूप्यमानाम् ग्रहेणेव रोहिणीं धूमकेतुना 		వేష్టమానామ్ తథావిష్టాం - పన్నగేంద్ర వధూమ్ ఇవ
ధూప్యమానాం గ్రహేణేవ - రోహిణీం ధూమ కేతునా
వేష్టమానామ్ = చుట్టుకొనియున్న; తథ = మరియు; అవిష్టాం = బంధింపబడి; పన్నగేంద్ర వధూమ్ ఇవ = ఆడ సర్పమువలె; ధూప్యమానాం = మణిమంత్రాదులచే కట్టబడి; గ్రహేణేవ = బాధింపబడిన; రోహిణీం = రోహిణీ నక్షత్రమువలె; ధూమ కేతునా = కేతుగ్రహముచే
మణిమంత్రాదులచే కట్టబడి నేలపై చుట్టుకొనియున్న నాగుపామువలె మరియు కేతుగ్రహముచే బాధింపబడిన రోహిణీ నక్షత్రమువలె, ఆమె బంధింపబడి ఉండెను – 19.9

वृत्तशील कुलेजाताम् आचारवति धार्मिके
पुनः संस्कारमापन्नां जातामिव च दुष्कुले 		వృత్త శీలే కులే జాతామ్ - ఆచారవతి ధార్మికే
పున: సంస్కారమ్ ఆపన్నాం - జాతామివ చ దుష్కులే
వృత్త శీలే = మంచి ప్రవర్తన/ సదాచారము ; కులే జాతామ్ = ఉత్తమ వంశమునందు జనించి; ఆచారవతి = సాంప్రదాయ; ధార్మికే = ధార్మికమగు; పున: = మళ్ళీ; సంస్కారమ్ = వివాహం; ఆపన్నాం = చేయబడి; జాతామ్ ఇవ = జన్మించినట్లు/ చేయబడిన; దుష్కులే = చెడు కుటుంబంలో
సదాచారము కలిగి ధార్మికమగు ఉత్తమ వంశమునందు జనించి, చెడ్డ కులమున వివాహము చేయబడిన స్త్రీవలె ఆమె కానవచ్చెను – 19.10

अभूतेन अपवादेन कीर्तीम् निपतितामिव
अम्नायानाम् अयोगेन विद्यां प्रशिथिलामिव 		అభూతేన అపవాదేన - కీర్తిం నిపతితాం ఇవ
ఆమ్నాయానాం అయోగేన - విద్యాం ప్రశిధిలాం ఇవ
అభూతేన = కల్పితమైన /తప్పుడు; అపవాదేన = అపవాదుల వలన; కీర్తిం = కీర్తి; నిపతితాం ఇవ = పడిపోయింది/ చెడిన; ఆమ్నాయానాం = వేదవిద్యలు నేర్చుకోవడం; అయోగేన = ఆచరించక; విద్యాం ఇవ = వేదవిద్యలవలె ; ప్రశిధిలాం = మరుపునకు/ శిధిలమైన;

सन्नामिव महाकीर्तिं श्रद्धामिव विमानिताम्

आयतीमिव विध्वस्ताम् आज्ञां प्रतिहतामिव

भावेन रघुमुख्यस्य दशग्रीवपराभवम् 		ఆయాచమానాం దుఃఖార్తాం - ప్రా౦జలిం దేవతామివ
భావేన రఘు ముఖ్యస్య - దశగ్రీవ పరాభవమ్
ఆయాచమానాం = ప్రార్ధించుచున్నట్లుండెను; దుఃఖార్తాం =దుఃఖంతో; ప్రా౦జలిం = చేతులు జోడించి; దేవతామ్ ఇవ = దేవతలను; భావేన = ఆలోచన ద్వారా/భావముతో; రఘు ముఖ్యస్య = శ్రీరాముని మూలముగ; దశగ్రీవ పరాభవమ్ = రావణ యొక్క ఓటమి;
ఆయాసమును పొంది దు:ఖార్తయై చేతులు జోడించి సీతదేవి, శ్రీరాముని మూలముగ దశకంఠునకు పరాభవము కలుగునట్లు చేయవలెనని దేవతలను ప్రార్ధించుచున్నట్లుండెను – 19.22

समीक्षमाणां रुदतीमनिन्दितां
सुपक्ष्मताम्रायत शुक्ललोचनाम्
अनुव्रतां राममतीव मैथिलीं
प्रलोभयामास वधाय रावणः 		సమీక్షమాణాం రుదతీమ్ అనిన్దితాం
సుపక్ష్మ తామ్రాయత శుక్లలోచనామ్
అనువ్రతాం రామమతీవ మైథిలీం
ప్రలోభయామాస వధాయ రావణః
సమీక్షమాణాం = నలువైపులను దృష్టిని ప్రసరింపజేయుచు; రుదతీమ్ = రోదన చేయుచు; అనిన్దితాం = అనిందితయు; సుపక్ష్మ తామ్రాయత = అందమైన కనురెప్పలతో ఎర్రని కను కొనలతో; శుక్లలోచనామ్ = తెల్లని కన్నులు; అనువ్రతాం = మనస్సు ధ్యానించుచు; అతీవ = సదా ; రామమ్ = శ్రీరాముని; మైథిలీం = సీతామాతను; ప్రలోభయా మాస = మంచిమాటలచే /ప్రలోభపెట్టుట; వధాయ = చావుమూడిన; రావణః = రావణుడు;
నలువైపులను దృష్టిని ప్రసరింపజేయుచు రోదనము చేయుచు, అనిందితయు, అందమైన కనురెప్పలతో ఎర్రని కను కొనలతో తెల్లని కన్నులు కలదియు, సదా శ్రీరాముని యందే మనస్సు ఉంచు ఆ మిధిలేశుని పుత్రికయగు సీతామాతను, చావుమూడిన రావణుడు తన మంచిమాటలచే తనపై అభిమానము కలిగించుకొనుటకు పూనుకొనెను – 19.23
