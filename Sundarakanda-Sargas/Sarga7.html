సుందరకాండ - ఏడవ సర్గ

రావణ భవన - పుష్పక విమానముల వర్ణనము

स वेश्म जालम् बलवान् ददर्श
व्यासक्त वैदूर्य सुवर्ण जालम्
यथा महत् प्राव्ऱ्षि मेघ जालम्
विद्युत् पिनद्धम् सविहम्ग जालम् 		స వేశ్మ జాలం బలవాన్ దదర్శ
వ్యాసక్త వైడూర్య సువర్ణ జాలమ్
యథా మహత్ ప్రావృషి మేఘ జాలం
విద్యుత్ పినద్ధం సవిహంగ జాలమ్
వేశ్మ = ఇళ్ళు ; జాలం = సమూహం/ శ్రేణులను; స: బలవాన్ = మహాబలశాలియగు మారుతి; దదర్శ= చూచెను; వ్యాసక్త = జాలకములు /గవాక్షములు/ కిటికీలు; వైడూర్య = వైడూర్య ; సువర్ణ జాలమ్ = సువర్ణ నిర్మితము; మహత్ = మహా; ప్రావృషి= వర్షకాలమున; మేఘ జాలం = మేఘాల సమూహం; విద్యుత్ = విద్యుత్ ; పినద్ధం= కాంతులు; సవిహంగ = కలిసి ఎగిరే పక్షులు; జాలమ్ = సమూహం
మహాబలశాలియగు మారుతి రావణుడు నివశించు మందిరముల శ్రేణులను చూచెను. వాటి జాలకములు (గవాక్షములు) వైఢూర్య సువర్ణ నిర్మితములగుట వలన మిక్కిలి మెరయుచుండుటచే, వర్షకాలమున మెరుపులతో కూడిన మేఘమాలిక వలె ఒప్పుచుండెను. వివిధ విహంగములు అందలి సౌందర్యమును పెంపొందింప చేయుచుండెను – 7.1

निवेशनानाम् विविधाः च शालाः
प्रधान शन्ख आयुध चाप शालाः
मनो हराः च अपि पुनर् विशाला
ददर्श वेश्म अद्रिषु चन्द्र शालाः 		నివేశనానాం వివిధాశ్చ శాలాః
ప్రధాన శ౦ఖాయుధ చాప శాలాః
మనోహరాశ్చాపి పున: విశాలా
దదర్శ వేశ్మాద్రిషు చన్ద్ర శాలాః
నివేశనానాం= ఇళ్ళు; వివిధాశ్చ = వివిధ; శాలాః =శాలలు/గదులు ; ప్రధాన = ముఖ్యములైన ; శ౦ఖాయుధ = శంఖములను, ఆయుధములను; చాప= ధనస్సులను; శాలాః= గదులు/ శాలలను ; మనోహరాశ్చాపి = మనోహరములను; పున: = మరియు; విశాల = విశాలమైన; దదర్శ= చూసెను; వేశ్మాద్రిషు = కొండలవంటి ఆ మందిరములను ; చన్ద్ర శాలాః = చంద్రశాలలను
విడిదికి అనుకూలమైన వివిధశాలలను, ముఖ్యములైన శంఖములను, ఆయుధములను, ధనస్సులను ఉంచిన శాలలను, మనోహరములను, సువిశాలములను, చంద్రశాలలను, కొండలవంటి ఆ మందిరములను చూచెను – 7.2

गृहाणि नाना वसु राजितानि
देव असुरैः च अपि सुपूजितानि
सर्वैः च दोषैः परिवर्जितानि
कपिर् ददर्श स्व बल अर्जितानि 		గృహాణి నానా వసురాజితాని
దేవాసురైశ్చాపి సుపూజితాని
సర్వైశ్చ దోషైః పరివర్జితాని
కపిర్ దదర్శ స్వ బలార్జితాని
గృహాణి = గృహాలు; నానా = వివిధ ;వసురాజితాని = ధనవంతులు ద్వారా; దేవాసురైశ్చాపి = దేవతలు మరియు అసురులు; సుపూజితాని = పూజితములై; సర్వైశ్చ = అన్ని ; దోషైః పరివర్జితాని =దోష/లోపాలు రహితములైన; కపి:= హనుమ; దదర్శ= చూసెను; స్వ = స్వీయ/స్వయం (రావణుని) ;బలార్జితాని = శౌర్యమువలన ఆర్జితములై;
రావణుని శౌర్యమువలన ఆర్జితములై, దేవాసురలచే పూజితములై, దోషరహితములైన నానారత్నములచే నిండినట్టి గృహములను సీతామాత అన్వేషణ తత్పరుడైన మారుతి పరికించెను – 7.3

तानि प्रयत्न अभिसमाहितानि
मयेन साक्षाद् इव निर्मितानि
मही तले सर्व गुण उत्तराणि
ददर्श लन्का अधिपतेर् गृहाणि 		తాని ప్రయత్నాభి సమాహితాని
మయేన సాక్షాత్ ఇవ నిర్మితాని
మహీతలే సర్వ గుణోత్తరాణి
దదర్శ ల౦కాధిపతేర్ గృహాణి
తాని = అవి; ప్రయత్నాభి = ప్రయత్నపూర్వకముగ ; సమాహితాని = నిర్మించెనో; మయేన = మయుడు; సాక్షాత్ = సాక్షాత్తుగా; ఇవ నిర్మితాని= ఇవి నిర్మిచాడా? ; మహీతలే = భూమి మీద; సర్వ = సర్వం; గుణోత్తరాణి = ఉత్తమ లక్షణాలతో; దదర్శ= చూసెను; ల౦కాధిపతేర్= లంకాధిపతి (రావణుడు) ; గృహాణి = గృహములు
దానవశిల్పి అగు మయుడే ప్రయత్నపూర్వకముగ నిర్మించెనో అనదగినవి, గొప్ప ప్రయత్నమున కట్టబడినవియు, భూమండలమందలి ఎల్లవిధముల అతిశయించినవి అగు రావణుని నివాసగృహములనెల్ల మారుతి చూచెను – 7.4

ततो ददर्श उच्च्रित मेघ रूपम्
मनो हरम् कान्चन चारु रूपम्
रक्षो अधिपस्य आत्म बल अनुरूपम्
गृह उत्तमम् हि अप्रतिरूप रूपम् 		తతో దదర్శ ఉచ్ఛ్రిత మేఘ రూపం
మనోహరం కా౦చన చారు రూపమ్
రక్షోధి పస్యాత్మ బలాను రూపం
గృహోత్తమం హి అప్రతిరూప రూపమ్
తత: = అనంతరం; దదర్శ= చూసెను; ఉచ్ఛ్రిత= మహోన్నత; మేఘ రూపం = మేఘ రూపం ; మనోహరం = మనోహరం/ ఆకర్షణీయమైన; కా౦చన = బంగారం; చారు రూపమ్ =ఒక అందమైన రూపం కలిగి; రక్షోధి పస్యా = రాక్షసాధిపతి (రావణుడు) ; బలాను రూపం = బలానికి/శక్తి సరైన రూపం ; గృహోత్తమం = ఉత్తమ మందిరము; అప్రతిరూప రూపమ్ = పోలిక లేద
అనంతరము ఆ రావణునకు సంబంధించిన ఒకానొక మందిరమును చూచెను. అది మహోన్నత జలదమును/ మేఘ రూపం పోలి మనోహరమై, ఎల్ల విధముల తగినదై సాటిలేనిదై ఉండెను – 7.5

मही तले स्वर्गम् इव प्रकीर्णम्
श्रिया ज्वलन्तम् बहु रत्न कीर्णम्
नाना तरूणाम् कुसुम अवकीर्णम्
गिरेर् इव अग्रम् रजसा अवकीर्णम् 		మహీతలే స్వర్గమ్ ఇవ ప్రకీర్ణం
శ్రియా జ్వలన్తం బహురత్న కీర్ణమ్
నానా తరూణాం కుసుమావకీర్ణం
గిరే: ఇవాగ్రం రజసావకీర్ణమ్
మహీతలే = భూమి మీద; స్వర్గమ్ = స్వర్గం లాంటిది; ఇవ ప్రకీర్ణం= దిగిన; శ్రియా = కీర్తి తో; జ్వలన్తం = ప్రజ్వలించు; బహు రత్న= బహురత్న; కీర్ణమ్ = తొడుగు/ఖచితము; నానా = వివిధ ; తరూణాం= వృక్షాలతో ; కుసుమావకీర్ణం = పుష్పములతో నిండినదై; గిరే: ఇవ అగ్రం = పర్వతం యొక్క శిఖరం వంటి; రజసావకీర్ణమ్ = శోభించుచుండెను
భూమికి దిగిన స్వర్గమును పోలుచు బహురత్నఖచితము అగుటవలన కాంతితో విరాజిల్లుచు, వివిధ వృక్షాలతో, పుష్పములతో నిండినదై, పర్వతం శిఖరం వలె శోభించుచుండెను – 7.6

नारी प्रवेकैर् इव दीप्यमानम्
तडिद्भिर् अम्भोदवद् अर्च्यमानम्
हंस प्रवेकैर् इव वाह्यमानम्
श्रिया युतं खे सुकृतां विमानम् 		నారీ ప్రవేకైర్ ఇవ దీప్యమానం
తడిద్భిర్ అంబోదవద ఆర్చ్యమానమ్
హంస ప్రవేకైర్ ఇవ వాహ్యమానం
శ్రియా యుతం ఖే సుకృతాం విమానమ్
నారీ = మహిళల్లో ; ప్రవేకైర్ =అత్యుత్తమంగా; ఇవ దీప్యమానం = ప్రకాశించింది; తడిద్భిర్ = మిరుమిట్లు గొలుపు; అంబోదవద= మేఘమువలె; ఆర్చ్యమానమ్ = అర్చించడం/సేవించడం; హంస = హంసలచే ;ప్రవేకైర్= ఉత్తమమైన; ఇవ వాహ్యమానం= మోయబడుతున్నట్టు / వహింపబడునట్లు; శ్రియా యుతం = తేజస్సంపన్నమై; ఖే =ఆకాశంలో; సుకృతాం = పుణ్యాత్ములకు; విమానమ్= విమానము
మెరుపు తీగలవలె మిరుమిట్లు గొలుపు కాంతి కల ప్రధానులగు స్త్రీలచే శోభించునట్టిదియు, హంసలచే వహింపబడునట్లు కానవచ్చు, తేజస్సంపన్నమై, పుణ్యాత్ములకు మాత్రమే లభ్యమగు ఆ విమానమును ఆకాశంలో చూచెను – 7.7

यथा नग अग्रम् बहु धातु चित्रम्
यथा नभः च ग्रह चन्द्र चित्रम्
ददर्श युक्ती कृत मेघ चित्रम्
विमान रत्नम् बहु रत्न चित्रम् 		యథా నగాగ్రం బహు ధాతు చిత్రం
యథా నభశ్చ గ్రహ చన్ద్ర చిత్రమ్
దదర్శ యుక్తీ కృత మేఘ చిత్రం
విమాన రత్నం బహు రత్న చిత్రమ్
యథా నగాగ్రం =పర్వత శిఖరం వంటి; బహు ధాతు = అనేక ఖనిజాలతో/ గైరికాది ధాతువులచే ; చిత్రం = అలంకృతమగు; యథా నభశ్చ = ఆకాశం వంటి; గ్రహ చన్ద్ర = చంద్రగ్రహాదులచే; చిత్రమ్= అలంకృతమగు; దదర్శ = చూసెను; యుక్తీ కృత = అనేక రంగులతో ఉన్న; మేఘ చిత్రం =ఒక అందమైన మేఘం వంటిది; విమాన రత్నం = ఉత్తమమైన విమానం; బహురత్న = అనేక విలువైన రాళ్ళతో ; చిత్రమ్ =అలంకరించబడిన
గైరికాది ధాతువులచే రమ్యమైన పర్వతశిఖరములచే, చంద్రగ్రహాదులచే అలంకృతమగు ఆకాశమువలె, విరాజమానమగు మేఘమువలెను బహురత్న శోభాప్రకాశమానమగు ఆ విమానమును సీతామాతని అన్వేషించుచు హనుమంతుడు చూచెను – 7.8

मही कृता पर्वत राजि पूर्णा
शैलाः कृता वृक्ष वितान पूर्णाः
वृक्षाः कृता: पुष्प वितान पूर्णाः
पुष्पम् कृतम् केसर पत्र पूर्णम् 		మహీ కృతా పర్వత రాజి పూర్ణా
శైలాః కృతా వృక్ష వితాన పూర్ణాః
వృక్షాః కృతాః పుష్ప వితాన పూర్ణాః
పుష్పం కృతం కేసర పత్ర పూర్ణమ్
మహీ = భూమిపై; కృతా = చేయబడ్డాయి/నిర్మింపబడి; పర్వత రాజి పూర్ణా = పర్వత శ్రేణులతో నిండిన; శైలాః = పర్వతాలు; కృతా = చేయబడ్డాయి/నిర్మింపబడి; వృక్ష వితాన = చెట్లతో/వృక్షాలతో; పూర్ణాః = విస్తారంగా ఉండటం; వృక్షాః = చెట్లు/వృక్షాలు; కృతాః = చేయబడ్డాయి/నిర్మింపబడి; పుష్ప వితాన = నానా సుమజాల పుష్పాలతో; పూర్ణాః = విస్తారంగా ఉండటం; పుష్పం = పుష్పాలతో; కృతం= చేయబడ్డాయి/నిర్మింపబడి; కేసర పత్ర = కేసర దళాలు/ పత్రాలు; పూర్ణమ్= విస్తారంగా ఉండటం;
ఆ పుష్పకవిమానము మహాద్భుతమైనది. అందు భూమిపై చిత్రమైన పర్వత శ్రేణులు, పర్వత శ్రేణులపై వృక్ష సమూహములు, వృక్షములపై పుష్పాలు, వాటిపై కేసరవములు, పత్రములు (రేకులు మరియు తంతువులు నిండి) విరాజిల్లుచుండెను- 7.9

कृतानि वेश्मानि च पाण्डुराणि
तथा सुपुष्पा अपि पुष्करिण्यः
पुनः च पद्मानि सकेसराणि
धन्यानि चित्राणि तथा वनानि 		కృతాని వేశ్మానిచ పాణ్డురాణి
తథా సుపుష్పాపి పుష్కరాణి
పునశ్చ పద్మాని సకేసరాణి
ధన్యాని చిత్రాణి తథా వనాని
కృతా = చేయబడ్డాయి/నిర్మింపబడి; వేశ్మాని= భవనాలు; పాణ్డురాణి = తెలుపు; తథా = అదే విధంగా; సుపుష్పాపి = అందమైన పూలతో; పుష్కరాణి = ఉత్తమములైన సరస్సులు; పునశ్చ = మరియు; పద్మాని = కమలములు; సకేసరాణి = మనోజ్ఞ కేసర సమేతములు; ధన్యాని = ఉత్తమమైన; చిత్రాణి =అద్భుతమైన; తథా = మరియు; వనాని = వనము;
ఆ విమానమునందలి గదులు శుభ్రముగా ఉన్నవి. ఉత్తమములైన సరస్సులు, పుష్పసంవాసితములు, కమలములు, మనోజ్ఞ కేసర సమేతములు, వనములు అందు నిర్మింపబడియుండెను – 7.10

पुष्प आह्वयम् नाम विराजमानम्
रत्न प्रभाभिः च विवर्धमानम्
वेश्म उत्तमानाम् अपि च उच्च मानम्
महा कपिः तत्र महा विमानम् 		పుష్పాహ్వయం నామ విరాజమానం
రత్న ప్రభాభిశ్చ వివర్ధమానమ్
వేశ్మోత్తమానామ్ అపి చ ఉచ్చమానం
మహా కపి: తత్ర మహా విమానమ్
పుష్పాహ్వయం = పుష్పక విమానమని; నామ= పేరుతో; విరాజమానం= ప్రఖ్యాతి గాంచి; రత్న =రత్న; ప్రభాభిశ్చ = కాంతులచే; వివర్ధమానమ్ = పెంపొందుచు; వేశ్మోత్తమానామ్= ఉత్తమ భవనములకంటె; అపి చ = ఇది; ఉచ్చమానం = ఉత్తమమైన మహావిమానము; మహాకపి: = హనుమ; తత్ర = అక్కడ; మహావిమానం = గొప్ప విమానం;
పుష్పక విమానమని ప్రఖ్యాతి గాంచి, రత్నకాంతులచే పెంపొందుచు, ఉత్తమ భవనములకంటె సువిశాలమై ఉన్న ఆ మహావిమానమును అతడు చూచెను – 7.11

कृताः च वैदूर्यमया विहम्गा
रूप्य प्रवालैः च तथा विहम्गाः
चित्राः च नाना वसुभिर् भुजम्गा
जात्या अनुरूपाः तुरगाः शुभ अन्गाः 		కృతాశ్చ వైడూర్య మయావిహంగా
రూప్య ప్రవాళైశ్చ తథా విహంగాః
చిత్రాశ్చ నానా వసుభి: భుజంగా
జాత్యాను రూపా: తురగాః శుభా౦గా:
కృతాశ్చ = చేయబడ్డాయి/చెక్కిన; వైడూర్య = వైఢూర్యములు ; మయావిహంగా = పక్షులు; రూప్య ప్రవాళైశ్చ = వెండి మరియు పగడపు; తథా విహంగాః = ఆ పక్షులు; చిత్రాశ్చ = అద్భుతమైన ; నానా వసుభి: = వివిధ ఆభరణాల; భుజంగా = సర్పములు; జాత్యాను = ఉత్తమ జాతి; రూపా: = నిర్మితములు ; తురగాః = గుర్రములు ; శుభా౦గా: = అందమైన అవయవములతో;
అందలి పక్షులలో కొన్ని వైఢూర్యమయములు, మరికొన్ని విహంగములు వెండి చేతను, పగడముల చేతను వినిర్మితములు, సర్పములు నానా రత్నములచే నిర్మితములు, శుభ అవయవములతో ఉన్న గుర్రములు సజీవలుమలా? అని అనిపించెను – 7.12

प्रवाल जाम्बूनद पुष्प पक्षाः
सलीलम् आवर्जित जिह्म पक्षाः
कामस्य साक्षाद् इव भान्ति पक्षाः
कृता विहम्गाः सुमुखाः सुपक्षाः 		ప్రవాళ జామ్బూనద పుష్ప పక్షాః
సలీలమ్ ఆవర్జిత జిహ్మ పక్షాః
కామస్య సాక్షాత్ ఇవ భాన్తి పక్షాః
కృతా విహంగాః సుముఖాః సుపక్షాః
ప్రవాళ = పగడాలతో; జామ్బూనద = బంగారు; పుష్ప = పూలతో; పక్షాః = వారి రెక్కలపై ; సలీలమ్= సవిలాసముగా; ఆవర్జిత జిహ్మ= కొంత వంకరగా ; పక్షాః = రెక్కలతో; కామస్య = మదన దేవుని ; సాక్షాత్ = సాక్షాత్తు/ తానే ; ఇవ భాన్తి = కనిపించెను; పక్షాః = సహాయకులు వలె; కృతా = చేయబడ్డాయి; విహంగాః = పక్షులు; సుముఖాః = చక్కని ; సుపక్షాః = మంచి రెక్కలతో;
అందలి పక్షులు సుందరములగు ముక్కులు, చక్కని రెక్కలు తీర్చిదిద్దబడియుండెను. ప్రవాళ మణుల యొక్క, కనకము యొక్క పువ్వులచే వాటి పక్షములు అలంకరింపబడినవి. వాని రెక్కలు సవిలాసముగా కొంత వంకరగా వంచబడినవి. ఆ పక్షులు మదన దేవుని సేనలా? అనునట్లు కామోద్రేకమును ప్రజ్వలింప జేయుచుండెను – 7.13

नियुज्यमानाः च गजाः सुहस्ताः
सकेसराः च उत्पल पत्र हस्ताः
बभूव देवी च कृता सुहस्ता
लक्ष्मीः तथा पद्मिनि पद्म हस्ता 		నియుజ్య మానాశ్చ గజాః సుహస్తాః
సకేసరా: చ ఉత్పల పత్రహస్తాః
బభూవ దేవీ చ కృతా సుహస్తా
లక్ష్మీ స్తథా పద్మిని పద్మహస్తా
నియుజ్యమానాశ్చ= ఆరాధన /అంకితం; గజాః = ఏనుగులు ; సుహస్తాః = చక్కని తొండములు; సకేసరా: చ = శరీరం మీద కలువరేకలతో; ఉత్పల = తొండము; పత్రహస్తాః= పట్టుకొనినట్టు; బభూవ దేవీ = లక్ష్మీ దేవీ; చ కృతా = ఉంచబడింది/; సుహస్తా= సొగసైన చేతులతో; లక్ష్మీ స్తథా = లక్ష్మీ దేవీ అక్కడ; పద్మిని = సరస్సున; పద్మ హస్తా = ఆమె చేతిలో కమలములను పట్టుకొని;
అందలి సరస్సులందు ఏనుగులు చక్కని తొండములు, కేసర భూషితములగు శరీరములు కలిగి, కలువరేకులను తొండము పట్టుకొనినట్టు చిత్రింపబడియుండెను. లక్ష్మీ దేవి, కరములందు కమలములను పూని నిల్చినట్లు చిత్రింపబడియుండెను - 7.14

इतीव तद् गृहम् अभिगम्य शोभनम्
सविस्मयो नगमिव चारु शोभनम्
पुनः च तत् परम सुगन्धि सुन्दरम्
हिम अत्यये नगम् इव चारु कन्दरम् 		ఇతీవ తద్ గృహమ్ అభిగమ్య శోభనం
సవిస్మయో నగమివ చారు శోభనమ్
పునశ్చ తత్పరమ సుగన్ధి సున్దరం
హిమాత్యయే నగమ్ ఇవ చారుకన్దరమ్
ఇతీవ= ఈ పద్ధతిలో; తద్ గృహమ్ = ఆ ఇంటి; అభిగమ్య = సమీపము; శోభనం =మెరుస్తూ ఉంది; సవిస్మయో= ఆశ్చర్యపడి; నగమివ = ఒక కొండ వలె; చారు శోభనమ్ = అందంగా మెరుస్తూ; పునశ్చ= మరల; తత్పరమ = ఆ ఇల్లు; సుగన్ధి సున్దరం = గొప్ప సువాసనతో; హిమాత్యయే= వసంతకాలంలో; నగమ్ ఇవ =ఒక కొండ వలె; చారుకన్దరమ్= చక్కని గుహలతో;
ఆంజనేయుడిట్లు ఆ సుందర మందిరమును చేరి విస్మయ భరితుడై, మరల సుందర వృక్షమువలె ఒప్పారుచు, పరమ సుగంధభరితమై, సుందరమై హిమవసానమున చక్కని గుహలతో ఒప్పు కొండను పోలిన ఆ విమానము నందు సీతామాతను వెదకుచు తిరిగెను - 7.15

ततः स ताम् कपिर् अभिपत्य पूजिताम्
चरन् पुरीम् दश मुख बाहु पालिताम्
अदृश्य ताम् जनक सुताम् सुपूजिताम्
सुदुह्खिताम् पति गुण वेग निर्जिताम् 		తతః స తాం కపిర్ అభిపత్య పూజితాం
చరన్ పురీం దశముఖ బాహు పాలితామ్
అదృశ్య తాం జనక సుతాం సుపూజితాం
సుదుఃఖిత: పతి గుణ వేగ నిర్జితామ్
తతః = అనంతరం; స తాం కపిర్ = హనుమ; అభిపత్య =సమీపంలో; పూజితాం = పూజ్యము; చరన్ = తిరిగుచు ; పురీం= లంకయందు; దశముఖ =పది తలల ; బాహు = పరాక్రముడు; పాలితామ్ = పాలించుచున్న; అదృశ్య =కనపడక; తాం = జానకిమాత ; జనక సుతాం = జనక మహారాజు పుత్రిక ; సుపూజితాం = పూజ్యురాలు; సుదుఃఖిత: = దు:ఖించెను; పతి = పతి /భర్త; గుణ వేగ నిర్జితామ్ = శ్రీ రాముడు యొక్క గొప్పతనాన్ని (సద్గుణ ప్రభావతముచే జయింపబడిన సీతామాత)
అనంతరము మారుతి పూజ్యమును పది తలల రావణ పరాక్రమ సంరక్షితము అయిన ఆ లంకయందు తిరిగుచు శ్రీరామ సుగుణా కృష్ణమనస్కయు, పూజ్యురాలు అయిన జానకిమాత కానక ఆద్యంతము దు:ఖించెను – 7.16

ततः तदा बहु विध भावित आत्मनः
कृत आत्मनो जनक सुताम् सुवर्त्मनः
अपश्यतो अभवद् अतिदुह्खितम् मनः
सुचक्षुषः प्रविचरतो महात्मनः 		తత: తదా బహు విధ భావితాత్మనః
కృతాత్మనో జనక సుతాం సువర్త్మనః
అపశ్యతో అభవద్ అతిదుఃఖితం మనః
సుచక్షుషః ప్రవిచరతో మహాత్మనః
తత: = దాని తరువాత; తదా = అప్పుడు; బహు విధ= అనేక విధాలుగా ; భావితాత్మనః = ఆలోచించి; కృతాత్మనో = కృత నిశ్చయముతో ; జనక సుతాం = జనక మహారాజు పుత్రిక ; సువర్త్మనః = నీతిమార్గమును అనుసరించి; అపశ్యతో = కనపడలేదు; అభవద్ =నిండినది; అతిదుఃఖితం= అపారమగు దు:ఖముతో; మనః= మనస్సు; సుచక్షుషః = దూరదృష్టి; ప్రవిచరతో = సంచరించడం; మహాత్మనః= మహాత్మునకు (హనుమ)
పరిపరివిధముల ఆలోచించి, స్వకర్తవ్యమును నిర్ణయించుకొని, చక్కని నీతిమార్గమును అనుసరించి, దూరదృష్టి కలిగి సీతామాతను వెదుకు ఆ మహాత్మునకు ఆమె గోచరింపదయ్యెను. అపుడతని మనస్సు అపారమగు దు:ఖముతో నిండియుండెను – 7.17
