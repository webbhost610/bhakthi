अवधूय च ताम् बुद्धिम् बभूव अवस्थितः तदा
जगाम च अपराम् चिन्ताम् सीताम् प्रति महा कपिः 		అవధూయ చ తాం బుద్ధిం - బభూవ అవస్థిత: తదా
జగామ చ అపరాం చిన్తాం - సీతాం ప్రతి మహా కపిః
అవధూయ = మార్చుకోవడం (భ్రమను బాసి); తాం బుద్ధిం = ఆ ఆలోచన; బభూవ అవస్థిత:= సరైన అవగాహన కలవాడయ్యెను; తదా = అప్పుడు; జగామ = మొదలుపెట్టడం (ఆలోచన); అపరాం చిన్తాం = మరొక ఆలోచన; సీతాం ప్రతి = సీత గురించి; మహాకపి = హనుమ;
మండోదరిని సీతామాతయని భ్రమించి క్షణకాలము సంతోషించిన ఆంజనేయుడు ఆ భ్రమను బాసి సీతామాతను గురించి మరియొక ఆలోచన కలవాడయ్యెను – 11.1

न रामेण वियुक्ता सा स्वप्तुमर्हति भामिनी
न भोक्तुम् न अपि अलम्कर्तुम् न पानम् उपसेवितुम् 		న రామేణ వియుక్తా సా - స్వప్తుమర్హతి భామినీ
న భోక్తుం న అపి అలంకర్తుం - న పానమ్ ఉపసేవితుమ్
రామేణ వియుక్తా = రాముడి నుండి వేరుచేసిన; స్వప్తుమ్ = నిద్ర; సా భామినీ = ఆ సీత; న అర్హతి = ఆసక్తిలేదు; న భోక్తుం = తినదు; న అపి అలంకర్తుం = అలంకరించదు; న పానమ్ ఉపసేవితుమ్ = పానీయం త్రాగదు;
రామ వియోగమును పొందిన సీతామాత మహాసాధ్వి. నిద్రించుటకును, భుజించుటకును, అలంకరించుకొనుటకును, పానీయము త్రావుటకును ఆసక్తురాలు కాజాలదు -11. 2

नान्यं नरमुपस्थातुं सुराणामपि चेश्वरम्
न हि रामसमः कश्चिद्विद्यते त्रिदशेष्वपि
अन्येयमिति निश्चित्य पानभूमौ चचार सः 		నాన్యం నరమ్ ఉపస్థాతుం-సురాణామపి చేశ్వరమ్
న హి రామ సమః కశ్చిద్ - విద్యతే త్రిదశేష్వపి
అన్యేయమ్ ఇతి నిశ్చిత్య - పాన భూమౌ చచార సః
అన్యం నరమ్ = వేరే పురుషులు; ఉపస్థాతుం = సమీపించడం; సురాణామ్ = దేవతల ; ఈశ్వరమ్ = రాజు/ దేవేంద్రుడైనను; రామ సమః = శ్రీరామునకు సమం; కశ్చిద్ = ఎవ్వరూ ; న విద్యతే హి = సాటి లేరు ; త్రిదశేష్వపి = దేవతలందు; అన్య = వేరొక; ఇయం = యువతి ఈమె (సీతామాత కాదు); ఇతి నిశ్చిత్య = అని నిశ్చయించుకొని; పాన భూమౌ= పానశాలయందు; చచార= వెదకెను; సః = మారుతి;
పరపురుషుడు దేవేంద్రుడైనను సరే, ఆమె అతనిని సమీపింపదు. దేవతలందు శ్రీరామునకు సాటి వచ్చువాడు లేడు. కావున ఈమె సీతామాత కాదు. వేరొక యువతియై ఉండును అని నిశ్చయించుకొని మారుతి పానశాలయందు వెదకెను -11.3

क्रीडितेन अपराः क्लान्ता गीतेन च तथा पराः
नृत्तेन च अपराः क्लान्ताः पान विप्रहताः तथा 		క్రీడితేన అపరాః క్లాన్తా - గీతేన చ తథా పరాః
నృత్తేన చ అపరాః క్లాన్తాః - పాన విప్రహతా: తథా
క్రీడితేన = క్రీడలచే ; అపరాః = వేరే మహిళలను; క్లాన్తా: = డస్సిన/అలిసిన; గీతేన = పాడుటచే; తథా = అక్కడున్న; పరాః = ఎందరో యువతులు; నృత్తేన = నృత్యము చేసి; అపరాః = కొందరు మహిళలను; క్లాన్తా; = డస్సిన/అలిసిన; పాన విప్రహతా = మద్యంతో మత్తిల్లిన వారు; తథా= మరియు;
వివిధ క్రీడలచే డస్సిన వారును, పాడుటచే అలసిన ఎందరో యువతులు అచట పరుండిరి. నృత్యముచే డస్సినవారు (అలసినవారు) కొందరు , మధుపానముచే మత్తిల్లిన వారు కొందరు -11. 4

मुरजेषु मृदङ्गेषु पीठिकासु च संस्थिताः
तथाऽस्तरण मुख्येषु संविष्टाश्चापराः स्त्रियः 		మురజేషు మృద౦గేషు - పీఠికాసు చ సంస్థితాః
తథాస్తరణ ముఖ్యేషు - సంవిష్టా: చాపరాః స్త్రియ:
మురజేషు = మురజములు;మృద౦గేషు = మృదంగములు ; పీఠికాసు = పీఠలపై; సంస్థితాః = ఉన్నారు; తథా = అక్కడ; ఆస్తరణ ముఖ్యేషు = శ్రేష్ఠమైన తివాచీలు; సంవిష్టా: = ఉండిరి; చ అపరాః స్త్రియ: = కొందరు మహిళలను;
కొందరు స్త్రీలు మురజములను, మృదంగములను చిన్న చిన్న పీఠములను ఆనుకుని నిద్రించుచుండిరి. ఆ విధమున పరవశులై శ్రేష్ఠములగు ఆస్తరణములపై (తివాచీలు )చాలమంది సౌందర్యవతులుండిరి – 11.5

अङ्गनानाम् सहस्रेण भूषितेन विभूषणैः
रूपसल्लाप शीलेन युक्त गीतार्थ भाषिणा
देश कालाभि युक्तेन युक्त वाक्य अभिधायिना 		అ౦గనానాం సహస్రేణ - భూషితేన విభూషణైః
రూప సల్లాప శీలేన - యుక్త గీతార్థ భాషిణా
దేశ కాలాభి యుక్తేన - యుక్త వాక్య అభిధాయినా
అ౦గనానాం = స్త్రీలు ; సహస్రేణ= వేలకొలదిగా; భూషితేన = ఆభరణ ; విభూషణైః= భూషితులై; రూప =రూపోచితమైన; సల్లాప శీలేన = సరస సంభాషణము; యుక్త = చేయగలిగి ; గీతార్థ = గేయములు ;భాషిణా= సరైన అర్ధం; దేశ= దేశము/ స్థానం; కాలాభి= కాలము ప్రకారం; యుక్తేన= నైపుణ్యమును; యుక్త వాక్య = తగిన పదాలు ; అభిధాయినా= చూపగలిగి ఉండిరి;
ఆభరణ భూషితులై ఆ స్త్రీలు వేలకొలదిగా ఉండిరి . ఆ స్త్రీలు రూపోచితమైన సరస సంభాషణము చేయగలిగి, ఆలోచిత (తగిన పదాలు) గేయములు పాడి, అర్ధమును చెప్పగల చాతుర్యము కలిగి, దేశకాలోచితమైన నైపుణ్యమును చూపగలిగి ఉండిరి -11.6 & 7

रताभिरत संसुप्तम् ददर्श हरि यूथपः
तासाम् मध्ये महा बाहुः शुशुभे राक्षसेश्वरः 		రతాభిరత సంసుప్తం - దదర్శ హరియూథపః
తాసాం మధ్యే మహాబాహుః - శుశుభే రాక్షసేశ్వరః
రతాభిరత = రత్యాది క్రీడాసక్తుడై; సంసుప్తం = నిద్రలో మునిగిపోయాడు; దదర్శ = చూసెను; హరియూథపః = వానర యోధుడు; తాసాం= ఆ మహిళల సమూహం; మధ్యే = మధ్యలో; మహా బాహుః = మహాబాహుడగు; శుశుభే = ప్రకాశిస్తూ ; రాక్షసేశ్వరః = రావణుడు;
మహాబాహుడగు రావణుడు రత్యాది క్రీడాసక్తుడై ఆ స్త్రీజన మధ్యమున పరుండియుండగా, ఆంజనేయుడు చూచెను -11.8

गोष्ठे महति मुख्यानाम् गवाम् मध्ये यथा वृषः
स राक्षसेन्द्रः शुशुभे ताभिः परिवृतः स्वयम् 		గోష్ఠే మహతి ముఖ్యానాం - గవాం మధ్యే యథా వృషః
స రాక్షసేన్ద్రః శుశుభే - తాభిః పరివృతః స్వయమ్
గోష్ఠే = గోశాలయందు ; మహతి = విశాలమైన ; ముఖ్యానాం = ముఖ్యమైన; గవాం = గోవుల/ ఆవుల ; మధ్యే = మధ్యలో; యథా వృషః = ఆంబోతువలె; స రాక్షసేన్ద్రః = ఆ రావణుడు; శుశుభే = ప్రకాశిస్తూ ; తాభిః = ఆ స్త్రీలచే ; పరివృతః = పరివృతుడగు ; స్వయమ్ = స్వయముగా;
ఒక విశాలమైన గోశాలయందు ఉత్తమజాతి ఆవుల మధ్య ఒక ఆంబోతువలె, ఆ స్త్రీలచే పరివృతుడగు రావణుడు ఆ సమయమున ఆలమంద యందు శ్రేష్ఠములగు గోవుల నడుమ ఉన్న ఆంబోతువలె శోభించుచుండెను- 11.9

करेणुभिर् यथारण्यम् परिकीर्णो महाद्विपः
सर्वकामै: उपेताम् च पान भूमिम् महात्मनः 		కరేణుభి: యథారణ్యం - పరికీర్ణో మహాద్విపః
సర్వకామై: ఉపేతాం చ - పాన భూమిం మహాత్మనః
కరేణుభి:= ఆడ ఏనుగులచే; యథారణ్యం= మహారణ్య మధ్యన; పరికీర్ణో= చుట్టు చేరిన; మహా ద్విపః= మదగజమువలె; సర్వకామై:= సకల వాంఛలను; ఉపేతాం= తృప్తి పరచు ; పాన భూమిం= పానశాలను; మహాత్మనః= మహాత్ముడైన హనుమంతుడు
మహారణ్య మధ్యన ఆడ ఏనుగులచే సేవింపబడు మదగజమువలె అతడు ఉండెను. పిదప హనుమంతుడు సకల వాంఛలను తృప్తి పరచు పానశాలను కాంచెను -11.10

ददर्श कपि शार्दूलः तस्य रक्षः पतेर्गृहे
मृगाणाम् महिषाणाम् च वराहाणाम् च भागशः 		దదర్శ కపి శార్దూల: - తస్య రక్షః పతేర్గృహే
మృగాణాం మహిషాణాం చ - వరాహాణాం చ భాగశః
దదర్శ =చూసెను; కపి శార్దూల: = హనుమ; తస్య = ఆ ; రక్షః పతేర్ = రక్షోనాధుని (రావణుని ); గృహే = గృహములో; మృగాణాం =జింక; మహిషాణాం =దున్నపోతు; వరాహాణాం = పందుల; భాగశః = మాంసపు భాగాలు;
పిదప ఆ కపీంద్రుడు రక్షోనాధుని భవనమున జింక, దున్నపోతు, పందుల యొక్క మాంసమును విడివిడిగా చూచెను - 11.11

तत्र न्यस्तानि मांसानि पान भूमौ ददर्श सः

ఏ ప్రాణి ఏ జాతియందు జనించునో, ఆ ప్రాణిని ఆ తెగయందే అన్వేషింపవలయును, కనబడని స్త్రీని ఆడుజింకల నడుమ వెదకిన ఫలమేమి - 11.43
