सत्त्वम् आस्थाय मेधावी हनुमान् मारुत आत्मजः 		స లంబ శిఖరే లంబే లంబ తోయద సన్నిభే
సత్త్వమ్ ఆస్థాయ మేధావీ హనుమాన్ మారుతాత్మజః
లంబ శిఖరే = ఉన్నత శిఖరాలపై; లంబే = లంబ అని పిలువబడే కొండ; లంబ తోయద = మేఘమువంటి ;సన్నిభే = సమానమైన /సమం; సత్త్వమ్ = ధైర్యం; ఆస్థాయ = కూడగట్టుకొనెను; మేధావీ = మేధావి; స: హనుమాన్ = ఆ హనుమ; మారుతాత్మజః = వాయుదేవుని కుమారుడు;
వాయు కుమారుడైన హనుమ, గొప్ప బలం కలవాడు, మేధావి, మేఘమువంటి సమానమైన లంబ అని పిలువబడే పర్వతపు ఉన్నత శిఖరాలపై నిలబడి, ధైర్యం కూడగట్టుకొనెను – 3.1

निशि लन्काम् महा सत्त्वः विवेश कपि कुन्जरः
रम्य कानन तोय आढ्याम् पुरीम् रावण पालिताम् 		నిశి లంకాం మహాసత్త్వో వివేశ కపి కుఞ్జరః
రమ్య కానన తోయాఢ్యాం పురీం రావణ పాలితామ్
నిశి = రాత్రి; లంకాం= లంక; మహాసత్వా = గొప్ప శక్తి గల; వివేశ = ప్రవేశించెను; కపి కుఞ్జరః = హనుమ; రమ్య = రమ్యమైన ; కానన = అడవి /వనము; తోయాఢ్యాం = అందమైన జలశయాలు ; పురీమ్ = నగరం; రావణ పాలితామ్ = రావణునిచే పాలించబడుతున్న
మహాబలుడైన హనుమ రాత్రి సమయమున, వనములతోను, జలములతోను రమ్యమైన రావణ సంరక్షితమైన ఆ లంకను ప్రవేశించెను – 3.2

शारद अम्बु धर प्रख्यैः भवनैः उपशोभिताम्
सागर उपम निर्घोषाम् सागर अनिल सेविताम्
सुपुष्ट बल सम्पुष्टां यथैव विटपावतीम्
चारु तोरण निर्यूहाम् पाण्डुर द्वार तोरणाम् 		శారదాంబుధర పఖ్యై: భవనై: ఉపశోభితామ్
సాగరోపమ నిర్ఘోషాం సాగర అనిల సేవితామ్
సుపుష్ట బల సంఘుష్టాం యథైవ విటపావతీమ్
చారు తోరణ నిర్యూహాం పాండుర ద్వార తోరణామ్
శారదాంబుధర = శరదృతువు మేఘాలు ; పఖ్యై: = సమం; భవనై: = భవనాలు; ఉపశోభితామ్ = ప్రకాశించింది; సాగరోపమ = సాగరము పైనుండి; నిర్ఘోషాం= ఘోషించు/ధ్వనితో; సాగర = సముద్రపు; అనిల = గాలి/ వాయువు; సేవితామ్= సేవించుచుండెను; సుపుష్ట బల = బలవంతులు; సంఘుష్టాం=సైన్యాలు నిండి; యథైవ = సమానంగా; విటపావతీమ్ = విటపావతీవలె (అలకాపురి - కుబేరుని రాజ్యం); చారు తోరణ నిర్యూహాం = అందమైన తోరణాలు వద్ద ఏనుగులు ఉండెను; పాండుర= తెల్లని; ద్వార తోరణామ్ = ద్వార తోరణములు
శరత్కాలమందు మేఘమువలె స్వచ్ఛములైన సౌధములచే ఆ లంక నగరం మిక్కిలి శోభించుచుండెను. సాగరము పైనుండి వీచు వాయువు సేవించుచుండెను. బలవంతులగు రాక్షసుల గర్జనలు కలకల ధ్వనులచే ఆ పురమప్పుడు సముద్రమువలె ఘోషించుచు విటపావతివలె (అలకాపురి - కుబేరుని రాజ్యం) ఉండెను. సుందర తోరణములతో కూడిన ద్వారములు, బయటి ద్వారం వద్ద ఏనుగులు చెక్కబడి ఉండెను – 3.3 & 4

भुजग आचरिताम् गुप्ताम् शुभाम् भोगवतीम् इव
ताम् सविद्युत् घन आकीर्णाम् ज्योतिःमार्ग निषेविताम् 		భుజగ ఆచరితాం గుప్తాం శుభాం భోగవతీమ్ ఇవ
తాం స విద్యుద్ఘనా కీర్ణాం జ్యోతిర్మార్గ నిషేవితామ్
భుజగ = పాము; ఆచరితాం =కదిలే; గుప్తాం = రక్షించటం; శుభాం = పవిత్రమైన; భోగవతీమ్ ఇవ = భోగవతీ నగరమువలె; తాం = లంకా; విద్యుద్ఘనా కీర్ణాం = మెరుపులతో కూడిన మేఘము; జ్యోతిర్మార్గ = నక్షత్రమండలమువలె; నిషేవితామ్= శోభిల్లుచుండెను
ఆ లంకానగరము భోగవతీ నగరమువలె భుజగ(పాము) సమాకీర్ణమై, గుప్తమై, శుభప్రదమై ఉండెను. ఆకాశమును అంటుచు ఎత్తైన భవనములు మెరుపులతో కూడిన మేఘమండలమును, నక్షత్రమండలమువలె శోభిల్లుచుండెను – 3.5

मन्दमारुतसञ्चारां यथा इन्द्रस्य अमरावतीम्
शातकुम्भेन महता प्राकारेण अभिसम्वृताम् 		మంద మారుత సంచారాం యధా ఇంద్రస్య అమరావతీం
శాతకుంభేన మహతా ప్రాకారేణాభి సంవృతామ్
మంద మారుత = మందమారుతము మెల్లమెల్లగా; సంచారాం = సంచరించుచుండెను; యధా = సమానంగా ; ఇంద్రస్య అమరావతీం= ఇంద్రుడు యొక్క అమరావతి నగరం; శాతకుంభేన = బంగారంతో; మహతా = గొప్ప ; ప్రాకారేణాభి =కోటగోడ; సంవృతామ్ = చుట్టురా;
అందు మందమారుతము మెల్లమెల్లగా సంచరించుచుండెను (రావణుని భయముచే). కనకమయమగు (బంగారంతో ) గొప్ప కోటగోడను కలిగి ఆ పురము అమరావతిని పోలియుండెను – 3.6

किन्किणी जाल घोषाभिः पताकाभिः अलम्कृताम्
आसाद्य सहसा हृष्टः प्राकारम् अभिपेदिवान् 		కింకిణీ జాల ఘోషాభిః పతాకాభి: అలంకృతామ్
ఆసాద్య సహసా హృష్టః ప్రాకారమ్ అభిపేదివాన్
కింకిణీ = చిరుగంటలు; జాల = సమూహాల; ఘోషాభిః =శబ్దాలు; పతాకాభి: = పతాకాలు/జెండాలు; అలంకృతామ్ = అలంకరించబడిన; ఆసాద్య సహసా = త్వరితముగా చేరి; హృష్టః = సంతోషం; ప్రాకారమ్ = ప్రాకారము/గోడ ; అభిపేదివాన్ = చూసెను;
చిరుగంటల ధ్వనులు కలిగిన పతాకములు అలంకరించి ఉండెను. అట్టి నగరమును మహావేగమున చొచ్చి సంతుష్ట మనస్సు కలవాడై, ప్రాకారమును ఎక్కి లంకాపురిని చూసెను – 3.7

विस्मय आविष्ट हृदयः पुरीम् आलोक्य सर्वतः
जाम्बूनदमयैः द्वारैः वैदूर्य कृत वेदिकैः 		విస్మయావిష్ట హృదయః పురీమ్ ఆలోక్య సర్వతః
జంబూనదమయై: ద్వారై: వైడూర్య కృత వేదికైః
విస్మయావిష్ట = విస్మయ ; హృదయః = హృదయముతో ; పురీమ్= లంకా నగరం; ఆలోక్య= ఆవలోకించి (చూచి); సర్వతః= అంతటా; జంబూనదమయై: = స్వర్ణమయమై; ద్వారై: = ద్వారములు; వైడూర్య కృత = వైఢూర్యరత్నలు; వేదికైః = వేదికలు/ అరుగులు
అతడా పురము నలువైపుల చూచి ఆశ్చర్యభరితుడాయ్యెను/చకితుడాయెను. అచటి ద్వారములు స్వర్ణమయములు, అరుగులు వైఢూర్యరత్న విరాజమానములు – 3.8

मणि स्फटिक मुक्ताभिः मणि कुट्टिम भूषितैः
तप्त हाटक निर्यूहैः राजत अमल पाण्डुरैः 		మణి స్ఫటిక ముక్తాభి: మణి కుట్టిమ భూషితైః
తప్త హాటక నిర్యూహై రాజతామల పాండురై:
మణి = ముత్యాలు,వజ్రాలతో; స్ఫటిక= స్ఫటికాలతో; ముక్తాభి: = పొదిగిన; మణి = రత్నాలు, వైడూర్యములతో; కుట్టిమ= నేలపై; భూషితైః = అలంకరింపబడినది; తప్త హాటక నిర్యూహై = పుటము పెట్టిన శుద్ధ బంగారపు ఏనుగులతో అలంకరించబడ్డ; రాజతామల పాండురై: = మచ్చలేని వెండి తెలుపు
వజ్రములు, స్ఫటికములు, ముత్యములు తాపిన నేలలు ప్రకాశించుచుండెను. అరుగులు వైడూర్యములచే కట్టబడియుండెను. ముంజూరులు పుటము పెట్టిన బంగారు ముతో చేసిన ఏనుగులు. వెండి వలె స్వచ్చములైనట్టి సోపానములు – 3.9

वैदूर्य तल सोपानैः स्फाटिक अन्तर पांसुभिः
