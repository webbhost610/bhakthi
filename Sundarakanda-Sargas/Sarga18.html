రావణుడు అశోకవాటికకు వచ్చుట -- అతనిని హనుమంతుడు చూచుట

तथा विप्रेक्षमाणस्य वनम् पुष्पितपादपम्
विचिन्वतश्च वैदेहीम् किंचिच्चेषा निशाभवत् 		తథా విప్రేక్షమాణస్య - వనం పుష్పిత పాదపమ్
విచిన్వతశ్చ వైదేహీం - కించిచ్ఛేషా నిశా భవత్
తథా = ఆ విధంగా ; విప్రేక్షమాణస్య = వెదకుచు; వనం= అశోకవనమును; పుష్పిత పాదపమ్= బాగుగా పూచిన వృక్షములుగల; విచిన్వతశ్చ = అన్వేషణలో; వైదేహీం = సీతాదేవిని; కించిత్ ఏషా = కొద్దిగా మిగిలిన; నిశా = రాత్రి; అభవత్ = మిగిలింది (తెల్లవారింది);
బాగుగా పూచిన వృక్షములుగల ఆ అశోకవనమును చూచుచు, హనుమ సీతాదేవిని వెదకుచు ఆమెను చూచుసరికి, తెల్లవారు జామయ్యెను – 18.1

षडङ्गवेदविदुषाम् क्रतुप्रवरयाजिनाम्
शुश्राव ब्रह्मघोषान् स विरात्रे ब्रह्मरक्षसाम् 		షడ౦గ వేద విదుషాం - క్రతు ప్రవర యాజినామ్
శుశ్రావ బ్రహ్మఘోషాం స - విరాత్రే బ్రహ్మరక్షసామ్
షడ౦గ = ఆరు ; వేద విదుషాం= వేద విదులు (ఎరిగినవారు); క్రతు ప్రవర యాజినామ్= యజ్ఞ క్రతువులు తెలిసినవారు;శుశ్రావ = వినెను ; బ్రహ్మఘోషాం= వేద శబ్దాలు; స: = హనుమ; విరాత్రే =తెల్లవారుజామున; బ్రహ్మరక్షసామ్= బ్రహ్మరాక్షసులు;
ఆ తెల్లవారుజామున హనుమంతుడు షడంగములతో (ఆరు), వేదములు ఎరిగినవారు, యాగము ఒనరించువారైన బ్రహ్మరాక్షసుల వేద ఘోషలను వినెను – 18.2

अथ मङ्गलवादित्रै: शब्दैः श्रोत्रमनोहरैः
प्रबुध्यत महाबाहु: दशग्रीवो महाबलः 		అథ మ౦గళ వాదిత్రైః - శబ్దైః శ్రోత్ర మనోహరైః
ప్రాబుధ్యత మహాబాహు: - దశగ్రీవో మహాబలః
అథ = అప్పుడు; మ౦గళ వాదిత్రైః = మంగళవాద్య; శబ్దైః = ధ్వనులతో ; శ్రోత్ర మనోహరైః = చెవులకు వీనులకు విందుగా ; ప్రాబుధ్యత = మేలుకొలుపు; మహాబాహు:= పెద్ద బాహువులు కలవాడు; దశగ్రీవో = రావణుని; మహా బలః = మహాబలుడైన;
అప్పుడు వీనులకు విందుగానుండు మంగళవాద్య ధ్వనులతో పెద్ద బాహువులు కలవాడు, మహాబలుడైన రావణుని మేలుకొల్పపడెను – 18.3

विबुध्य तु यथाकालम् राक्षसेन्द्रः प्रतापवान्
स्रस्तमाल्याम्बरधरो वैदेहीम् अन्वचिन्तयत् 		విబుధ్య తు యథా కాలం - రాక్షసేన్ద్రః ప్రతాపవాన్
స్రస్త మాల్యామ్బర ధరో - వైదేహీమ్ అన్వచిన్తయత్
విబుధ్య = మేలుకొని; యథా కాలం = కాలానుగుణముగా; రాక్షసేన్ద్రః = రాక్షసరాజు; ప్రతాపవాన్ = ప్రతాపశాలియగు; స్రస్త మాల్యామ్బర ధరో = నిద్రయందు జారిపోయిన నిర్మాల్యం, వస్త్రమును సరిచేసుకొని ; వైదేహీమ్ = సీతాదేవిని గురించి ; అన్వచిన్తయత్ = జ్ఞాపకం
ప్రతాపశాలియగు ఆ రాక్షసరాజు కాలానుగుణముగా మేలుకొని, నిద్రయందు జారిపోయిన మాల్యమును, వస్త్రమును సరిచేసుకొని, సీతాదేవిని గురించి తలచుకొనెను – 18.4

भृशं नियुक्त: तस्याम् च मदनेन मदोत्कटः
न स तं राक्षसः कामं शशाकात्मनि गूहितुम् 		భృశం నియుక్త : తస్యాం చ - మదనేన మదోత్కటః
న స తం రాక్షసః కామం - శశాకాత్మని గూహితుమ్
భృశం = ఎక్కువ; నియుక్త : = చిక్కుకుని/బంధియుడై; తస్యాం చ = సీత విషయంలో; మదనేన = మన్మధునిచే; మదోత్కటః = మిక్కిలి ప్రేరేపితుడగు; తం రాక్షసః = ఆ రాక్షసుడు; తం కామం = ఆ కోరిక; న శశాక = సాధ్యం కాలేదు ;ఆత్మని = మనస్సుని; గూహితుమ్ = అణచివేయడం;
ఎక్కువ మదముగల ఆ రాక్షసుడు మన్మధునిచే మిక్కిలి ప్రేరేపితుడగుటవలన సీతాదేవి యెడల కామమును మస్సున అణుచుకొనలేకపొయెను – 18.5

स सर्वाभरणैर्युक्तो बिभ्रच्छ्रियम् अनुत्तमाम्
तां नगैर्बहुभिर्जुष्टाम् सर्वपुष्प फलोपगैः 		స సర్వాభరణైర్యుక్తో - బిభ్రచ్ఛ్రియమ్ అనుత్తమామ్
తాం నగై: బహుభిర్జుష్టాం - సర్వ పుష్ప ఫలోపగైః
స: = రావణ; సర్వాభరణై: = అన్ని ఆభరణాలు; యుక్త: = ధరించి; బిభ్రత్ శ్రియం అనుత్తమామ్ = లోకోత్తరమగు కాంతితో ప్రకాశించు; తాం నగై: = అశోకవనమును ప్రవేశించెను; బహుభి: జుష్టాం = పెద్ద వృక్షాలతో శోభించుచుండెను; సర్వ పుష్ప = అన్ని పువ్వులతో; ఫలోపగైః= పండ్లు
ఆ రావణుడు నగలనన్నిటిని ధరించి లోకోత్తరమగు కాంతితో ప్రకాశించుచు ఆ అశోకవనమును ప్రవేశించెను. అయ్యది సమస్తములగు పుష్పములతోను, ఫలములతోను కూడిన పెద్ద వృక్షములతో శోభించుచుండెను – 18.6

वृतां पुष्करिणीभिश्च नानापुष्पोप शोभिताम्
सदा मदैश्च विहगै: विचित्राम् परमाद्भुताम् 		వృతాం పుష్కరిణీభిశ్చ - నానా పుష్పోప శోభితామ్
సదామదైశ్చ విహగై: - విచిత్రాం పరమాద్భుతామ్
వృతాం = కలిసి; పుష్కరిణీభిశ్చ = సరస్సులతో; నానా పుష్పోప శోభితామ్ = వివిధ పుష్పాలతో ప్రకాశించు; సదా మదైశ్చ = ఎల్లప్పుడూ గర్వముతో సంచరించు; విహగై: = పక్షులతో; విచిత్రాం= ఆశ్చర్యము; పరమాద్భుతామ్ = గొప్ప అద్భుతమైనది;
అది పెక్కు కొలనులతో కూడి పుష్పములతో అందముగా ఉండెను. ఎల్లప్పుడును గర్వముతో సంచరించు పక్షులు గలిగి పవిత్రమై, చూపరులకు ఆశ్చర్యము కలిగించుచుండెను – 18.7

ईहामृगैश्च विविधैर्जुष्टां दृष्टि मनोहरैः
वीथीः संप्रेक्षमाणश्च मणिकाञ्चन तोरणाः 		ఈహా మృగైశ్చ వివిధై: - జుష్టాం దృష్టి మనోహరైః
వీథీ: సంప్రేక్షమాణశ్చ - మణి కా౦చన తోరణాః
ఈహా = తోడేళ్ళతోను ;మృగైశ్చ = జింకతోలను ;వివిధై: = పలు విధముల మృగములతోను;జుష్టాం= అలలారుచుండెను; దృష్టి మనోహరైః= అందముగా ఉన్న;వీథీ: = మార్గాలు; సంప్రేక్షమాణశ్చ = పరిశీలించడం; మణి కా౦చన తోరణాః= మణులు పొదిగిన బంగారు తోరణముల;
అందముగా ఉన్న పలు విధముల తోడేళ్ళతోను, జింకతోలను . మృగముల గుంపులతోను అలలారుచుండెను, అందముగా ఉన్న ముఖద్వారా మార్గాలు మణులు పొదిగిన బంగారు తోరణములను చూచెను – 18.8

नानामृग गणाकीर्णां फलैः प्रपतितै: वृताम्
अशोकवनकामेव प्राविशत् संतत द्रुमाम् 		నానా మృగ గణాకీర్ణాం - ఫలైః ప్రపతితై: వృతామ్
అశోక వనికామ్ ఏవ - ప్రావిశత్ సంతత ద్రుమామ్
నానా మృగ గణాకీర్ణాం= పెక్కు మృగముల గుంపులతోను; ఫలైః ప్రపతితై: = పడిపోయిన పండ్లతో; వృతామ్= నిండియుండెను; అశోక వనికామ్ ఏవ= అశోకవనమునకు; ప్రావిశత్ = ప్రవేశించెను ; సంతత ద్రుమామ్ = దట్టమైన చెట్లు;
అందముగా ఉన్న పలు విధముల మృగముల గుంపులతోను, రాలి నేలపైబడిన పండ్లతోను నిండియుండెను. అతడా వనమునందు తోరణములను దట్టమైన చెట్ల వరసలను చూచుచు ప్రవేశించెను – 18.9

अङ्गना शतमात्रं तु तं व्रजन्तम् अनुव्रजत्
महेन्द्रमिव पौलस्त्यम् देवगन्धर्व योषितः 		అ౦గనా శతమాత్రం తు - తం వ్రజన్తమ్ అనువ్రజత్
మహేన్ద్రమ్ ఇవ పౌలస్త్యం - దేవ గన్ధర్వ యోషితః
అ౦గనా = యువతులు ;శత మాత్రం = వంద మంది; వ్రజన్తమ్ = వెళుతున్నప్పుడు; అనువ్రజత్ = వెంబడి/తోడువచ్చిరి; మహేన్ద్రమ్ = ఇంద్రుడు; ఇవ పౌలస్త్యం = ఆ రావణుని; దేవ గన్ధర్వ = దేవ, గంధర్వ ; యోషితః = యువతులు;
దేవతాస్త్రీలను, గంధర్వ యువతులు దేవేంద్రుని వెంట వచ్చుచున్నట్లు అశోకవనమునకు రావణుని నూరుగురు యువతులు వెంబడించి వచ్చిరి – 18.10

दीपिकाः काञ्चनीः काश्चित् जगृहु: तत्र योषितः
वाल व्यजन हस्ताश्च तालवृन्तानि चापराः 		దీపికాః కా౦చనీః కాశ్చిత్ - జగృహు: తత్ర యోషితః
వాల వ్యజన హస్తాశ్చ - తాలవృంతాని చాపరాః
దీపికాః = దీపములు; కా౦చనీః = బంగారం; కాశ్చిత్ = కొందరు; జగృహు:= పట్టుకొని; తత్ర = అక్కడ; యోషితః = మహిళలు; వాల వ్యజన= విసనకర్రలను; హస్తాశ్చ = చేతిలో పట్టుకొని; తాలవృంతాని = చామరములను; చ అపరాః = ఇంకొందరు;
వారిలో కొందరు స్త్రీలు స్వర్ణదీపికలను పట్టుకొని ఉండిరి. కొందరు చామరములను చేకొనియుండిరి. కొందరు విసనకర్రలను చేతిలో పట్టుకొని ఉండిరి – 18.11

काञ्चनैरपि भृङ्गारै: जह्रुः सलिलमग्रतः
मण्डलाग्रान् बृसीश्चापि गृह्यान्याः पृष्ठतो ययुः 		కా౦చనైరపి భృ౦గారై: - జహ్రుః సలిలమగ్రతః
మణ్డలాగ్రాన్ బృసీంశ్చాపి - గృహ్యాన్యాః పృష్ఠతో యయుః
కా౦చనై:= బంగారం; భృ౦గారై: = బిందెలతో (పూర్ణకుంభం); జహ్రు= తీసుకెళ్లారు; సలిలమ్ = నీరు; అగ్రతః= ముందు నడచిరి; మణ్డలాగ్రాన్ బృసీంశ్చాపి = గుండ్రని ఆసనములను (దిండ్లు); గృహ్య = తీసుకుని; అన్యాః = కొందరు స్త్రీలు; పృష్ఠతో యయుః = వెనుక వెళ్ళిరి
కొందరు బంగారు బిందెలతో (పూర్ణకుంభం) జలము తెచ్చుచు అతనికి ముందు నడచిరి. కొందరు స్త్రీలు కత్తులను, ఆసనములను పట్టుకొని అతని వెనుక వెళ్ళిరి – 18.12

काचिद रत्नमयीं स्थालीं पूर्णां पानस्य भामिनी
दक्षिणा दक्षिणेनैव तदा जग्राह पाणिना 		కాచిద్ రత్నమయీం స్థాలీం - పూర్ణాం పానస్య భామినీ
దక్షిణా దక్షిణేనైవ - తదా జగ్రాహ పాణినా
కాచిద్ భామినీ = ఒక యువతి; రత్నమయీం= మణిమయ; స్థాలీం = పాత్ర; పూర్ణాం= నిండిన; పానస్య=మద్యముతో; దక్షిణా = కుడివైపున; దక్షిణే నైవ = కుడిచేతితో ; తదా జగ్రాహ పాణినా= పట్టుకొని తీసుకెళుతూ ఉండెను;
నేర్పుగల ఒక యువతి మద్యముతో నిండిన మణిమయ పాత్రమును కుడిచేతితో పట్టుకొని కుడివైపున ఉండెను – 18.13

राजहंस प्रतीकाशं चत्रं पूर्ण शशिप्रभम्
सौवर्ण दण्डमपरा गृहीत्वा पृष्ठतो ययौ 		రాజహంస ప్రతీకాశం - ఛత్రం పూర్ణ శశిప్రభమ్
సౌవర్ణ దణ్డమపరా - గృహీత్వా పృష్ఠతో యయౌ
రాజహంస ప్రతీకాశం = రాజహంసవలె; ఛత్రం = గొడుగు/ ఛత్రమును; పూర్ణ శశి ప్రభమ్= పూర్తి చంద్రుడి ప్రకాశముతో; సౌవర్ణ దణ్డమ్ = బంగారు దండముతో; అపరా = మరొక స్త్రీ; గృహీత్వా = తీసుకుని; పృష్ఠతో యయౌ = వెనుకకు వెళ్ళింది;
మరియొక యువతి, రాజహంసవలె, నిండుచంద్రుని పోలిన కాంతి కలిగి బంగారు దండముతో వున్న ఛత్రమును ధరించి అతని వెనుకనే వచ్చుచుండెను – 18.14

निद्रा मदपरीताक्ष्यो रावनस्योत्तमाः स्त्रियः
अनुजग्मुः पतिं वीरम् घनम् विद्युल्लता इव 		నిద్రా మద పరీతాక్ష్యో - రావణస్యోత్తమాః స్త్రియః
అను జగ్ముః పతిం వీరం - ఘనం విద్యుల్లతా ఇవ
నిద్రా మద పరీతాక్ష్యో = నిద్రమత్తు కళ్ళతో; రావణస్య = రావణుని ; ఉత్తమాః స్త్రియః = ఉత్తమ మహిళలు; అను జగ్ముః = అనుసరించారు; పతిం వీరం = వీరుడైన భర్త; ఘనం విద్యుల్లతా ఇవ = మేఘము వెంటవచ్చు మెరుపువలె
ఉత్తమ వనితలగు రావణుని భార్యలు నిద్రమత్తుతో, మద్యపు మత్తుతో కూడిన కన్నులు కలవారై మేఘము వెంటవచ్చు మెరుపువలె వీరుడైన భర్త వెంటవచ్చిరి – 18.15

व्याविद्ध हारकेयूराः समा मृदित वर्णकाः
समागळित केशान्ताः सस्वेद वदनास्तथा 		వ్యావిద్ధ హార కేయూరా: - సమా మృదిత వర్ణకా:
సమాగళిత కేశా౦తా: - సస్వేద వదనా: తథా
వ్యావిద్ధ = స్థానభ్రంశం/చెదరిపోయిన ; హార కేయూరా =కంఠాహారములను, భుజకీర్తులను; సమా మృదిత వర్ణకా: = విడిపోతున్న గంధం మైపూతతో ; సమాగళిత కేశా౦తా = తల వెంట్రుకల జారి వ్రేలాడుచుండగా ;సస్వేద= చెమట; వదనా = ముఖములతో; తథా= మరియు
రావణునికి ప్రీతిపాత్రములైన అతని భార్యలు నిద్రనుండి లేచి వచ్చుటచే చెదరిపోయిన కంఠాహారములను, భుజకీర్తులను కలిగి, పెటిలిన మైపూతతో కూడి, తల వెంట్రుకల కొనలు జారి వ్రేలాడుచుండగా, ముఖములు చెమటతో – 18.16

घूर्णन्त्यो मदशेषेण निद्रया च शुभाननाः
स्वेद क्लिष्टाङ्ग कुसुमाः सुमाल्या कुलमूर्धजाः 		ఘూర్ణ్య౦త్యో మదశేషేణ - నిద్రయా చ శుభాననా:
స్వేద క్లిష్టా౦గ కుసుమా: - సుమాల్యా కుల మూర్ధజా:
ఘూర్ణ్య౦త్య: = తడపడుతూ; మద శేషేణ = మధ్యపాన ప్రభావముతో; నిద్రయా చ = నిద్రచేతను; శుభాననా:= అందమైన ముఖములు కలవారై; స్వేద= వారి అవయవాలపై చెమట ద్వారా; క్లిష్టా౦గ కుసుమా= చెదరిన పూదండలతో; సుమాల్యా కుల మూర్ధజా: = అలంకరించబడిన జుట్టుతో;
మధ్యపాన ప్రభావముతోను నిద్రచేతను తడపడుతూ, అందమైన ముఖములు కలవారై, వారి అవయవాలపై చెమట ద్వారా చెదరిన చెదరిన పూదండలతో, అందంగా అలంకరించబడిన జుట్టు/కేశములు కలవారై – 18.17

प्रयान्तं नैरृतपतिं नार्यो मदिरलोचनाः
बहुमानाच्च कामाच्च प्रिया भार्यास्तम् अन्वयुः 		ప్రయాంతం నైరృతపతిం - నార్యో మదిర లోచనా:
బహుమానాచ్చ కామాచ్చ - ప్రియా భార్యాస్తమన్వయ:
ప్రయాంతం=వెంట; నైరృత పతిం = రాక్షసరాజగు (రావణుని) ;నార్య: = మహిళలు; మదిర లోచనా: = మత్తు కళ్ళతో; బహుమానాచ్చ= గౌరవముతోను; కామాచ్చ = కామముతో; ప్రియా భార్యాస్తం = ప్రియమైన భార్యలు; అన్వయ: = అనుసరించిరి;
ఆ ప్రియమైన భార్యలు మత్తు కళ్ళతో భర్త వెంట గౌరవముతోను, కామముతో రాక్షసరాజగు ఆ రావణుని వెంట అనుసరించిరి – 18.18

स च कामपराधीनः पतिस्तासां महाबलः
सीतासक्त मना मन्दो मन्दाञ्चित गतिर्बभौ 		స చ కామ పరాధీన: - పతి: తాసాం మహాబల:
సీతాసక్త మనా మందో - మదా౦చిత గతిర్బభౌ
స : = ఆ రావణుడు ; కామ పరాధీన: = కామమునకు లొంగినవాడు ; పతి: = భర్త; తాసాం = ఆ (మహిళలు); మహా బల: = బలము కలవాడు; సీతా ఆసక్త = సీతాదేవిపై ఆసక్తి కలిగి; మనా మందో = మందబుద్ధితో ; మదా౦చిత గతి: = మెల్లని అందమైన నడకతో ; బభౌ = శోభించుచుండెను;
గొప్ప బలము కలవాడును, ఆ స్త్రీలకు భర్త అయిన రావణుడు, మదన వ్యధకు లొంగి, సీతాదేవి యందు లగ్నమైన మనస్సుతో మందబుద్ధితో మెల్లని నడకతో వచ్చుచుండెను – 18.19

ततः काञ्चीनिनादं च नूपुराणाम् च निस्वनम्
शुश्राव परमस्त्रीणां स कपि: मारुतात्मजः 		తతః కా౦చీ నినాదం చ - నూపురాణాం చ నిస్వనం
శుశ్రావ పరమ స్త్రీణాం - స కపి: మారుతాత్మజః
తతః = దాని తరువాత; స కపి: = హనుమ; మారుతాత్మజః = వాయుకుమారుడు; శుశ్రావ = వినెను; కా౦చీ నినాదం చ = బంగారు అందెల చప్పుడును; నూపురాణాం చ నిస్వనం = మొలనూళ్ళ ధ్వనిని; పరమ స్త్రీణాం = ఉత్తమ వనితల
వాయుపుత్రుడగు హనుమంతుడు ఆ ఉత్తమ వనితల బంగారు మొలనూళ్ళ ధ్వనిని అందెల చప్పుడును వినెను – 18.20

तं चाप्रतिम कर्माणम् अचिन्त्य बलपौरुषम्
द्वारदेशम् अनुप्राप्तं ददर्श हनुमान् कपिः 		తం చ అప్రతిమ కర్మాణమ్ - అచిన్త్య బల పౌరుషమ్
ద్వార దేశమ్ అనుప్రాప్తం - దదర్శ హనుమాన్ కపిః
అప్రతిమ కర్మాణమ్ = సాటిలేని గొప్ప పనులు చేయువాడు; అచిన్త్య బల పౌరుషమ్ = ఊహింపరాని బల పౌరుషములు కలవాడు; ద్వార దేశమ్ = వనద్వార ప్రదేశమునందు; అనుప్రాప్తం = చేరిన (రావణుని); దదర్శ = చూచెను; హనుమాన్ కపిః = హనుమ;
సాటిలేని గొప్ప పనులు చేయువాడును, ఊహింపరాని బల పౌరుషములు కలవాడు అయిన రావణుని, వానరోత్తమడగు హనుమ, వనద్వార ప్రదేశమునందే చూచెను – 18.21

दीपिकाभि: अनेकाभिः समन्तात् अवभासितम्
