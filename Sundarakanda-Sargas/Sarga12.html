లతాగృహామ: చిత్రగృహాన్ నిశాగృహాన్

తస్య = ఆ; మధ్యే = మధ్యలో; భవనస్య = భవనం; స: వానరో = హనుమ; లతాగృహాం = లతాగృహాం; చిత్ర గృహాన్ = చిత్ర గృహాలు; నిశా గృహాన్ = రాత్రియందు పరుండు గృహము; జాగామా = వెళ్ళారు; సీతాం ప్రతి = సీతామాత కోసం; దర్శనోత్సుకో = దర్శన ఆసక్తి; తాం = ఆమెని; న చ ఏవ పశ్యతి = కానీ చూడలేదు; చారు దర్శనామ్= అందంగా కనిపించే;
సీతామహాసాధ్విని సందర్శింపవలయునని మనస్సు తహతహలాడుచుండగా ఆంజనేయుడు ఆ అంత:పుర భవన మధ్యమందలి లతా గృహములను, చిత్ర మందిరములను, రాత్రియందు పరుండు గృహములను చక్కగా పరిశీలించెను. కాని ఆమెను చూడలేకపోయెను -12. 1

శ్రీరామప్రియ అయిన జానకిని చూడలేక అతడిట్లు తలంచెను, ఎంత వెదకినను ఆ తల్లి కానరాకున్నది. ఆమె మరణించెనేమో -12.2

सा राक्षसानाम् प्रवरेण बाला

हता भवेद् आर्यपथे परेस्थिता 		సా రాక్షసానాం ప్రవరేణ బాలా
స్వశీల సంరక్షణ తత్పరా సతీ
అనేన నూనం ప్రతిదుష్ట కర్మణా

సా = ఆ; రాక్షసానాం ప్రవరేణ = రాక్షసరాజుచే; బాలా = సీత ; స్వశీల = తన పాతివ్రత్యమును; సంరక్షణ = సంరక్షించుటయందు ; తత్పరా = పట్టుదల కల ; సతీ = మహాసాధ్వి ; అనేన = ఇది; నూనం = ఖచ్చితంగా; ప్రతిదుష్ట కర్మణా = దుష్టాత్ముడగు; హతా భవేత్ = హత్య చేసివుండాలి; ఆర్యపథే = సన్మార్గమును; పరే స్థితా = నిలబడి;
తన పాతివ్రత్యమును సంరక్షించుటయందు పట్టుదల కల ఆ మహాసాధ్వి సన్మార్గమును వీడనందున, దుష్టాత్ముడగు రాక్షసరాజుచే ఖచ్చితంగా చంపబడెను కాబోలును -12.3

विरूप रूपा विकृता विवर्चसो

भयाद्विनष्टा जनकेश्वरात्मजा 		విరూప రూపా వికృతా వివర్చసో
మహాననా దీర్ఘ విరూప దర్శనాః
సమీక్ష్య సా రాక్షస రాజయోషితో
భయాద్వినష్టా జనకేశ్వరాత్మజా
విరూప రూపా = వక్రమైన రూపముతో; వికృతా = వికృత రూపులు/భయంకరమైన వారు; వివర్చసో = కాంతి లేకుండా; మహాననా = పెద్ద ముఖాలతో; దీర్ఘ విరూప దర్శనాః = పొడవైన వికార రూపము; సమీక్ష్య = చూసిన; సా రాక్షస రాజయోషితో = రావణ భార్యలు; భయాత్ = భయపడి; వినష్టా = మరణించెనేమో; జనకేశ్వరాత్మజా = జనకరాజు కుమార్తె;
విషమ స్వరూపము కలవారును, సౌందర్యము ఆవంతయులేని ఆకారము కలవారును, నిస్తేజస్కలు, పెద్ద ముఖము కలవారును, పొడవైన వికార రూపముగల ఆ రాక్షస స్త్రీలను చూచి జానకిమాత భయపడి మృతినొందెనేమో -12. 4

सीताम् अदृष्ट्वा हि अनवाप्य पौरुषम्

విహృత్య కాలం సహ వానరైశ్చిరమ్
న మేస్తి సుగ్రీవ సమీపగా గతిః
సుతీక్ష్ణ దండో బలవాంశ్చ వానరః
సీతామ్ = సీతామాత; అదృష్ట్వా = చూడకుండా; అనవాప్య = సాధించకుండా; పౌరుషం = గొప్ప పురుష ప్రయత్నము; విహృత్య = వదలి తిరుగుతూ; కాలం = దీర్ఘకాలమును; సహ వానరై శ్చిరమ్ = సహ వానరలతో గడపి; న మే అస్తి = నాకు లేదు; సుగ్రీవి సామీపగా = సుగ్రీపుడి సమీపంకి; గతిః = దారి; సుతీక్ష్ణ దండో = కఠినముగా శిక్షించువాడు ; బలవాంశ్చ = బలమైన; వానరః= వానరుడు
సుగ్రీపుడు బలవంతుడు, కఠినముగా శిక్షించువాడు. సీతామాత జాడ ఎరుగక వానరులను వదలి దీర్ఘకాలమును గడపి, గొప్ప పురుష ప్రయత్నము సాధించకుండా నేనతని కడకు ఎటుల పోగలను? దారి ఏది ? -12.5

दृष्टम् अन्तःपुरम् सर्वम् दृष्ट्वा रावण योषितः
न सीता दृश्यते साध्वी वृथा जातो मम श्रमः 		దృష్టమ్ అన్తఃపురం సర్వం - దృష్ట్వా రావణ యోషితః
న సీతా దృశ్యతే సాధ్వీ - వృథా జాతో మమ శ్రమః
అన్తఃపురం = అంత:పురము; దృష్టమ్ = చూచితిని; సర్వం = అంతా; రావణ యోషితః = రావణ మహిళలు; దృష్ట్వా = చూచితిని; సాధ్వీ = మాహాసాధ్వి; సీతా = సీతామాత; న దృశ్యతే = కానరాకున్నది; మమ శ్రమః = నా ప్రయత్నం; జాతో వృథా = అంతయు వ్యర్థమైనది
రావణ అంత:పురమెల్ల అన్వేషించితిని. రావణుని స్త్రీలను చూచితిని. కాని మాహాసాధ్వి సీతామాత మాత్రము కానరాకున్నది. నా శ్రమ అంతయు వ్యర్ధమయ్యెను -12. 6

ध्रुवम् प्रायम् उपेष्यन्ति कालस्य व्यतिवर्तने 		అదృష్ట్వా కిం ప్రవక్ష్యామి - తామహం జనకాత్మజామ్
ధ్రువం ప్రాయమ్ ఉపైష్యన్తి - కాలస్య వ్యతివర్తనే

సీతామాతను చూడలేదని నేను వారితో ఎలా చెప్పగలను? చాలా కాలము గడిచి పోవుటవలన వారు బహుశ: పస్తుపడి మృత్యుముఖమును పొందియుందురేమో? -12.8

సముద్రమును లంఘించి మరిలివచ్చిన నన్ను గాంచి, నా చుట్టుజేరి వృద్ధుడగు జాంబవంతుడును, అంగదుడును ఇతర వానరులు నన్ను ఏమందురో కదా ? -12.9

अनिर्वेदः श्रियो मूलम् अनिर्वेदः परम् सुखम्
अनिर्वेदो हि सततम् सर्वार्थेषु प्रवर्तकः 		అనిర్వేదః శ్రియో మూలమ్ - అనిర్వేదః పరం సుఖమ్
అనిర్వేదో హి సతతం - సర్వార్థేషు ప్రవర్తక:
అనిర్వేదః = దు:ఖము/నిరాశ లేకపోవడం; శ్రియో మూలమ్ = అభివృద్ధి/ఐశ్వర్యము; మూలమ్ = మూలకారణము; అనిర్వేదః = దు:ఖము/నిరాశ లేకపోవడం; పరం సుఖమ్ = గొప్ప సౌకర్యం; అనిర్వేదో హి = నిర్వేదము లేకుండుటయే; సతతం = ఎల్లప్పుడూ; సర్వార్థేషు = సకల వాంఛలను/కార్యములను; ప్రవర్తక:=నెరవేరునట్లు చేయును ;
దు:ఖమావంతయు లేకుండుటయే ఐశ్వర్యమునకు మూలకారణము. అదియే సౌఖ్యహేతువు. నిర్వేదము లేకుండుటయే సకల వాంఛలను, కార్యములను నెరవేరునట్లు చేయును -12.10

करोति सफलम् जन्तोः कर्म यत्तत् करोति सः
तस्माद् अनिर्वेद कृतम् यत्नम् चेष्टेऽहमुत्तमम् 		కరోతి సఫలం జంతో: - కర్మ యత్తత్ కరోతి స:
తస్మాత్ అనిర్వేద కృతం - యత్నం చేష్టే అహముత్తమం
కరోతి= చేయునో; సఫలం = సఫలం; జంతో: = ప్రాణి /మనిషి; కర్మ = చర్య/పని; యత్తత్ = ఆ; కరోతి స:= చేయును/ఒనర్చును; తస్మాత్ = ఆ కారణంగా; అనిర్వేద = దు:ఖమును వీడి; చేష్టే కృతం యత్నం = ప్రయత్నిస్తాను; అహం = నేను; ఉత్తమం = మంచి/ శక్తి కొలది;
ప్రాణి ఏ పనిని చేయునో, ఆ పనిని అనిర్వేదమే సఫల మొనర్చును. అందువలన నేను ఇపుడు దు:ఖమును వీడి శక్తి కొలది ప్రయత్నమొనర్చి సీతామాతను వెదకెదను -12.11

अदृष्टामः च विचेष्यामि देशान् रावण पालितान्
आपान शाला विचिताः तथा पुष्प गृहाणि च 		అదృష్టాంశ్చ విచేష్యామి - దేశాన్ రావణ పాలితాన్
ఆపాన శాలా విచితా: - తథా పుష్ప గృహాణి చ
అదృష్టాంశ్చ= ఇంకా చూడనివి; విచేష్యామి= వెదకెదను ; దేశాన్= దేశములన్నింటినీ; రావణ పాలితాన్= రావణ పాలిత; ఆపానశాలా=పానశాలలను; విచితా=వెదకడం;తథా=అదేవిధంగా;పుష్పగృహాణి = పుష్ప మందిరములను
రావణ సంరక్షిత దేశములన్నింటినీ మరల వెదకెదను. నేనింత వరకు చూడని ప్రదేశములను అన్వేషించెదను. ఇంతవరకు పానశాలలను, పుష్ప మందిరములను చూచితిని గదా -12.12

चित्रशालाश्च विचिता भूयः क्रीडा गृहाणि च
निष्कुटान्तर रथ्याश्च विमानानि च सर्वशः 		చిత్రశాలాశ్చ విచితా భూయః - క్రీడా గృహాణి చ
నిష్కుటాన్తర రథ్యాశ్చ - విమానాని చ సర్వశః
చిత్ర శాలాశ్చ= చిత్రశాలను; విచితా= వెదకడం; భూయః= మళ్ళీ; క్రీడా గృహాణి చ= క్రీడామందిరములను;నిష్కుటా న్తర = తోటలు మధ్య మార్గాలు; రథ్యాశ్చ = రాజమార్గములు ; విమానాని = విమానమంతటా ;సర్వశః = అన్నిటినీ;
చిత్రశాలలను క్రీడామందిరములను లెస్సగా వెదుకబడినవి కదా. గృహారామములు, రాజమార్గములు లెస్సగా పరికింపబడినవి -12.13

इति संचिन्त्य भूयोऽपि विचेतुम् उपचक्रमे
भूमी भूमीगृहांश्चैत्य गृहान् गृहातिगृहकानपि 		ఇతి సంచిన్త్య భూయోపి - విచేతుమ్ ఉపచక్రమే
భూమీ గృహాంశ్చైత్య గృహాన్ - గృహాతి గృహకానపి
ఇతి = ఇట్లు ; సంచిన్త్య = ఆలోచిస్తూ; భూయ: అపి = మళ్ళీ; విచేతుమ్ = అన్వేషణ; ఉపచక్రమే = ప్రారంభించెను; భూమీ గృహాం = భూగర్భములు; చైత్య గృహాన్ = చైత్య గృహములను ; గృహాతి గృహకాన్= విడివిడిగా నున్న మందిరములను; అపి = మరియు
ఇట్లు ఆలోచిస్తూ ఆ మహానుభవుడు మరల అన్వేషణ కార్యమును ప్రారంభించెను. భూగర్భమందలి రహస్య గృహములను, చైత్య గృహములను విడివిడిగా నున్న మందిరములను చొచ్చి వెదుకసాగెను -12.14

उत्पतन् निष्पतंश्चापि तिष्ठन् गच्चन् पुनः पुनः
अपावृण्वंश्च द्वाराणि कवाटानि अघाटयन् 		ఉత్పతన్ నిపతంశ్చాపి - తిష్ఠన్ గచ్ఛన్ పునః పునః
అపావృణ్వంశ్చ ద్వారాణి - కవాటాని అఘాటయన్
ఉత్పతన్= కుప్పించి ఎగురుచు; నిపతంశ్చాపి= క్రిందికి దూకుచు; తిష్ఠన్= నిలబడి; గచ్ఛన్= పోవుచు ; పునః పునః= మళ్ళీ మళ్ళీ; అపావృణ్వంశ్చ= తెరుచుచు; ద్వారాణి = వాకిళ్ళు/ ద్వారాలు ; కవాటాని = తలుపులు; అఘాటయన్ = మూయుచు;
ఉన్నత స్ధలములకు కుప్పించి ఎగురుచు, అటునుండి క్రిందికి దూకుచు, అచ్చటచ్చట నిలిచి పరికించుచు, మరల పోవుచు, వాకిళ్ళు తెరుచుచు, కవాటములను(తలుపులు) మూయుచు వెదకసాగెను -12.15

प्रविशन् निष्पतंश्चापि प्रपतन्न् उत्पतन्नपि
सर्वम् अपि अवकाशम् स विचचार महा कपिः 		ప్రవిశన్ నిష్పతంశ్చాపి - ప్రపతన్ ఉత్పతన్నపి
సర్వం అపి అవకాశం - స విచచార మహా కపిః
ప్రవిశన్= చొరబడడం/వెళ్ళడం; నిష్పతంశ్చాపి= బయటికి వచ్చుచు; ప్రపతన్= క్రిందికి దిగుచు;ఉత్పతన్నపి= పైకి ఎక్కడం; సర్వం అపి అవకాశం= అన్ని అవకాశాలు; విచచార = వెదకెను ; స: మహా కపిః = హనుమ
ఆ మహాకపీంద్రుడు ఇండ్లలోనికి జొచ్చుచు, బయటికి వచ్చుచు, పైకి ఎక్కుచు, క్రిందికి దిగుచు, ఇచ్చట అచ్చట అనక అవకాశమున్న చోట్లనెల్ల వెదకెను -12.16

चतुरङ्गुल मात्रोऽपि नावकाशः स विद्यते
रावणान्तःपुरे तस्मिन् यम् कपिर् न जगाम सः 		చతుర౦గుళ మాత్రోపి - నావకాశః స విద్యతే
రావణాన్తఃపురే తస్మిన్ - యం కపిర్ న జగామ సః
చతుర౦గుళ మాత్రోపి = నాలుగు అంగుళాల స్ధలం ; న అవకాశః = అవకాశం లేని చోటు; విద్యతే = లేదు; రావణాన్తఃపురే = రావణ అంత: పురములో; తస్మిన్ = ఆ ; సః కపిర్ = హనుమ; న జగామ = వెళ్ళని చోటు ;
ఆ రావణుని భవనము నందు ఆ మహాత్ముడు చోరనిచోటు బెత్తెడైననూ (నాలుగు అంగుళాల స్ధలం) లేకుండెను – 12.17

प्राकारान्तर रथ्याश्च वेदिकः चैत्य संश्रयाः
दीर्घिकाः पुष्करिण्यश्च सर्वम् तेनावलोकितम् 		ప్రాకారాన్తర రథ్యాశ్చ - వేదిక: చైత్య సంశ్రయాః
దీర్ఘికా: పుష్కరిణ్యశ్చ - సర్వం తేనావలోకితమ్
ప్రాకారాన్తర = కోట గోడల మధ్యలో ; రథ్యాశ్చ = వీధులు; వేదిక = వేదికలు; చైత్య సంశ్రయాః = నాలుగు వీధులు కలిసే; దీర్ఘికా: = దిగుడు బావులను; పుష్కరిణ్యశ్చ = పుష్కరిణిలు /సరస్సులు; సర్వం = సర్వము; తేనా = అతని ద్వారా ;అవలోకితమ్ = వెదకెను / చూసెను
కోటయందలి దారులను, తిన్నెలను, చైత్యశాలలను, దిగుడు బావులను సరస్సులను సర్వమును వెదకెను – 12.18

राक्षस्यो विविधाकारा विरूपा विकृताः तथा
दृष्टा हनूमता तत्र न तु सा जनकात्मजा 		రాక్షస్యో వివిధాకారా - విరూపా వికృతా: తథా
దృష్టా హనూమతా తత్ర - న తు సా జనకాత్మజా
రాక్షస్యో = రాక్షస మహిళలు; వివిధాకారా = వివిధ రకాల రూపాల్లో; విరూపా = వంకర రూపాలతో; తథా = ఉన్నారు; వికృతా = భయంకరమైన రూపాలు; దృష్టా = కానవచ్చిరి; హనూమతా = హనుమకు; తత్ర = అక్కడ; న తు = కానీ; సా జనకాత్మజా = ఆ సీతామాత
ఆ మహానుభావుడు ఆయాచోట్ల వివిధ ఆకారములు కలిగి, వికార స్వరూపము కలిగిన రాక్షసస్త్రీలను చాలామందిని చేసెను. కానీ సీతామాత మాత్రము గోచరించలేదు – 12.19

रूपेण अप्रतिमा लोके वरा विद्याधर स्त्रियः
दृटा हनूमता तत्र न तु राघव नन्दिनी 		రూపేణ అప్రతిమా లోకే - వరా విద్యాధర స్త్రియః
దృష్టా హనూమతా తత్ర - న తు రాఘవ నన్దినీ
రూపేణ అప్రతిమా = సాటిలేని సౌందర్యము ; లోకే = లోకములో; వరా: = ఉత్తమ; విద్యాధర స్త్రియః = విద్యాధర స్త్రీలు; దృష్టా = కానవచ్చిరి; హనూమతా = హనుమకు; తత్ర = అక్కడ; న తు = కానీ ; రాఘవ నన్దినీ = ఆ సీతామాత
లోకములో సాటిలేని సౌందర్యము కల విద్యాధర స్త్రీలు మారుతికి అచట కనపడిరి. కాని సీతామాత మాత్రము కానదయ్యెను -12.20

नागकन्या वरारोहाः पूर्णचन्द्र निभाननाः
दृष्टा हनूमता तत्र न तु सीता सुमध्यमा 		నాగకన్యా వరారోహాః - పూర్ణచన్ద్ర నిభాననాః
దృష్టా హనూమతా తత్ర - న తు సీతా సుమధ్యమా
నాగకన్యా = నాగ మహిళలు; వరారోహాః = మంచి రూపవైభవము కలవారు; పూర్ణ చన్ద్ర నిభాననాః = పూర్ణచంద్ర ముఖుతో; దృష్టా = కానవచ్చిరి; హనూమతా = హనుమకు; తత్ర = అక్కడ; న తు = కానీ ; సీతా = ఆ సీతామాత; సుమధ్యమా = తను మధ్యయగు
పూర్ణచంద్ర ముఖబింబము కలవారు, మంచి రూపవైభవము కలవారు అయిన నాగ కన్యలు హనుమకు కనపడిరి.. కాని తను మధ్యయగు జానికిమాత జాడ పొడసూపదయ్యెను -12.21

प्रमथ्य राक्षसेन्द्रेण देवकन्या बलाद् धृताः
दृष्टा हनूमता तत्र न सा जनक नन्दिनी 		ప్రమథ్య రాక్షసేన్ద్రేణ - దేవకన్యా బలాద్ధృతా:
దృష్టా హనూమతా తత్ర - న సా జనకనన్దినీ
ప్రమథ్య = ఓడించి; రాక్షసేన్ద్రేణ = రావణ ద్వారా; దేవ కన్యా = దేవ కన్యాలు;బలాద్ధృతాః = బలవంతంగా/ దర్దించి తెచ్చిన ; దృష్టా = కానవచ్చిరి; హనూమతా = హనుమకు; తత్ర = అక్కడ; న తు = కానీ ; సా జనకనన్దినీ = ఆ సీతామాత;
రావణుడు దేవతలను దర్దించి బలాత్కారముగా తెచ్చిన దేవాంగనలు పెక్కుమంది కానవచ్చిరి. కాని జానకిమాత గోచరింపదయ్యెను -12.22

सोऽपश्यंस्तां महाबाहुः पश्यंश्चान्या वरस्त्रियः
विषसाद मुहुर्धीमान् हनुमान् मारुतात्मजः 		సోపశ్యంస్తాం మహాబాహుః - పశ్యంశ్చాన్యా వర స్త్రియః
విషసాద ముహుర్ధీమాన్ - హనుమాన్ మారుతాత్మజః
సో అపశ్యం = కనుగొనలేక ; తాం = ఆ సీత; మహాబాహుః = మహాబాహుడు; పశ్యంమ: = చూసిన; అన్యా = ఇతర; వర స్త్రియః = ఉత్తమ మహిళలు; విషసాద = విషాదమున; ముహు: = మరల ; ధీమాన్ = ధీమంతుడు; హనుమాన్ = హనుమ; మారుతాత్మజః = వాయు కుమారుడు;
మహాబాహుడు, ధీమంతుడు అయిన ఆ మారుతి చాలామంది స్త్రీలను చూచియు, సితామాతను మాత్రము కనుగొనలేక, మరల విషాద నిమగ్నుడాయెను -12.23

उद्योगम् वानरेन्द्राणां प्लवनम् सागरस्य च
व्यर्थम् वीक्ष्यानिलसुतश्चिन्ताम् पुन: उपागमत् 		ఉద్యోగం వానరేన్ద్రాణం - ప్లవనం సాగరస్య చ
వ్యర్థం వీక్ష్యానిల సుత: - చింతాం పున: ఉపాగమత్
ఉద్యోగం = ప్రయత్నం; వానరేన్ద్రాణం = వానరేంద్రుడు; ప్లవనం = దాటుట; సాగరస్య చ = మహాసముద్రం; వ్యర్థం = వృధా అవుతున్నట్లు; వీక్ష్య = చూసి; అనిల సుత: = (అనిలసుతుడు)హనుమ ; చింతాం = చింత/ విచారమును; పునర్ = మళ్ళీ; ఉపాగమత్ = పొందెను
వానరవీరులు చేసిన ప్రయత్నమును, తానొనర్చిన సాగర తరణము వ్యర్ధములయ్యెనని భావించి, ఆంజనేయుడు మరల విచారమును పొందెను - 12.24

अवतीर्य विमानाच्च हनुमान् मारुतात्मजः
चिन्ताम् उपजगाम अथ शोकोपहत चेतनः 		అవతీర్య విమానాచ్చ - హనూమాన్ మారుతాత్మజః
చిన్తామ్ ఉపజగామాథ - శోకోపహత చేతనః
అవతీర్య = కిందకి దిగి; విమానాచ్చ = పుష్పక విమానము నుండి; హనుమాన్ = హనుమ; మారుతాత్మజః = వాయు కుమారుడు; చిన్తామ్ = చింత; ఉపజగామ = అనుభవించాడు (పొందాడు ); అథ = అప్పుడు; శోకోపహత = శోకంతో ; చేతనః = చేతనుడు (అయ్యాడు)
వాయుసుడైన హనుమంతుడు విమానము నుండి క్రిందకి దిగి, మిక్కిలి కలవరపాటుకులోనై దీర్ఘ ఆలోచనలో శోకపరవశ చిత్తుడై చింతింపసాగెను -12.25
