प्रशस्यतु प्रशस्तव्याम् सीताम् ताम् हरिपुम्गवः

గుణాభిరామం రామం చ - పున: చింతా పరోభవత్
ప్రశస్యతు = ప్రశంసించడం; ప్రశస్తవ్యాం = ప్రశంసింపదగిన; సీతాం తాం = ఆ సీతాదేవిని; హరి పుంగవః = హనుమ; గుణాభిరామం = సుగుణాభిరాముడగు; రామం చ = శ్రీరాముని కూడా; పున: = మళ్ళీ; చింతాపర అభవత్ = ఆలోచనలో మునిగి పోయెను;
వానర శ్రేష్ఠుడగు హనుమంతుడిట్లు ప్రశంసింపదగిన సీతాదేవిని సుగుణాభిరాముడగు శ్రీరాముని ప్రశంసించి తిరిగి ఆలోచింపసాగెను – 16.1

स मुहूर्तम् इव ध्यात्वा बाष्प पर्याकुलेक्षणः
सीताम् आश्रित्य तेजस्वी हनुमान् विललाप ह 		స ముహూర్తమివ ధ్యాత్వా - బాష్ప పర్యాకులేక్షణః
సీతామాశ్రిత్య తేజస్వీ - హనుమాన్ విలలాప హ
స := ఆ (హనుమ); ముహూర్తమ్ ఇవ = ఒక క్షణం ; ధ్యాత్వా =ఆలోచిస్తూ; బాష్ప = కన్నీటితో ; పర్యాకులేక్షణః = నిండిన నేత్రములు; సీతామ్ =సీతాదేవి; ఆశ్రిత్య = అవస్ధను; తేజస్వీ = తేజస్వియగు; హనుమాన్ = హనుమ; విలలాప హ = విలపించెను;
తేజస్వియగు హనుమంతుడు కొంచెము సేపు ఆలోచించి, కన్నీటితో నిండిన నేత్రములు కలవాడై ఆమె అవస్ధను చూచి ఇట్లు విలపించెను – 16.2

मान्या गुरु विनीतस्य लक्ष्मणस्य गुरुप्रिया
यदि सीतापि दुःखार्ता कालो हि दुरतिक्रमः 		మాన్యా గురు వినీతస్య - లక్ష్మణస్య గురుప్రియా
యది సీతాపి దుఃఖార్తా - కాలోహి దురతిక్రమః
మాన్యా = గౌరవనీయమైన; గురు వినీతస్య = గురువుల వలన తగిన శిక్షణను పొందిన; లక్ష్మణస్య = లక్ష్మణునకు; గురు ప్రియా =పూజ్యురాలైన; యది = ఒకవేళ; సీత అపి = సీతాదేవికి కూడ; దుఃఖార్తా = దు:ఖమున మునిగి; కాలోహి = విధి/ కాలము; దురతిక్రమః = అతిక్రమించుట కష్టం
శ్రీరామునికి ప్రియురాలై, గురువుల వలన తగిన శిక్షణను పొందిన లక్ష్మణునకు పూజ్యురాలైన ఈ సీతాదేవికి కూడ దు:ఖమున మునిగి ఉండుటకు కాలమే కారణము. కాలమును అతిక్రమించుట ఎవ్వరికిని సాధ్యము కాదు -16.3

रामस्य व्यवसायज्ञा लक्ष्मणस्य च धीमतः
नात्यर्थम् क्षुभ्यते देवी गङ्गेव जलदागमे 		రామస్య వ్యవసాయజ్ఞా - లక్ష్మణస్య చ ధీమతః
నాత్యర్థం క్షుభ్యతే దేవీ - గ౦గేవ జలదాగమే
రామస్య = శ్రీరాముడు; వ్యవసాయజ్ఞా = ప్రయత్నము/కృషి; లక్ష్మణస్య = లక్ష్మణుని ; ధీమతః = ధీశాలియగు; అత్యర్థం = అత్యర్ధము; న క్షుభ్యతే = కలత చెందక ఉన్నది; దేవీ = సీతామాత; గంగా ఇవ = గంగానది వలె; జలదాగమే = వర్షాకాలము నందు
తనని శత్రువు నుండి విడిపించుటకు శ్రీరాముడు చేయు ప్రయత్నమును, ధీశాలియగు లక్ష్మణుని పరాక్రమమును గుర్తెరిగిన ఈ సీతామాత వర్షాకాలము నందు గంగానది వలె అత్యర్ధము కలత చెందక ఉన్నది – 16.4

तुल्य शील वयोवृत्ताम् तुल्याभिजन लक्षणाम्
राघवोर्हति वैदेहीम् तम् चेयम असितेक्षणा 		తుల్య శీల వయోవృత్తాం - తుల్యాభిజన లక్షణామ్
రాఘవోర్హతి వైదేహీం - తం చేయమసితేక్షణా
తుల్య = సరిసమానము ;శీల =ఉత్తమ స్వభావము; వయోవృత్తాం = వయస్సు మరియు ప్రవర్తన; తుల్య = సరిసమా; అభిజన = వంశము; లక్షణామ్ = లక్షణాలు; రాఘవ = శ్రీరాముడు; అర్హతి = తగియున్నది; వైదేహీం = సీతాదేవి; ఇయమ్ = ఈ; అసితేక్షణా = నల్లని కన్నులు;
సరిసమానములగు ఉత్తమ స్వభావముచేతను, వయస్సు చేతను, సదాచారము చేతను, వంశము చేతను శ్రీరాముడు సీతాదేవికి తగినవాడు. నల్లని కన్నులు గల ఈ సీతాదేవి అతనికి అన్ని విధముల తగియున్నది – 16.5

तां दृष्ट्वा नवहेमाभां लोककान्तामिव श्रियम्
जगाम मनसा रामम् वचनम् च इदमब्रवीत् 		తాం దృష్ట్వా నవహేమాభాం - లోకకాన్తామ్ ఇవ శ్రియమ్
జగామ మనసా రామం - వచనం చ ఇదమబ్రవీత్
తాం దృష్ట్వా = సీతాదేవిని చూచి; నవ హేమాభాం = కొత్త బంగారు కాంతితో; లోకకాన్తామ్ = ప్రపంచమునకు ప్రీతిపాత్రమై;ఇవ శ్రియమ్ = లక్ష్మిదేవి వంటి; జగామ = తలంచుకొని; మనాసా = మనస్సున; రామం = శ్రీ రాముడు; ఇదం వచనం చ = ఈ పదాలు; అబ్రవీత్ = మాట్లడుకొనెను;
కొత్త బంగారు కాంతి వలె ప్రకాశించుచు, ప్రపంచమునకు ప్రీతిపాత్రమై, లక్ష్మీదేవిని పోలి ఉన్న ఈ సీతాదేవిని చూచి, హనుమంతుడు శ్రీరాముని తలంచుకొని తనలో ఇట్లనుకొనెను - 16. 6

अस्या हेतोर् विशालाक्ष्या हतो वाली महाबलः
रावण प्रतिमो वीर्ये कबन्धश्च निपातितः 		అస్యా హేతోర్ విశాలాక్ష్యా - హతో వాలీ మహాబలః
రావణ ప్రతిమో వీర్యే - కబంధశ్చ నిపాతితః
అస్యా =ఈ సీతాదేవి; హేతోర్= కొరకు; విశాలాక్ష్యా= విశాలమగు నేత్రములు; హతో = సంహరింపబడెను; వాలీ = వాలి; మహా బలః = బలశాలియగు;రావణ ప్రతిమో=రావణునకు సమానుడైన;వీర్యే= బలం/ పరాక్రమములో; కబంధశ్చ = కబంధుడు కూడ; నిపాతితః = చంపబడెను
విశాలమగు నేత్రములు కల ఈ సీతాదేవిని కూర్చియే గొప్ప బలశాలియగు వాలి శ్రీరామునిచే సంహరింపబడెను. పరాక్రమములో రావణునకు సమానుడైన కబంధుడు కూడ చంపబడెను – 16.7

विराधश्च हतः सम्ख्ये राक्षसो भीम विक्रमः
वने रामेण विक्रम्य महेन्द्रेण इव शम्बरः 		విరాధశ్చ హతః సంఖ్యే - రాక్షసో భీమ విక్రమః

चतुर्दश सहस्राणि रक्षसाम् भीम कर्मणाम्

अस्या निमित्ते सुग्रीवः प्राप्तवान् लोक सत्कृतम् 		ఐశ్వర్యం వానరాణాం చ - దుర్లభం వాలి పాలితమ్

ఐశ్వర్యం = ఐశ్వర్యం /సంపద; వానరాణాం = వానరులకు; దుర్లభం = దుర్లభము; వాలి పాలితమ్ = వాలి పాలించిన; అస్యా = ఆమె; నిమిత్తే = కొరకు/ కోసం; సుగ్రీవః = సుగ్రీవునకు; ప్రాప్తవాన్ = ఇవ్వబడెను; లోక సత్కృతమ్ = లోకమున గౌరనీయమైన (రాజు పదవి)
వానరులకు దుర్లభమును, వాలిచే అనుభవింపదగినది, లోకమున గౌరనీయమైన సంపద ఈ సీతాదేవి కారణముగా శ్రీరామునిచే సుగ్రీవునకు ఇవ్వబడెను – 16.11

सागरश्च मयाक्रान्तः श्रीमान् नद नदीपतिः
अस्या हेतोर् विशालाक्ष्याः पुरी चेयम् निरीक्षिता 		సాగరశ్చ మయాక్రాన్తః - శ్రీమాన్ నద నదీపతిః
అస్యా హేతోర్ విశాలాక్ష్యా: - పురీ చేయం నిరీక్షితా
సాగరశ్చ = సముద్రము; మయా క్రాన్తః = నాచే దాటబడి; శ్రీమాన్ =శ్రీమంతుడు; నద నదీ పతిః = నదులు మరియు ప్రవాహాల ప్రభువు; అస్యా హేతోర్ = ఆమె కొరకు; విశాలాక్ష్యా:= విశాలమగు నేత్రములు; పురీ చ ఇయం = ఈ నగరం కూడా; నిరీక్షితా = చూడబడినది
శ్రీమంతుడును, నదములకు, నదులకు భర్త అయిన సముద్రుడు, సీతామాత కారణముగా అతికష్టముతో నాచే దాటబడి లంకా నగరమును చూడబడినది – 16.12

यदि रामः समुद्रान्ताम् मेदिनीम् परिवर्तयेत्
अस्याः कृते जगच्चापि युक्तम् इत्येव मे मतिः		 యది రామః సముద్రాన్తాం - మేదినీం పరివర్తయేత్
అస్యాః కృతే జగచ్చాపి - యుక్తమ్ ఇత్యేవ మే మతిః
యది = ఒకవేళ; రామః = శ్రీరాముడు; సముద్రాన్తాం = సముద్రపర్యంతము; మేదినీం = భూమిని; పరివర్తయేత్ = తలక్రిందులుగా చేసినను; అస్యాః కృతే = ఆమె కొరకు; జగచ్చాపి =లోకములన్నిటిని; యుక్తమ్ = ఇది సరైనదే; ఇత్యేవ = ఇది ;మే = నా ; మతిః = అభిప్రాయం
ఈమె కొరకై శ్రీరాముడు సముద్రపర్యంతము వ్యాపించియున్న భూమిని, లోకములన్నిటిని కూడ తారుమారు చేసినను యుక్తమేనని నాకు తోచుచున్నది – 16.13

राज्यम् वा त्रिषु लोकेषु सीता वा जनकात्मजा
त्रैलोक्य राज्यम् सकलम् सीताया नाप्नुयात् कलाम् 		రాజ్యం వా త్రిషు లోకేషు - సీతా వా జనకాత్మజా
త్రైలోక్య రాజ్యం సకలం - సీతాయా నాప్నుయాత్ కళామ్
త్రిషు లోకేషు = మూడులోకాలు; రాజ్యం వా = రాజ్యం లేదా?; సీతా వా = సీతాదేవి నా? ; జనకాత్మజా = జనక రాజు కుమార్తె; త్రైలోక్య రాజ్యం = త్రైలోక్యరాజ్యము; సకలం = సకలం / సర్వమును ; సీతాయా = సీతాదేవి ; న అప్నుయాత్ = సమంకాదు; కళామ్ = పదహారవ భాగం
త్రైలోక రాజ్యము గొప్పదా? జనక రాజపుత్రియగు సీతాదేవి గొప్పదా ? అని ఆలోచన కలిగినపుడు, త్రైలోక్యరాజ్యము సర్వమును సీతాదేవి పదునారవ వంతునకును తలతూగదు/ సమంకాదు – 16.14

इयम् सा धर्म शीलस्य मैथिलस्य महात्मनः
सुता जनकराजस्य सीता भर्तृ दृढ व्रता 		ఇయం సా ధర్మ శీలస్య - మైథిలస్య మహాత్మనః
సుతా జనక రాజస్య - సీతా భర్తృ దృఢ వ్రతా
ఇయం= ఈమె; ధర్మ శీలస్య = ధర్మశీలుడును; మైథిలస్య = మిధిల రాజ్యము యొక్క; మహాత్మనః= మహాత్ముడును; సుతా = కుమార్తె; జనక రాజస్య= జనక మహారాజు; సా సీతా = ఈ సీతామాత; భర్తృ దృఢ వ్రతా = భర్త యందు దృఢ చిత్తముతో ఉన్నది
ఈ సీతామాత ధర్మశీలుడును, మిధిలాపతియు, మహాత్ముడును అయిన జనక మహారాజు కుమార్తె, మహా పతివ్రత, భర్త యందు దృఢ చిత్తముతో బ్రతుకు చున్నది – 16.15

उत्थिता मेदिनीम् भित्त्वा क्षेत्रे हल मुख क्षते
पद्म रेणु निभैः कीर्णा शुभैः केदार पांसुभिः 		ఉత్థితా మేదినీం భిత్త్వా - క్షేత్రే హల ముఖక్షతే
పద్మ రేణు నిభైః కీర్ణా - శుభైః కేదార పాంసుభిః
ఉత్థితా = పైకి వచ్చిన; మేదినీం భిత్త్వా = భూమిని భేదించుకొని; క్షేత్రే హల ముఖ క్షతే = ఒక మైదానం దున్నబడుతున్నప్పుడు; పద్మ రేణు నిభైః = పద్మపరాగ రేణువులు పోలి;కీర్ణా = కప్పబడి; శుభైః = మంగళకరమగు; కేదార పాంసుభిః = పొలములోని రేణువుల;
పద్మపరాగ రేణువులు పోలి, మంగళకరమగు పొలములోని రేణువులచే కప్పబడి, నాగలిచే దున్నబడిన క్షేత్రములో భూమిని భేదించుకొని వచ్చిన మహానీయురాలు సీతాదేవి – 16.16

विक्रान्तस्य आर्य शीलस्य सम्युगेषु अनिवर्तिनः
स्नुषा दशरथस्य एषा ज्येष्ठा राज्ञो यशस्विनी 		విక్రాన్తస్యార్య శీలస్య - సంయుగేషు అనివర్తినః
స్నుషా దశరథస్యైషా - జ్యేష్ఠా రాజ్ఞో యశస్వినీ
విక్రాన్తస్య = పరాక్రమశాలియు; ఆర్య శీలస్య= ఉత్తమ శీలుడును; సంయుగేషు = యుద్ధంలో; అనివర్తినః = తిరుగులేని/ఎదురులేని; జ్యేష్ఠా= పెద్ద; స్నుషా= కోడలు ; రాజ్ఞో దశరథస్య= దశరధ మహారాజుకు; ఏషా= ఈమె ; యశస్వినీ = కీర్తి కలది;
గొప్ప పరాక్రమశాలియు, ఉత్తమ శీలుడును, యుద్ధమున వెనుదీయని వాడైన దశరధ మహారాజుకు ఈమె పెద్ద కోడలు, గొప్ప కీర్తి కలది – 16.17

धर्मज्ञस्य कृतज्ञस्य रामस्य विदितात्मनः
इयम् सा दयिता भार्या राक्षसी वशमागता 		ధర్మజ్ఞస్య కృతజ్ఞస్య - రామస్య విదితాత్మనః
ఇయం సా దయితా భార్యా - రాక్షసీ వశమాగతా
ధర్మజ్ఞస్య= ధర్మాత్ముడు; కృతజ్ఞస్య= కృతజ్ఞుడును; రామస్య= శ్రీరామునికి; విదితాత్మనః= పరమాత్మ తత్వము తెలిసినవాడు;ఇయం సా = ఈమె;దయితా = ప్రియమైన ; భార్యా = భార్య ;రాక్షసీ = రాక్షస స్త్రీలకు ;వశమ్ ఆగతా= లొంగియున్నది;
ధర్మములన్నిటిని ఎరిగినవాడును, కృతజ్ఞుడును, పరమాత్మ తత్వము తెలిసినవాడు అయిన శ్రీరామునికి ప్రీతి పాత్రమగు భార్య ఈమె. ఇప్పుడీ రాక్షస స్త్రీలకు లొంగియున్నది – 16.18

सर्वान् भोगान् परित्यज्य भर्तृ स्नेह बलात् कृता
अचिन्तयित्वा दुःखानि प्रविष्टा निर्जनम् वनम् 		సర్వాన్ భోగాన్ పరిత్యజ్య - భర్తృ స్నేహ బలాత్కృతా
అచిన్తయిత్వా దుఃఖాని - ప్రవిష్టా నిర్జనం వనమ్
సర్వాన్= అన్ని; భోగాన్= సుఖాలు; పరిత్యజ్య= విడిచిపెట్టడం; భర్తృ = భర్తయందలి ;స్నేహ బలాత్ కృతా= స్నేహబలమును;అచిన్తయిత్వా = లెక్కచేయక; దుఃఖాని = దు:ఖములను ;ప్రవిష్టా = ప్రవేశించిన;నిర్జనం = నిర్మానుష్యమైన ;వనమ్= వనమును;
ఈమె భర్తయందలి స్నేహబలమును సమస్త భోగములను విడిచి, దు:ఖములను ఏ మాత్రము లెక్కచేయక నిర్మానుష్యమైన ఈ వనమును ప్రవేశించినది – 16.19

सम्तुष्टा फल मूलेन भर्तृ शुश्रूषणे रता
या पराम् भजते प्रीतिम् वनेऽऽपि भवने यथा 		సంతుష్టా ఫల మూలేన - భర్తృ శుశ్రూషణే రతా
యా పరాం భజతే ప్రీతిం - వనేపి భవనే యథా
సంతుష్టా= సంతృప్తి/ సంతోషంగా ఉండటం ; ఫల= ఫలమును; మూలేన= దుంపలను; భర్తృ= భర్త; శుశ్రూషణే= సేవయందు; రతా=ఆసక్తురాలై;యా = ఈమె; పరాం = మహోన్నతమైన; భజతే ప్రీతిం= ప్రీతిని పొందుతున్నది; వనేపి= అడవిలో కూడా; భవనే యథా= ఒక రాజభవనంలో వలె;
భర్త సేవయందు ఆసక్తురాలైన ఈమె ఫలమును, దుంపలను తినుటలోనే సంతృప్తి పొందుచు, వనమున ఉన్ననూ ఒక రాజభవనంలో ఉండినట్లు మహోన్నతమైన ప్రీతిని పొందుతున్నది – 16.20

सा इयम् कनकवर्णाङ्गी नित्यम् सुस्मित भाषिणी
सहते यातनाम् एताम् अनर्थानाम् अभागिनी 		సేయం కనక వర్ణా౦గీ - నిత్యం సుస్మిత భాషిణీ
సహతే యాతనామ్ ఏతామ్ - అనర్థానామభాగినీ
సా ఇయం= ఆమె; కనక వర్ణా౦గీ= బంగారు మేనిఛాయగల; నిత్యం= ఎల్లప్పుడును; సుస్మిత= చిరు నవ్వుతోనే; భాషిణీ= మాటలాడుచుండును;సహతే = సహించుచున్నది; యాతనామ్ = యాతనలు; ఏతామ్ = ఈ ; అనర్థా నామ్ = అనర్ధాలు ; అభాగినీ = ఆపదలను పొందకూడని;
బంగారువంటి మేనిఛాయగల ఈమె, ఎల్లప్పుడును చిరు నవ్వుతోనే మాటలాడుచుండును! ఆపదలను పొందకూడని ఈమె ఇప్పుడెంతయో బాధను సహించుచున్నది – 16.21

इमाम् तु शील सम्पन्नाम् द्रष्टुम् इच्चति राघवः
रावणेन प्रमथिताम् प्रपाम् इव पिपासितः 		ఇమాం తు శీల సంపన్నాం - ద్రష్టుమర్హతి రాఘవః
రావణేన ప్రమథితాం - ప్రపామివ పిపాసితః
రాఘవ = రామ; ద్రష్టుమ్ అర్హతి = చూడవలసి యున్నది; ఇమాం = ఈ సీత; శీల సంపన్నాం = ఉత్తమ శీలముతో; రావణేన = రావణునిచే ; ప్రమథితాం = హింసింపబడుచున్న; ప్రపామ్ = లభించే చోటుకు/ చలిపందిరిని చూచినట్లు ; ఇవ పిపాసితః = దాహం కలిగిన
ఉత్తమ శీలముతో ఉండి, రావణునిచే హింసింపబడుచున్న ఈమెను, దప్పిగొన్నవాడు చలిపందిరిని చూచినట్లు చూడవలసి యున్నది – 16.22

अस्या नूनम् पुनर्लाभाद् राघवः प्रीतिमेष्यति
राजा राज्य परिभ्रष्टः पुनः प्राप्येव मेदिनीम् 		అస్యా నూనం పునర్లాభాత్ - రాఘవః ప్రీతిమేష్యతి
రాజా రాజ్య పరిభ్రష్టః - పునః ప్రాప్యేవ మేదినీమ్
అస్యా= ఆమెను; నూనం = ఖచ్చితంగా; పునర్లాభాత్ = మళ్ళీ పొంది; రాఘవ = శ్రీరాముడు; ప్రీతిమ్ = ఆనందం; ఏష్యతి = పొంది; రాజా = రాజు; రాజ్య పరిభ్రష్టః = రాజ్యమును కోల్పోయి ; పునః ప్రాప్య ఇవ = తిరిగి పొందడం; మేదినీమ్ = భూమి/ రాజ్యమును
రాజ్యభ్రష్ఠుడగు రాజు తిరిగి తన రాజ్యమును పొంది ఆనందించునట్లు, శ్రీరాముడు తిరిగి ఈమెను పొంది ఖచ్చితంగా సంతోషించును – 16.23

काम भोगैः परित्यक्ता हीना बन्धु जनेन च
धारयति आत्मनो देहम् तत् समागम काङ्क्षिणी 		కామ భోగైః పరిత్యక్తా - హీనా బంధు జనేన చ
ధారయత్యాత్మనో దేహం - తత్ సమాగమ కా౦క్షిణీ
కామ భోగైః = సుఖములను; పరిత్యక్తా = విడిచి; హీనా = దూరమైపోయిన; బంధు జనేన చ = బంధుజనులకు కూడా; ధారయతి ఆత్మనో = జీవించియున్నది; దేహం = శరీరముతో; తత్ సమాగమ = ఆ శ్రీరాముని సమాగమమును; కా౦క్షిణీ = కోరియే
అన్ని సుఖములను విడిచి, బంధుజనులకు దూరమైపోయిన ఈమె శ్రీరాముని సమాగమమును కోరియే శరీరముతో జీవించియున్నది – 16. 24

न एषा पश्यति राक्षस्यो न इमान् पुष्प फल द्रुमान्
एकस्थ हृदया नूनम् रामम् एव अनुपश्यति 		నైషా పశ్యతి రాక్షస్యో - నేమాన్ పుష్ప ఫల ద్రుమాన్
ఏకస్థ హృదయా నూనం - రామమేవ అనుపశ్యతి
ఏషా = ఆ; న పశ్యతి = చూచుటలేదు; రాక్షస్యో = రాక్షస స్త్రీలను గాని; ఇమాన్ = ఈ; పుష్ప = పువ్వులు; ఫల = పండ్లు ; ద్రుమాన్ = చెట్లు; న = చూచుటలేదు; ఏకస్థ = ఏకాగ్రచిత్తయై ; హృదయా = హృదయమున; నూనం = ఖచ్చితంగా ; రామమ్ = శ్రీరాముని ; ఏవ = మాత్రమే; అనుపశ్యతి = చూచుచున్నది /ధ్యానించుచున్నది;
ఈమె తన చుట్టు వున్న రాక్షస స్త్రీలను గాని, ఫల పుష్పములతో కూడిన వృక్షములను కాని చూచుటలేదు. ఏకాగ్రచిత్తయై తన హృదయమునున్న శ్రీరాముని మాత్రమే చూచుచున్నది – 16.25

भर्ता नाम परम् नार्या भूषणम् भूषणाद् अपि
एषा तु रहिता तेन भूषणार्हा न शोभते 		భర్తా నామ పరం నార్యా - భూషణం భూషణాదపి
ఏషా తు రహితా తేన - భూషణార్హా న శోభతే
భర్తా నామ= భర్తయే; పరం= ఉత్తమ; నార్యా భూషణం= భార్యలకు భూషణం; భూషణాత్ అపి= ఆభరణాల కంటే ఎక్కువ; ఏషా = ఈ సీతాదేవి; రహితా తేన = భర్త సమీపమున లేకుండటచే; భూషణార్హా = అలంకరణ అర్హమైనప్పటికీ; న శోభతే = శోభింపకున్నది;
ఉత్తమ వనితకు నగలకంటె భర్తయే మేలైన భూషణము. భూషనార్హయగు ఈ సీతాదేవి, భర్త సమీపమున లేకుండటచే శోభింపకున్నది – 16.26

दुष्करम् कुरुते रामो हीनो यदनया प्रभुः
धारयति आत्मनो देहम् न दुःखेन अवसीदति 		దుష్కరం కురుతే రామో - హీనో యదనయా ప్రభుః
ధారయత్యాత్మనో దేహం - న దుఃఖే నావసీదతి
దుష్కరం= దుష్కరమైన; కురుతే =కార్యాలు చేస్తూ; రామో= శ్రీరాముడు; హీనోయద = విడిచిన; అనయా= ఈమెను; ప్రభుః= ప్రభువైన;ధారయతి = ఉండుట;ఆత్మనో దేహం = ప్రాణములతో; దుఃఖేన= దుఃఖంతో; న అవసీదతి = అసాధ్యము
ప్రభువైన శ్రీరాముడు ఈమెను విడిచిన దు:ఖము వలన మరణింపక ప్రాణములతో ఉండుట వింతయే! ఇంట్లుండుట అసాధ్యము – 16. 27

इमाम् असित केशान्ताम् शतपत्र निभेक्षणाम्
सुखार्हाम् दुःखिताम् दृष्ट्वा ममापि व्यथितम् मनः 		ఇమామసిత కేశాన్తాం - శతపత్ర నిభేక్షణామ్
సుఖార్హాం దుఃఖితాం దృష్ట్వా - మమాపి వ్యథితం మనః
ఇమామ్ = ఆమె; అసిత కేశాన్తాం = నల్లని కేశముల కలిగి; శత పత్ర = పద్మమును పోలు ;నిభేక్షణామ్ = కన్నులు కలిగి; సుఖ అర్హాం = సుఖములకు అర్హురాలైనను; దుఃఖితాం = దు:ఖమునే; దృష్ట్వా = చూడగా ; మమ = నా ; వ్యథితం= వ్యధ నొందుచున్నది; మనః = మనస్సు
నల్లని కేశముల కలిగి, పద్మమును పోలు కన్నులు కలిగి, సుఖములకు అర్హురాలైనను దు:ఖమునే అనుభవించుచున్న ఈమెను చూడగా నా మనస్సు మిగుల వ్యధ నొందుచున్నది – 16.28

क्षिति क्षमा पुष्कर सन्निभाक्षी
या रक्षिता राघव लक्ष्मणाभ्याम्
सा राक्षसीभि: विकृतेक्षणाभिः
सम्रक्ष्यते सम्प्रति वृक्ष मूले 		క్షితి క్షమా పుష్కర సన్నిభాక్షీ
యా రక్షితా రాఘవ లక్ష్మణాభ్యామ్
సా రాక్షసీభి: వికృతేక్షణాభిః
సంరక్ష్యతే సంప్రతి వృక్ష మూలే
క్షితి క్షమా = భూమికి సమాన ఓర్పు; పుష్కర సన్నిభాక్షీ = పద్మములవంటి కన్నులు; యా = ఎవరైతే (సీత); రక్షితా = రక్షించబడింది; రాఘవ లక్ష్మణాభ్యామ్ = రామలక్ష్మణులచే; సా = ఈ సీతామాత; రాక్షసీభిర్ = రాక్షస స్త్రీలచే; వికృతేక్షణాభిః = వికారముగా ఉన్న; సంరక్ష్యతే = రక్షింపబడుచున్నది ; సంప్రతి = ఇప్పుడు ; వృక్ష మూలే = వృక్షము క్రింద ;
భూమికి సమాన ఓర్పు కలదియు, పద్మములవంటి కన్నులు కల ఈ సీతామాత, ఇదివరలో రామలక్ష్మణులచే రక్షింపబడుచుండెడిది. ఇట్టి ఈమె ఇప్పుడీ వృక్షము క్రింద వికారముగా ఉన్న రాక్షస స్త్రీలచే రక్షింపబడుచున్నది కదా – 16. 29

हिम हत नलिनी इव नष्ट शोभा

जनक सुता कृपणाम् दशाम् प्रपन्ना 		హిమ హత నళినీవ నష్ట శోభా

సహచర రహితేవ చక్రవాకీ

అస్యా హి= ఈమె; పుష్ప = పూవుల ;అవనత = బరువుచే ;అగ్ర శాఖాః = వంగిన కొమ్మలు; శోకం= శోకము; దృఢం = అధికము; వై జనయతి = కలిగించుచున్నది/ చేయుచున్నవి; అశోకాః= అశోక వృక్షాలు; హిమ వ్యపాయేన చ = వసంతకాలం ప్రారంభంలో ; మన్దరశ్మి: = చల్లని చంద్రుడు కాంతితో;అభ్యుత్థితో =ఉదయించు; న ఏక సహస్ర రశ్మిః= వేల కిరణములతో(శోకం కలిగించే)
పూలభారముచే వంగిన కొమ్మలు గల అశోకవృక్షము ఈమె శోకమును అధికము చేయుచున్నవి. వసంత కాలంలో చంద్రుడు వేలకొలది చల్లని కిరణములతో ప్రకాశించుచు, ఈమె విరహతాపమును ఇనుమడింప చేయుచున్నాడు- 16.31

इति एवम् अर्थम् कपिरन्ववेक्ष्य
सीतेयम् इति एव निविष्ट बुद्धिः
संश्रित्य तस्मिन् निषसाद वृक्षे
बली हरीणाम् ऋषभः तरस्वी 		ఇత్యేవ మర్థం కపిరన్వవేక్ష్య
సీతేయమిత్యేవ నివిష్ట బుద్ధిః
సంశ్రిత్య తస్మిన్ నిషసాద వృక్షే
బలీ హరీణామ్ ఋషభ: తరస్వీ
ఇతి ఏవమ్ = ఆ విధంగా; అర్థం= విషయం; కపిర్ = హనుమ; అన్వవేక్ష్య = గమనించి; సీతేయమ్ = ఆమె సీతామాతయే ; ఇతి ఏవ = అని; నివిష్ట బుద్ధిః= నిశ్చయబుద్ధి; సంశ్రిత్య = ఇటుల ఆలోచించి ; తస్మిన్ = ఆ ; నిషసాద = కూర్చుని; వృక్షే = వృక్షము పైనే; బలీ =బలవంతుడు; హరీణామ్= వానరశ్రేష్ఠుడు; ఋషభ:= ఆంబోతువలె; తరస్వీ= వేగవంతుడును;
బలవంతుడును, వేగవంతుడును, వానరశ్రేష్ఠుడు అయిన హనుమంతుడు ఇటుల ఆలోచించి, ఆమె సీతామాతయే అని నిశ్చయబుద్ధి కలవాడై ఆ వృక్షము పైనే కూర్చుండి ఉండెను – 16.32
