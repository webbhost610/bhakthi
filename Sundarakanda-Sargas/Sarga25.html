अथ तासां वदन्तीनां परुषं दारुणं बहु
राक्षसीनामसौम्यानां रुरोद जनकात्मजा 		అథా తాసాం వదన్తీనాం - పరుషం దారుణం బహు
రాక్షసీనామ్ అసౌమ్యానాం - రురోద జనకాత్మజా
అథా = దాని తరువాత; తాసాం = ఆ; వదన్తీనాం = మాట్లాడుచున్న; పరుషం = పరుషముగాను; దారుణం = దారుణముగాను; బహు= చాలా సార్లు; రాక్షసీనామ్ = రాక్షస స్త్రీలు; అసౌమ్యానాం = క్రూరమైన ఆకారము; రురోద = ఏడ్చెను; జనకాత్మజా = సీతామాత;
సౌమ్యములు కాక భయంకరమైన ఆకారముల గల ఆ రాక్షస స్త్రీలు ఆ రీతిన తనను పరుషముగాను, దారుణముగాను పెక్కుసార్లు బెదిరించుచుండగా, వారి మాటలు విని జనకరాజనందన అయిన సీతామాత ఏడ్చెను – 25.1

एवमुक्ता तु वैदेही राक्षसीभिर्मनस्विनी
उवाच परमत्रस्ताबाष्पगद्गदया गिरा 		ఏవమ్ ఉక్తాతు వైదేహీ - రాక్షసీభి: మనస్వినీ
ఉవాచ పరమ త్రస్తా - బాష్ప గద్గదయా గిరా
ఏవమ్ = ఆ విధంగా; ఉక్తాతు = మాట్లాడటం; వైదేహీ = సీతామాత; రాక్షసీభి: = రాక్షసస్త్రీలు; మనస్వినీ = మంచి మనస్సుగల; ఉవాచ = పలికెను; పరమ= మిక్కిలి; త్రస్తా = భమమొంది ; బాష్ప = కన్నీరు కారణంగా; గద్గదయా = గద్గద స్వరంతో ; గిరా = తడబడుచూ;
మంచి మనస్సుగల సీతామాత రాక్షసస్త్రీలు పలికిన ఆ మాటలు విని మిక్కిలి భమమొంది, కన్నీరు కారణంగా గద్గద స్వరంతో తడబడుచూ ఇట్లనెను - 25. 2

न मानुषी राक्षसस्य भार्या भवितुमर्हति
कामं खादत मां सर्वा न करिष्यामि वो वचः 		న మానుషీ రాక్షసస్య - భార్యా భవితుమ్ అర్హతి
కామం ఖాదత మాం సర్వా - న కరిష్యామి వో వచః
మానుషీ = మనుస్య స్త్రీ; రాక్షసస్య = రాక్షసునికి; భార్యా = భార్య; భవితుమ్ = ఉండుట; న అర్హతి = సరిపోదు/ తగదు; కామం = మీ కోరిక; ఖాదత = తినినను; మాం = నేను; సర్వా = మీరందరు; న కరిష్యామి = నేను చేయను; వ: = మీరు; వచః = చెప్పినట్లు;
నేను మనుస్య స్త్రీని రాక్షసునికి భార్యగా ఉండుటకెంతమాత్రము తగను. మీరందరును కలిసి నన్ను చంపి తినినను తినవచ్చును, మీరు చెప్పినట్లు నేను చేయను - 25. 3

सा राक्षसीमध्यगता सीता सुरसुतोपमा
न शर्म लेभे दुःखार्ता रावणेन च तर्जिता 		సా రాక్షసీ మధ్యగతా - సీతా సుర సుతోపమా
న శర్మ లేభే దుఃఖార్తా - రావణేన చ తర్జితా
సా రాక్షసీ = ఆ రాక్షస స్త్రీల; మధ్యగతా = నడుమ ఉండుటవలన ; సీతా = సీతామాత; సుర సుతోపమా = దేవకన్యవలె; న శర్మ లేభే = సుఖమును పొందలేకపోయెను; దుఃఖార్తా = దుఃఖంతో బాధపడుతున్న; రావణేన చ = రావణునిచే; తర్జితా = బెదిరింపబడి /భయపెట్టబడి;
దేవకన్యను పోలియున్న సీతామాత, అట్లు రావణునిచే బెదిరింపబడి, రాక్షస స్త్రీల నడుమ ఉండుటవలన క్షణకాలమైనను సుఖమును పొందలేక దుఃఖంతో బాధపడు చుండెను - 25.4

वेपते स्माधिकं सीता विशन्ती वाङ्गमात्मनः
वने यूथा परिभ्रष्टा मृगी कोकैरिवार्दिता 		వేపతేస్మ అధికం సీతా - విశంతీ వాంగమాత్మనః
వనే యూథ పరిభ్రష్టా - మృగీ కోకై: ఇవార్దితా
వేపతేస్మ= కంపింపసాగెను; అధికం = మిక్కిలి; విశంతీవ = లోనికిచొచ్చుకొని పోవుచున్న; అంగ = అవయవములన్ని; ఆత్మన:= శరీరము; వనే = అడవిలో; యూథ = తన గుంపునుండి ; పరిభ్రష్టా = విడిపోయి; మృగీ ఇవ = ఆడలేడి వలె; కోకై: = తోడేళ్ళు; అర్దితా = వేధింపు/బాధింపబడు;
సీతామాత తన అవయవములన్నింటిని శరీరములోనికి చొచ్చుకొని పోవుచున్నట్లున్నదై, అడవిలో తన గుంపునుండి విడిపోయి తోడేళ్ళచే బాధింపబడుచున్న ఆడలేడి వలె మిక్కిలి కంపింపసాగెను - 25.5

सा त्वशोकस्य विपुलां शाखामालम्ब्य पुष्पिताम्
चिन्तयामास शोकेन भर्तारं भग्नमानसा 		సా తు అశోకస్య విపులాం - శాఖామాలంబ్య పుష్పితామ్
చిన్తయా మాస శోకేన - భర్తారం భగ్న మానసా
సా తు = ఆ సీతామాత; అశోకస్య = అశోక వృక్షపు; విపులాం శాఖాం = పెద్ద కొమ్మను; ఆలంబ్య = పట్టుకొని; పుష్పితామ్ = పూచియున్న; చిన్తయా మాస = ఆలోచింపసాగెను; శోకేన = శోకం / దు:ఖించుచు ; భర్తారం = భర్తయగు శ్రీరాముని; భగ్న మానసా= భగ్న హృదయంతో;
భగ్నహృదయ అయిన సీతామాత, బాగుగా పూచియున్న ఆ అశోక వృక్షపు పెద్ద కొమ్మను పట్టుకొని దు:ఖించుచు, భర్తయగు శ్రీరాముని గూర్చియే ఆలోచింపసాగెను - 25.6

सा स्नापयन्ती विपुलौ स्तनौ नेत्रजलस्रवैः
चिन्तयन्ती न शोकस्य तदान्तमधिगच्छति 		సా స్నాపయన్తీ విపులౌ - స్తనౌ నేత్ర జల స్రవైః

సా = ఆమె; స్నాపయన్తీ = తడిసిపోయెను; విపులౌ స్తనౌ = విస్తృత స్తనములు; నేత్ర జల = కన్నీటిధారలు; స్రవైః= ప్రవహించి; చిన్తయన్తీ = విచారించుచున్న; శోకస్య = శోకమునకు; తదా అంతమ్ = ఆ అంతము; న అధిగచ్ఛతి = కానరాలేదు;
ఆమె కన్నీటిధారలు ప్రవహించి ఆమె స్తనములు పూర్తిగా తడిసిపోయెను. విచారించుచున్న ఆమె శోకమునకు అంతము కానరాలేదు -25.7

सा वेपमाना पतिता प्रवाते कदळी यथा
राक्षसीनां भयत्रस्ता विवर्ण वदनाभवत् 		సా వేపమానా పతితా - ప్రవాతే కదళీ యథా

దు:ఖార్తయగు సీతామాత, హా రామా, హా లక్ష్మణా, హా అత్తగారైన కౌసల్యా, హా సుమిత్రా అనుచు విలపించెను - 25.11

अकाले दुर्लभो मृत्युः स्त्रिया वा पुरुषस्य वा 		లోక ప్రవాదః సత్యోయం - పణ్డితైః సముదాహృతః
అకాలే దుర్లభో మృత్యుః - స్త్రియా వా పురుషస్య వా
లోక ప్రవాదః = ప్రజలలో/లోకంలోని మాట; సత్యోయం = నిశ్చయముగా/ నిజంగా ; పణ్డితైః = పండితులు ; సముదాహృతః = చెప్పుదురు ; అకాలే = అకాలముగా; దుర్లభో = దుర్లభం/కానిపని; మృత్యుః = మరణింపవలెనని; స్త్రియా వా = స్త్రీగాని ; పురుషస్య వా= పురుషుడుగాని;
పురుషుడుగాని, స్త్రీగాని - తాను మరణింపవలెనని కోరుకున్నాను, కాలము కలసిరానిచో మరణము కూడ సంభవింపదని పండితులు చెప్పుదురు. ఆ మాట నా యెడల నిశ్చయముగా కానవచ్చుచున్నది - 25.12

यदाहमेवं क्रूराभी राक्षसीभिरिहार्दिता
जीवामि हीना रामेण मुहूर्तमपि दुःखिता 		యత్రాహ మేవం క్రూరాభీ - రాక్షసీభి: ఇహార్దితా
జీవామి హీనా రామేణ - ముహూర్తమపి దుఃఖితా
యత్ర ఇహ = ఇక్కడ; మే = నేను ; ఇవం = ఆ విధంగా; క్రూరాభీ = క్రూరులగు ; రాక్షసీభి:= రాక్షసస్త్రీల ; ఇహ ఆర్దితా = ఇట్లు పీడింపబడుచు; హీనా రామేణ = శ్రీరాముని విడిచి; జీవామి = జీవించుచునే ఉన్నాను; ముహూర్తమ్ = ఒక క్షణం; అపి = కూడా; దుఃఖితా = దు:ఖితురాలై
నేనిచ్చట క్రూరులగు రాక్షసస్త్రీల వలన ఇట్లు పీడింపబడుచు, దు:ఖితురాలై శ్రీరాముని విడిచి ఇంకను జీవించుచునే ఉన్నాను - 25.13

एषाल्पपुण्या कृपणा विनशिष्याम्यनाथवत्
समुद्रमथ्ये नौः पूर्णा वायुवेगैरिवाहता		 ఏషాల్ప పుణ్యా కృపణా - వినశిష్యామి అనాథవత్
సముద్ర మధ్యే నౌ: పూర్ణా - వాయు వేగై: ఇవాహతా
ఏష= అయిన నేను; అల్ప పుణ్యా= అల్ప పుణ్యంతో; కృపణా= దౌర్భాగ్య/ దుర్జనురాలను; వినశిష్యామి = నశించిపోవు; అనాథ వత్= అనాధ/ దిక్కులేనిదానివలె; సముద్ర మధ్యే= సముద్ర మధ్యమున; నౌ: పూర్ణా = నిండు నౌక ; వాయు వేగై: = పెనుగాలి వలన ; ఇవ ఆహతా= నశించినట్లు;
సముద్ర మధ్యమున ప్రయాణము చేయు నిండు నౌక పెనుగాలి వలన గట్టి దెబ్బతిని నశించినట్లు, పాపాత్మురాలను, దుర్జనురాలను అయిన నేను దిక్కులేనిదానివలె నశించిపోవుచున్నాను - 25.14

भर्तारं तमपश्यन्ती राक्षसीवशमागता
सीदामि खलु शोकेन कूलं तोयहतं यथा 		భర్తారం తమ పశ్యన్తీ - రాక్షసీ వశమాగతా
సీదామి ఖలు శోకేన - కూలం తోయ హతం యథా
భర్తారం = నా భర్త; తమ్ = ఆ శ్రీరాముని; అపశ్యన్తీ = చూడజాలని; రాక్షసీ వశమ్ = రాక్షస స్త్రీలకు చిక్కిన; ఆగతా = దాననై; సీదామి= నశించిపోవుచున్నాను; ఖలు = నిశ్చయముగా; శోకేన = బాధతో; తోయ హతం = జల ప్రవాహపు తాకిడి వలన; కూలం యథా = ఏటి గట్టువలె;
రాక్షస స్త్రీలకు చిక్కిన నేను నా భర్తను చూడజాలని దాననై, జల ప్రవాహపు తాకిడి వలన నశించు ఏటిగట్టువలె దు:ఖితురాలనై నిశ్చయముగా నశించిపోవుచున్నాను - 25.15

तं पद्मदळ पत्राक्षं सिंहविक्रान्तगामिनम्
धन्याः पश्यन्ति मे नाथं कृतज्ञं प्रियवादिनम् 		తం పద్మదళ పత్రాక్షం - సింహ విక్రాన్త గామినమ్
ధన్యాః పశ్యన్తి మే నాథం - కృతజ్ఞం ప్రియ వాదినమ్
తం = ఆ శ్రీరాముడిని; పద్మదళ పత్రాక్షం = తామరపూవు రేకులవంటి కన్నులు; సింహ విక్రాన్త = సింహమువలె ఠీవిగా ; గామినమ్ = నడుచువాడును ; ధన్యాః = ధన్యులు మాత్రమే; పశ్యన్తి = చూడగలరు; మే = నావంటి; నాథం = భర్త; కృతజ్ఞం = ఉపకారము చిన్నదైనను గొప్పగా భావించువాడు; ప్రియ వాదినమ్ = ప్రియముగా మాటలాడువాడు;
తామరపూవు రేకులవంటి కన్నులు కలవాడు, సింహమువలె ఠీవిగా నడుచువాడును, ఇతరులు ఒనర్చు ఉపకారము చిన్నదైనను గొప్పగా భావించువాడు, ఎల్లప్పుడు ప్రియముగా మాటలాడువాడు అయిన నా భర్తయగు శ్రీరాముడిని, ధన్యులే చూడగలరు, నావంటి అధన్యురాలెట్లు చూడగలదు ? - 25.16

सर्वथा तेन हीनाया रामेण विदितात्मना
तीक्ष्णं विषमिवास्वाद्य दुर्लभं मम जीवितम् 		సర్వథా తేన హీనాయా - రామేణ విదితాత్మనా
తీక్ష్ణం విషమివాస్వాద్య - దుర్లభం మమ జీవితమ్
సర్వథా = అన్ని విధాలుగా; తేన = అతను; హీనాయా = లేకపోవడం/ విడిచి; రామేణ = శ్రీరాముని ; విదితాత్మనా = జ్ఞాని/ పరతత్వవేత్తయగు; తీక్ష్ణం విషమ్ = తీవ్రమైన విషము; ఇవ ఆస్వాద్య = త్రాగినవాని; దుర్లభం = కష్టం ; మమ = నా; జీవితమ్ = జీవితం;
పరతత్వవేత్తయగు శ్రీరాముని విడిచి నా జీవితము తీవ్రమైన విషము త్రాగినవాని జీవితమువలె నశింపవలసినదే కదా - 25.17

कीदृशं तु महापापं मया जन्मान्तरे कृतम्
येनेदं प्राप्यते दुःखं मया घोरं सुदारुणं 		కీదృశం తు మయా పాపం - పురా జన్మాంతరే కృతమ్
యేనేదం ప్రాప్యతే దుఃఖం-మయా ఘోరం సుదారుణమ్
కీదృశం = ఎట్టి /ఏ రకమైన; మయా = నేను; పాపం = పాపం; పురా జన్మాంతరే = వెనుకటి జన్మమున ; కృతమ్ = చేసియున్నానో; యేన = ఏ; ఇదం = ఇప్పుడు; ప్రాప్యతే = సంపాదించిన; దుఃఖం = దుఃఖము; మయా = నేను; ఘోరం = ఘోరమైన; సుదారుణమ్ = దారుణము;
నేను వెనుకటి జన్మమున ఎట్టి పాపము చేసియున్నానో? దానికి ఫలముగ ఇప్పుడు మిక్కిలి దారుణమును, ఘోరమును అయినట్టి దు:ఖమును అనుభవించుచున్నాను - 25.18

जीवितं त्यक्तुंइच्छामि शोकेन महता वृता
राक्षसीभिश्च रक्ष्यन्त्या रामो नासाद्यते मया 		జీవితం త్యక్తుమిచ్ఛామి - శోకేన మహతా వృతా
రాక్షసీభిశ్చ రక్షన్త్యా - రామో నాసాద్యతే మయా
జీవితం = జీవితం; త్యక్తుమ్ = విడవడం/వదిలివేయడం; ఇచ్ఛామి = కోరుకుంటున్నాను; శోకేన మహతా = గొప్ప శోకము; వృతా = చుట్టుముట్టిన; రాక్షసీభిశ్చ = రాక్షస స్త్రీల; రక్షన్త్యా = రక్షణలో ఉన్న; రామో= శ్రీరాముని; న ఆసాద్యతే = చేరుట జరుగదు; మయా= నేను;
గొప్ప శోకము అనుభవించుచున్న నాకు మరణింపవలెనని ఉన్నది. ఈ రాక్షస స్త్రీల రక్షణలో ఉన్న నేను ఇక శ్రీరాముని చేరుట జరుగదు - 25.19

धिगस्तु खलु मानुष्यं धिगस्तु परवश्यताम्
न शक्यं यत्परित्यक्तुमात्मच्छन्देन जीवितम् 		ధిగస్తు ఖలు మానుష్యం - ధిగస్తు పరవశ్యతామ్
న శక్యం యత్ పరిత్యక్తుమ్ - ఆత్మచ్ఛన్దేన జీవితమ్
ధిగస్తు = ఆధారపడి; ఖలు = నిశ్చయముగా; మానుష్యం = మానవులకు; ధిగస్తు = ఆధారపడి; పరవశ్యతామ్ = ఇతరుల నియంత్రణలో ఉండటం; న శక్యం = సాధ్యం కాదు; యత్ = ఏ కారణంతో; పరిత్యక్తుమ్ = విడుచుటకైనను; ఆత్మచ్ఛన్దేన = స్వేచ్ఛగా; జీవితమ్ = జీవితం;
నా జీవితము శ్రీరాముని అధీనములో ఉన్నది. నా ఇష్టానుసారము త్యజించుటకు నాకు అధికారము లేదు. అందువలన ఈ జీవితమును త్యజించుటకు నాకు అవకాశము లేదు - 25.20
