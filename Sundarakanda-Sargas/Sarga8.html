వసంత మాసాదపి కాంత దర్శన౦
స పుష్పకం తత్ర విమాన ముత్తమం
దదర్శ తద్వానర వీర సత్తమ :
వసంత= వసంత ఋతువున; పుష్పోత్కర= పుష్ప సమూహాల; చారు దర్శనం= అందంగా కనిపించెను; వసంతమ్ ఆసాద్ = వసంతకాలం; అపి= కూడా; కాంత దర్శన౦ = ఆకర్షణీయమై కనిపించెను; స: = ఆ; పుష్పకం = పుష్పకం అను పేరుతో; తత్ర = = అక్కడ; విమాన౦ = విమానం; ఉత్తమం = ఉత్తమమైనది; దదర్శ = చూసెను; తద్ = ఆ; వానర = హనుమ ;వీర సత్తమ : = వానరేంద్రుడు;
వసంత ఋతువున పూచు సకల పుష్పముల రాసులచే మనోజ్ఞమై, వసంత మాసముకంటె రమ్యమై, ఆ పుష్పకం అను విమానమును ఆ వానరేంద్రుడు చూచెను – 8.8

ఇది శ్రీమద్రామాయణమున సుందరాకాండమందు ఎనిమిదవ సర్గము సమాప్తము
