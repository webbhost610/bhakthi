సుందరకాండ - పదునేడవ/ సప్తదశ సర్గ

సీతామాతను కాంచి హనుమంతుడు ప్రసన్న మనస్కుడగుట

ततः कुमुद षण्डाभो निर्मलं निर्मलोदयः
प्रजगाम नभश्चन्द्रो हंसो नीलमिवोदकम् 		తతః కుముద షణ్డాభో - నిర్మలం నిర్మలోదయ:
ప్రజగామ నభశ్చన్ద్రో - హంసో నీలమివోదకమ్
తతః = దాని తరువాత; కుముద షణ్డాభో = తామర సమూహంతో సమానంగా ; నిర్మలం = నిర్మలమగు; నిర్మలః స్వయమ్ = స్వయముగా నిర్మలుడు (చంద్రుడు) ; ప్రజగామ = చేరెను; నభ:= ఆకాశము; చంద్రో = చంద్రుడు; హంసో = హంస; నీలమ్ ఇవ ఉదకమ్ = నీలిరంగు నీరు;
పిదప తెల్లకలువల గుంపుతో సరియగు కాంతికలవాడును, నిర్మలుడు అయిన చంద్రుడు, నీలవర్ణముగల జలమును హంస చేరినట్టు నిర్మలమగు ఆకాశమును చేరెను – 17.1

साचिव्यमिव कुर्वन् स प्रभया निर्मलप्रभः
चन्द्रमा रश्मभिः शीतैः सिषेवे पवनात्मजम् 		సాచివ్యమివ కుర్వన్ స - ప్రభయా నిర్మల ప్రభః
చన్ద్రమా రశ్మిభిః శీతైః - సిషేవే పవనాత్మజమ్
సాచివ్యమ్ = సాహాయము/పరిచర్యలు; ఇవ కుర్వన్ = చేయువానివలె; స: ప్రభయా = ఆ కాంతితో; నిర్మల ప్రభః = స్వచ్ఛమైన కాంతితో; చన్ద్రమా = ఆ చంద్రుడు; రశ్మిభిః శీతైః = చల్లని కిరణాలతో; సిషేవే = సేవించు; పవనాత్మజమ్ = హనుమ;
స్వచ్ఛమైన కాంతిగల ఆ చంద్రుడు, హనుమంతునిపై తన వెలుగును చూపి, సాహాయము చేయువానివలె తన చల్లని కిరిణములు అతనిపై వ్యాపింపచేసెను – 17.2

स ददर्श ततः सीताम् पूर्णचन्द्रनिभाननाम्
शोकभारैरिव न्यस्तां भारैर्नावमिवाम्भसि 		స దదర్శ తతః సీతాం - పూర్ణచన్ద్ర నిభాననామ్
శోకభారై: ఇవన్యస్తాం - భారైః నావమివాంభసి
స: = హనుమ; దదర్శ = చూచెను; తతః = తరువాత; సీతాం = సీతామాతను; పూర్ణ చన్ద్ర నిభాననామ్ = పూర్ణ చంద్రుడి ముఖంతో; శోక భారై: = శోకం భారముతో; ఇవ న్యస్తాం = మునిగివున్న; భారై: = బరువైన; నావం ఇవ = ఓడ వంటి; అంభసి = సముద్రములో;
పిమ్మట ఆ హనుమంతుడు, నీటిలో నావ మునిగినట్లు దు:ఖసముద్రములో మునిగివున్న పూర్ణచంద్ర ముఖి అయిన సీతామాతను చూచెను – 17. 3

दिदृक्षमाणो वैदेहीम् हनुमान् मारुतात्मजः
स ददर्शा विदूरस्था राक्षसी घोरदर्शनाः 		దిదృక్షమాణో వైదేహీం - హనూమాన్ మారుతాత్మజః
స దదర్శా విదూరస్థా - రాక్షసీ: ఘోరదర్శనాః
దిదృక్షమాణో=స్పష్టముగా చూడగోరి; వైదేహీం= సీతాదేవిని; స: హనూమాన్=హనుమ; మారుతాత్మజః = వాయు పుత్రుడు; దదర్శా = చూచెను ; విదూరస్థా = సమీపమున ;రాక్షసీర్ = రాక్షస స్త్రీలను; ఘోర దర్శనాః= భయము గొలుపు;
వాయుపుత్రుడగు హనుమంతుడు సీతాదేవిని చూస్తూనే, ఆమెకు సమీపమున ఉండి భయము గొలుపు ఆ రాక్షస స్త్రీలను చూచెను – 17.4

एकाक्षीम् एककर्णाम् च कर्णप्रावरणाम् तथा
अकर्णाम् शङ्कु कर्णाम् च मस्तकोच्छ्वास नासिकाम् 		ఏకాక్షీమ్ ఏక కర్ణాం చ - కర్ణ ప్రావరణాం తథా
అకర్ణాం శ౦కు కర్ణాం చ - మస్తకోచ్ఛ్వాస నాసికామ్
ఏకాక్షీమ్= ఒక కన్ను; ఏక కర్ణాం చ= మరియు ఒక చెవితో; కర్ణ ప్రావరణాం తథా= తలచుట్టు చెవులుగల దానిని;అకర్ణాం = చెవులులేని రాక్షసిని; శ౦కు కర్ణాం చ = శంఖము వంటి చెవులు ; మస్తక =తల వరకు; ఉచ్ఛ్వాస నాసికామ్ = ముక్కు కల దానిని;
ఒంటికన్ను కల ఒక రక్కసిని, ఒక్క చెవికల దానిని, తలచుట్టు చెవులు (పెద్ద పెద్ద) చెవులుగల దానిని, చెవులులేని రాక్షసిని, శంఖము వంటి (చుట్టుకొన్న) చెవులు కలదానిని, తల వరకు ముక్కు కల దానిని హనుమ చూచెను – 17.5

अति कायोत्त माङ्गीं च तनु दीर्घ शिरोधराम्
ध्वस्त केशीम् तथा केशीम् केशकम्बल धारिणीम् 		అతి కాయోత్తమా౦గీం చ - తను దీర్ఘ శిరోధరామ్
ధ్వస్త కేశీం తథా కేశీం - కేశ కమ్బళ ధారిణీమ్
అతి కాయోత్తమా౦గీం చ= శరీరమును మించిన తల కలదానిని ; తను దీర్ఘ శిరో ధరామ్= సన్నని పొడువాటి మెడగల దానిని;ధ్వస్త కేశీం = కొద్దిపాటి తలవెంట్రుకలు కల దానిని ; తథా అకేశీం = బట్టతల (జుట్టులేని) దానిని ;కేశ కమ్బళ ధారిణీమ్= దుప్పటిలాంటి జుట్టుతో;
అతడు శరీరమును మించిన తల కలదానిని, సన్నని పొడువాటి మెడగల దానిని, కొద్దిపాటి తలవెంట్రుకలు కల దానిని, బట్టతల దానిని, గొర్రెబొచ్చు వంటి (దుప్పటిలాంటి ) కేశములు కలదానిని చూచెను – 17.6

लम्बकर्णललाटाम् च लम्बोदरपयोधराम्
लम्बोष्ठीं चुबुकोष्ठीं च लम्बास्याम् लम्बजानुकाम् 		లమ్బ కర్ణ లలాటాం చ - లమ్బోదర పయోధరామ్
లమ్బౌష్ఠీం చిబుకౌష్ఠీం చ - లమ్బాస్యాం లమ్బ జానుకామ్
లమ్బ కర్ణ లలాటాం చ= వ్రేలుడు చెవులు మరియు నుదురు కలదానిని; లమ్బోదర పయోధరామ్= వ్రేలాడు స్తనములు కలదానిని; లమ్బౌష్ఠీం = వ్రేలాడు మీది పెదవి కలదానిని; చిబుకౌష్ఠీం చ = గడ్డముపై వ్రేలాడు క్రింది పెదవి కలదానిని; లమ్బాస్యాం లమ్బ జానుకామ్= వ్రేలాడు మోకాళ్ళు కలదానిని;
వ్రేలాడు చెవులు మరియు నుదురు కలదానిని, వ్రేలాడు స్తనములు, కడుపు కలదానిని, వ్రేలాడు మీది పెదవి కలదానిని, గడ్డముపై వ్రేలాడు క్రింది పెదవి కలదానిని, వ్రేలాడు మోకాళ్ళు కలదానిని హనుమ చూచెను - 17.7

ह्रस्वाम् दीर्घाम् च कुब्जाम् विकटाम् वामनां तथा
करालाम् भुग्नवक्त्राम् च पिङ्गाक्षीम् विकृताननाम् 		హ్రస్వాం దీర్ఘాం చ కుబ్జాం - వికటాం వామనాం తథా
కరాళా౦ భుగ్న వస్త్రాం చ - పి౦గాక్షీం వికృతాననామ్
హ్రస్వాం= పొట్టిదానిని; దీర్ఘాం చ= పొడుగైన దానిని; కుబ్జాం= కుబ్జురాలు (గూని); చ వికటాం= వంకరగా ఉన్నదానిని; వామనాం తథా= మరుగుజ్జు దానిని; కరాళా౦= ఎత్తు పళ్ళు; భుగ్న వస్త్రాం చ = వంకర నోరు ఉన్నది ;పి౦గాక్షీం = ఆకుపచ్చ కళ్ళు కలదానిని;వికృతాననామ్= వికారపు ముఖము కలదానిని;
అతడు పొట్టిదానిని, పొడుగైన దానిని, గూని దానిని , వంకరగా ఉన్నదానిని, మరుగుజ్జు దానిని, ఎత్తు పళ్ళు కలదానిని, వంకర మూతి కలదానిని, పిల్లి కన్నులు కలదానిని, వికారపు ముఖము కలదానిని చూచెను – 17.8

विकृताः पिङ्गलाः कालीः क्रोधनाः कलहप्रियाः
कालायस महाशूल कूटमुद्गर धारिणीः 		వికృతాః పి౦గళా: కాళీ: - క్రోధనాః కలహ ప్రియాః
కాలాయస మహాశూల - కూటముద్గర ధారిణీః
వికృతాః = వికారపు వేషముకల వారిని ; పి౦గళా: = పింగళవర్ణ శరీరము (పచ్చని) కలవారిని; కాళీ: = నల్లనిఛాయ కలవారిని; క్రోధనాః = కోపస్వభావము కలవారిని ; కలహ ప్రియాః = కలహాలు ఇష్టపడువారిని; కాలాయస =ఇనుముతో చేసిన; మహా శూల = మహా శూలము ; కూట ముద్గర ధారిణీః = సమ్మెట, ఇనుప గుదియలు ధరించిన వారిని;
వికారపు వేషముకల వారిని, పింగళవర్ణము కలవారిని, నల్లనిఛాయ కలవారిని, కోపస్వభావము కలవారిని, కలహమునకు సిద్ధపడువారిని, ఇనుముతో చేసిన శూలము, సమ్మెట, ఇనుప గుదియలు ధరించిన వారిని హనుమ చూచెను – 17.9

वराह मृगशार्दूल महिषाज शिवामुखीः
गजोष्ट्र हयपादीश्च निखात शिरसोपराः 		వరాహ మృగ శార్దూల - మహిషాజ శివా ముఖీ:
గజోష్ట్ర హయ పాదాశ్చ - నిఖాత శిరసోపరాః
వరాహ = పంది ; మృగ = జింక; శార్దూల = పులులు; మహిషా = దున్నపోతు ; అజ = మేకలు; శివా = నక్క ; ముఖీ: = ముఖము కలవారిని; గజ = ఏనుగు; ఓష్ట్ర =ఒంటె; హయ = గుర్రాలు; పాదాశ్చ = వంటి పాదములు ;నిఖాత శిరసోపరాః =శరీరములో తల చొచ్చి యున్న వారిని;
పంది, జింక, పులి, దున్నపోతు, మేక, నక్క ముఖము కలవారిని , ఏనుగు ,ఒంటె, గుర్రపు ,పాదముల వారిని, శరీరములో తల చొచ్చియున్న వారిని కూడ హనుమ చూచెను – 17.10

ప్రహర్షమతులం లేభే - మారుతిః ప్రేక్ష్య మైథిలీమ్
తాం క్షమాం = భూమివలె శాంతస్వభావియై; సువిభక్తా౦గీం = పొందికతోడి అవయవములు కలిగి; వినాభరణ = ఆభరణములను ధరింపకున్నను; శోభినీమ్ = శోభించుచుండెను; ప్రహర్షమ్ అతులం = గొప్ప ఆనందం; లేభే = పొందెను; మారుతిః = వాయుపుత్రుడు ; ప్రేక్ష్య = చూచి ; మైథిలీమ్ = సీతామాత ;
ఆమె భూమివలె శాంతస్వభావియై, పొందికతోడి అవయవములు కలిగి, ఆభరణములను ధరింపకున్నను శోభించుచుండెను. అట్టి మైధిలీదేవిని చూచి, హనుమంతుడు మిగుల సంతసించెను – 17.30

हर्षजानि च सोऽश्रूणि ताम् दृष्ट्वा मदिरेक्षणाम्
मुमोच हनुमांस्तत्र नमश्चक्रे च राघवम् 		హర్షజాని చ సోశ్రూణి - తాం దృష్ట్వా మదిరేక్షణామ్
ముమోచ హనుమాం తత్ర - నమశ్చక్రే చ రాఘవమ్
హర్షజాని చ =ఆనందంతో ;సోశ్రూణి = కన్నీళ్లు; తాం = ఆమెని; దృష్ట్వా = చూచి; మదిరేక్షణామ్ = మత్తును కలిగించు చూపులు; ముమోచ = (ఆనందభాష్పములను) విడిచెను ; హనుమాం = హనుమ; తత్ర నమశ్చక్రే = నమస్కరించెను ; చ రాఘవమ్ = శ్రీరామునకు
మత్తును కలిగించు చూపులు కల ఆమెను చూచి హనుమంతుడు ఆనందభాష్పములను విడిచెను. శ్రీరామునకు నమస్కరించెను – 17.31

नमस्कृत्वा च रामाय लक्ष्मणाय च वीर्यवान्
सीतादर्शन सम्हृष्टो हनुमान् सम्वृतोऽभवत् 		నమస్కృత్వా చ రామాయ - లక్ష్మణాయ చ వీర్యవాన్
సీతా దర్శన సంహృష్టో - హనూమాన్ సంవృతోభవత్
నమస్కృత్వా = నమస్కరించి ; చ రామాయ = శ్రీరామునకు ; లక్ష్మణాయ = లక్ష్మణునకు ; చ వీర్యవాన్ = పరాక్రమశాలియగు హనుమ; సీతా = సీతామాత ; దర్శన = దర్శనమైనందుకు ; సంహృష్టో =ఆనందం/సంతుష్టిని పొంది; హనూమాన్ = హనూమ ; సంవృతో = (ఆకులు) చాటుగా ; అభవత్ = ఉండెను
పరాక్రమశాలియగు హనుమ శ్రీరామునకు, లక్ష్మణునకు నమస్కరించి, సీతాదేవి తనకు కనబడినందుకు సంతుష్టిని పొంది, ఆ వృక్షముపై ఇతరులకు కానరాక చాటుగా ఉండెను – 17.32
