అందమైన కనులు, సౌశీల్యవతి అయిన సీతామాతను చూసి రావణుని భార్యలు అసూయగ్రస్తులై దుష్టబుద్ధితో ఆమెను భక్షించిరేమో – 13.12

సంపూర్ణ చన్ద్ర = పూర్ణ చంద్రుని ; ప్రతిమం = ప్రతిమ; పద్మ పత్ర = కమలము లాంటి; నిభేక్షణమ్= కన్నులు కలిగినట్టి; రామస్య = శ్రీరాముని ; ధ్యాయతీ = ధ్యానంలో; వక్త్రం =ముఖం; ప౦చత్వం = మరణం ; కృపణా = దైన్యముతో; గతా =పొంది ఉండాలి;
నిండు జాబిల్లిని ప్రతిబింబం, కమలము లాంటి కన్నులు కలిగిన శ్రీరామచంద్రుని ముఖమును ధ్యానించుచు ఆ సాధ్వి దైన్యముతో అసువులు బాసెనేమో! – 13.13

విలప్య బహు వైదేహీ - న్యస్త దేహా భవిష్యతి
హా రామ లక్ష్మణ = హా రామ! హా లక్ష్మణ ; ఇతి ఏవం = అని ; హా అయోధ్య చ ఇతి= హా అయోధ్య అని ; మైథిలీ = సీతామాత; విలప్య బహు = విలపించుచు ; వైదేహీ = విదేహరాజ్య పుత్రిక ; దేహా = తన శరీరాన్ని ; న్యస్త భవిష్యతి= విడిచి పెట్టి ఉండవచ్చు

अथवा निहिता मन्ये रावणस्य निवेशने

లేదా ఆమె రావణ భవనమున ఎచ్చటనో దాటిపెట్టబడెనేమో. రావణుని నిద్బంధము నుండి పంజరమునందలి గోరువంకవలె ఆమె విలపించుచుండెను కాబోలు – 13.15

विनष्टा वा प्रनष्टा वा मृता वा जनकात्मजा
रामस्य प्रिय भार्यस्य न निवेदयितुम् क्षमम् 		వినష్టా వా ప్రనష్టా వా - మృతా వా జనకాత్మజా
