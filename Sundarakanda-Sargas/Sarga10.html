तत्र दिव्योपमं मुख्यम् स्फाटिकम् रत्नभूषितम्
अवेक्षमाणो हनुमान् ददर्श शयनासनम् 		తత్ర దివ్యోపమం ముఖ్యం - స్ఫాటికం రత్న భూషితమ్
అవేక్షమాణో హనుమాన్ - దదర్శ శయనాసనమ్
తత్ర = అక్కడ; దివ్యోపమం = దివ్యమైన ; ముఖ్యం = ప్రధానమై ; స్ఫాటికం = స్ఫటికముతో; రత్న = రత్నములతో; భూషితమ్= అలంకరించబడి; అవేక్షమాణో = అన్వేషించ్చుచు; హనుమాన్ = హనుమ; దదర్శ = చూసెను ; శయన ఆసనమ్=శయనాగారము;
సీతామాతని అన్వేషించ్చుచు ఆంజనేయుడు, ఇంద్రుని శయనాగారమును పోలి, ప్రధానమై, స్ఫటిక నిర్మితమై, నానారత్నాలంకృతమై వెలయు రావణుని శయన గృహమును చూచెను – 10.1

दान्तकाञ्चनिचित्राङ्गेर्वैश्च वरासनैः
महार्हस्तरणोपेतैरुपपन्नम् महाधनैः 		దాంత కాంచన చిత్రాంగై: - వైడూర్యైశ్చ వరాసనై:
మహార్హ అస్తరణ ఉపేతై - ఉపపన్నం మహా ధనైః
దాంత = దంతములతో; కాంచన= బంగారముతో ;చిత్రాంగై: = చిత్రించిన; వైడూర్యైశ్చ = వైడుర్యములతో; వరాసనై: = వేదికలు; మహార్హ = గొప్ప; అస్తరణ =కంబళములతో / పరుపులతో ; ఉపేతై = కప్పబడి; ఉపపన్నం = ఎక్కువ; మహా ధనైః = విలువ గల;
దంత కనకశిల్పములచే అద్భుతాంగములు కలవై, వైడుర్యమణి ఖచితములై, ఎక్కువ విలువ కలవి, గొప్ప కంబళములతో కప్పబడి వెలయుచున్న ఉత్తమాసనములచే ఆ శయ్యాగారము సుందరమై ఉండెను – 10.2

तस्य च एकतमे देशे सोऽग्य्रमाल्य विभूषितम्
ददर्श पाण्डुरम् चत्रम् ताराधिपति सम्निभम् 		తస్య చ ఏకతమే దేశే - సోగ్ర్యమాల్య విభూషితమ్
దదర్శ పాణ్డురం ఛత్రం - తారాధిపతి సన్నిభమ్
తస్య = అక్కడ ; ఏకతమే దేశే = ఒక ప్రదేశమున ; సోగ్ర్యమాల్య = ఉత్తమ పూల దండలతో; విభూషితమ్ = అలంకరించారు;దదర్శ = చూసెను; పాణ్డురం = తెలుపు రంగులో; ఛత్రం = గొడుగు; తారాధిపతి = చంద్రుడు ; సన్నిభమ్ = పోలిన
అందొకానొక ప్రదేశమున ఉత్తమ హారములచే అలంకృతమును, చంద్రబింబ సదృశమగు ఒక శ్వేతచ్ఛత్రమును మారుతి చూచెను – 10.3

जातरूप परिक्षिप्तम् चित्रभानु समप्रभम्
अशोक मालाविततम् ददर्श परमासनम् 		జాత రూప పరిక్షిప్తం - చిత్ర భాను సమ ప్రభం
అశోక మాలా వితతం - దదర్శ పరమాసనం
జాత రూప పరిక్షిప్తం = బంగారంతో తయారు చేయబడినవి; చిత్ర భాను=అగ్నితో ; సమ ప్రభం = సమానము; అశోక= అశోక; మాలా = పుష్పమాలచే ;వితతం = సమలంకృతమైన ; దదర్శ =చూసెను ;పరమాసనం = అద్భుతమైన ఆసనం;
సువర్ణ వినిర్మిత్రము, అగ్నివలె ఉజ్జ్వలము, అశోక పుష్పమాలచే సమలంకృతమైన ఒక ఉత్తమ శయనమును అతడు చూచెను – 10.4

वाल व्यजन हस्ताभिर् वीज्यमानम् समन्ततः
गन्धैश्च विविधैर् जुष्टम् वर धूपेन धूपितम्		 వాల వ్యజన హస్తాభిర్ - వీజ్యమానం సమన్తతః
గన్ధైశ్చ వివిధై: జుష్టం - వరధూపేన ధూపితమ్
వాల వ్యజన = చామరములను; హస్తాభిర్ = చేతులందు ధరించి; వీజ్యమానం = విసురుచుండిరి ; సమన్తతః = అన్ని దిశలలో; గన్ధైశ్చ = గంధము; వివిధై: జుష్టం = వివిధ సువాసనలు; వరధూపేన= ఉత్తమమగు ధూపము; ధూపితమ్ = వ్యాపించుచుండెను
సుందరులు కొందరు చామరములను ధరించి విసురుచుండిరి. వివిధములగు సుగంధముల పరిమళము గుబాళించుచుండెను. ఉత్తమమగు ధూపధూమము అంతటను వ్యాపించుచుండెను – 10.5

परमास्तरण आस्तीर्णम् आविकाजिन संवृतम्
दामभिर् वर माल्यानाम् समन्ताद् उपशोभितम् 		పరమాస్తరణా ఆస్తీర్ణమ్ - ఆవికాజిన సంవృతమ్
దామభిర్ వర మాల్యానాం - సమన్తాత్ ఉపశోభితమ్
పరమ= సుందరములగు; ఆస్తరణా= ఆస్తరణములచే; ఆస్తీర్ణమ్= కప్పబడి; ఆవికాజిన= మృదువైన గొర్రె చర్మం; సంవృతమ్= అలంకృతమై; దామభిర్= వరుసగా; వర మాల్యానాం = అద్భుతమైన దండలు ;సమన్తాత్ = అంతటను; ఉపశోభితమ్= శోభించుచుండెను
సుందరములగు ఆస్తరణములచే కప్పబడి, మృదుతమములగు సజీనములచే అలంకృతమై, పూలమాలలచే అంతటను అలంకరింపబడి అది శోభించుచుండెను – 10.6

तस्मिन् जीमूत सम्काशम् प्रदीप्तोत्तम कुण्डलम्
लोहिताक्षम् महा बाहुम् महा रजत वाससम् 		తస్మిన్ జీమూత సంకాశం - ప్రదీప్తోత్తమ కుణ్డలమ్
లోహితాక్షం మహాబాహుం - మహా రజత వాససం
(హనుమ రావణుడు చూశాడు) తస్మిన్ = అతడు; జీమూత సంకాశం = కాలమేఘ శరీరుడగు; ప్రదీప్తోత్తమ = ఒక ప్రకాశవంతమైన; కుణ్డలమ్ =చెవిపోగులు ధరించి; లోహితాక్షం = ఎర్రని కళ్ళు; మహాబాహుం = దీర్ఘ బాహువులతో; మహా రజత = బంగారం బట్టలు; వాససం =ధరించి
ఆ శయ్యపై కాలమేఘ సదృశ శరీరుడగు రావణుడు శయనించి ఉండెను. మెరిసేటి కుండలముతోను, ఎర్రని కన్నులతోను, దీర్ఘములగు బాహువులతోను, కనకమయ చేలముతోను అతడు కానవచ్చెను – 10.7

लोहितेननुलिप्ताङ्गम् चन्दनेन सुगन्धिना
संध्या रक्तम् इवाकाशे तोयदम् सतटिद्गणम् 		లోహితేనానులిప్తా౦గం - చన్దనేన సుగన్ధినా
సంధ్యారక్తమ్ ఇవాకాశే - తోయదం సతటిద్గణమ్
అనులిప్తా౦గం = శరీరమంతట అలదుకొని; లోహితేన = ఎరుపు; చన్దనేన = చందనముతో; సుగంధినా = సువాసనతో; సంధ్యారక్తమ్ = సంధ్యా సమయములో ఎర్రటి; ఇవ ఆకాశే తోయదం = అతడు ఆకాశమున; సతటిద్గణమ్ = మెరుపు యొక్క సమూహాలతో
అతడు పరిమళించుచున్న రక్త చందనమును శరీరమంతట అలదుకొని ఉండెను. అందువలన అతడు ఆకాశమున సంధ్యారాగముచే ఎర్రని మెరుపులతో కూడిన మేఘమువలె ప్రకాశించుచుండెను – 10.8

वृतम् आभरणैर् दिव्यैः सुरूपम् काम रूपिणम्
सवृक्ष वन गुल्माढ्यम् प्रसुप्तम् इव मन्दरम् 		వృతమ్ ఆభరణై: దివ్యైః - సురూపం కామ రూపిణమ్
సవృక్ష వన గుల్మాఢ్యం - ప్రసుప్తమ్ ఇవ మన్దరమ్
వృతమ్ = ధరించి; ఆభరణై: దివ్యైః = అద్భుతమైన ఆభరణాలతో; సురూపం = మంచి రూపముతో; కామ రూపిణమ్ = కామరూపుడై; సవృక్ష వన = వృక్ష వనము ; గుల్మాఢ్యం = వృక్షములతోను, పొదలతోను; ప్రసుప్తమ్ = నిశ్చలముగా/నిద్రలో; ఇవ మన్దరమ్ = మందరగిరివలె;
ఆ రావణుడు దివ్యభరణములు దాల్చి సురూపుడై, కామరూపుడై, వృక్షములతోను, పొదలతోను నిండిన మందర పర్వతమువలె నిశ్చలముగా నిద్రించుచున్నాడా? అనినట్లు ఉండెను – 10.9

क्रीडित्वोपरतम् रात्रौ वराभरण भूषितम्
प्रियम् राक्षस कन्यानाम् राक्षसानाम् सुखावहम् 		క్రీడిత్వోపరతం రాత్రౌ - వరాభరణ భూషితమ్
ప్రియం రాక్షస కన్యానాం - రాక్షసానాం సుఖావహమ్
క్రీడిత్వ = క్రీడలు సలిపి తరువాత; ఉపరతం = విశ్రాంతి; రాత్రౌ = రాత్రివేళ; వరాభరణ= ఉత్తమ ఆభరణాలతో; భూషితమ్ = ధరించి; ప్రియం = ప్రియుడై; రాక్షస కన్యానాం = రాక్షస కన్యలకు; రాక్షసానాం = రాక్షసులకు ;సుఖావహమ్ = సౌఖ్యప్రదుడైన
రాత్రి రత్యాది వివిధ క్రీడలు సలిపి ఉపరుతుడై, ఉత్తమ ఆభరణములను ధరించి రాక్షసాంగనలకు ప్రియుడై, దానవులకు సౌఖ్యప్రదుడైన రావణుని మారుతి చూచెను – 10.10

पीत्वापि उपरतम् चापि ददर्श स महा कपिः
भास्करे शयने वीरम् प्रसुप्तम् राक्षसाधिपम् 		పీత్వాపి ఉపరతం చాపి - దదర్శ స మహా కపిః
