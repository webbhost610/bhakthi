स निर्जत्य पुरीम् लण्का श्रेष्ठाम् ताम् कामरूपिणीम्
विक्रमेण महातेजा हनुमान् कपिसत्तमः 		స నిర్జిత్య పురీం శ్రేష్ఠాం లంకాం తాం కామరూపిణీం
విక్రమేణ మహాతేజా హనూమాన్ కపిసత్తమ:
నిర్జిత్య = జయించడం; పురీం = నగరాన్ని; శ్రేష్ఠాం = శ్రేష్ఠమగు; తాం లంకాం = లంకా రాక్షసిని; కామరూపిణీం = కామరూపిణి; విక్రమేణ = పరాక్రమముచే; మహాతేజా = మహాతేజస్వి; స : హనూమాన్ = ఆ హనుమ; కపిసత్తమ: = వానరులలో ఉత్తమమైనవాడు;
కామరూపము దాల్చు పురశ్రేష్ఠమగు ఆ లంకా రాక్షాసిని పరాక్రమముచే మహాతేజస్వియగు ఆ హనుమ, కపీంద్రుడు జయించెను – 4.1

अद्वारेण च महाबाहुः प्राकारमभिपुप्लुवे
प्रविश्य नगरीम् लङ्काम् कपिराजहितम्करः 		అద్వారేణ మహాబాహు: ప్రాకారమ్ అభిపుప్లువే
ప్రవిశ్య నగరీం లంకాం కపిరాజ హితంకర:
అద్వారేణ = ద్వారము నుండి కాక; మహాబాహు: = మహాబాహు; ప్రాకార = ప్రాకారం/కోట గోడ; అభిపుప్లువే = దూకడం; ప్రవిశ్య = ప్రవేశించెను; నగరీం లంకాం = లంకా నగరం; కపిరాజ = కపిరాజు (సుగ్రీవుడు); హితం కర: = హితము కోరి;
అధిక సత్వమును కలిగిన శత్రునగరము అగుటవలన ద్వారము నుండి పగటిపూట ప్రవేశింపక రాత్రి సమయమున ప్రాకారమును, సుగ్రీవ హితకాముడై, మహాబాహుడైన హనుమ లంకను ప్రవేశించెను – 4.2

चक्रेणाऽथ पदम् सव्यम् शत्रूणाम् स तु मूर्धनि
प्रविष्टः सत्त्वसंपन्नो विशायाम् मारुतात्मजः		 చక్రేథ పాదం సవ్యం చ శత్రూణాం స తు మూర్ధని
ప్రవిష్టః సత్త్వ సంపన్నో నిశాయాం మారుతాత్మజః
అథ= మరియు; చక్రే = పెట్టెను; సవ్యం చ పాదం = ఎడమ పాదం; శత్రూణాం = శత్రువుల; మూర్ధని = తలపై; ప్రవిష్టః = ప్రవేశించారు; సత్త్వ సంపన్నో = గొప్ప బలంతో/ సత్వసంపన్నుడైన; నిశాయాం = రాత్రివేళ; మారుతాత్మజః = వాయు కుమారుడు;
హనుమంతుడు మొదట ఎడమ పాదమును పెట్టెను. అది శత్రువు తలపై పాదము పెట్టినట్లుండెను. సత్వసంపన్నుడైన వాయు కుమారుడు రాత్రి సమయము నందే లంకలో ప్రవేశించెను – 4.3

स महापथमास्थाय मुक्तापुष्पविराजितम्
ततस्तु ताम् पुरीम् लङ्काम् रम्यामभिययौ कपिः 		స మహాపథమ్ ఆస్థాయ ముక్తా పుష్ప విరాజితమ్
తతస్తు తాం పురీం లంకాం రమ్యం అభియయౌ కపి:
మహాపథమ్ = రాజమార్గము; ఆస్థాయ = చేరి; ముక్తా పుష్ప = ముత్యాలు మరియు పుష్పాలతో ; విరాజితం = విరాజిల్లు; తత = అప్పుడు; తాం పురీం లంకాం = అందమైన లంకా నగరం వైపు; రమ్యం= రమ్యమైన; అభియయౌ = పోసాగెను/ వెళ్ళెను; స: కపి: = హనుమ
ముత్యముల చేతను పుష్పముల చేతను శోభించు ఒక రాజమార్గమును చేరి అతడు శస్త్రహస్తులు, భయంకరులు అయిన దానవులతో నిండియున్న మనోజ్ఞమగు లంకయందు పోసాగెను – 4.4

हसित उद्घुष्ट निनदैः तूर्य घोष पुरः सरैः
वज्र अन्कुश निकाशैः च वज्र जाल विभूषितैः 		హసితోద్ఘుష్ట నినదై: తూర్య ఘోష పురస్సరైః
వజ్రా౦కుశ నికాశైశ్చ వజ్ర జాల విభూషితైః
హసితోద్ఘుష్ట = పెద్ద నవ్వులతో ; నినదై:=నినాదాలు; తూర్య ఘోష = సంగీత వాయిద్యాల ; పురస్సరైః = ధ్వనులతో; వజ్ర = వజ్రము; అంకుశ = అంకుశము; నికాశైశ్చ = మిక్కిలి కఠినములగు; వజ్ర జాల = వజ్రములతో నిర్మింపబడిన జలకములు ; విభూషితైః = శోభించుచుండెను
పకపక నవ్వులు ధ్వనులు సంగీత వాయిద్యాల ఘోషద్వనులు అచట సంకులముగా ఉండెను. వజ్రాంకుశమును పోలిన మిక్కిలి కఠినములగు వజ్రములతో నిర్మింపబడిన జలకములతో శోభించుచుండెను – 4.5

गृह मेधैः पुरी रम्या बभासे द्यौः इव अम्बुदैः
प्रजज्वाल तदा लन्का रक्षः गण गृहैः शुभैः 		గృహ మేఘై: పురీ రమ్యా బభాసే ద్యౌరివ అంబుదై:
ప్రజజ్వాల తదా లంకా రక్షోగణ గృహైః శుభైః
గృహ మేఘై: = ఆకాశమును తాకు గృహములు; పురీ = నగరం; రమ్యా = రమ్యమైన; బభాసే =శోభించు; ద్యౌ: = ఆకాశమునందలి ; ఇవ అంబుదై: = మేఘమువలె; ప్రజజ్వాల = ప్రకాశించింది; తదా = అప్పుడు; లంకా = లంకా నగరం; రక్షోగణ = రాక్షస గణం; గృహైః= గృహాలు; శుభైః = శోభించు;
ఉన్నత గృహములతో అలరారుచు ఆ లంకానగరము ఆకాశమునందలి మేఘమువలె భాసిల్లుచుండెను. రాక్షస భవనములచే ఆ లంకాపురి గగనమువలె శోభించుచుండెను –4.6

सित अभ्र सदृशैः चित्रैः पद्म स्वस्तिक संस्थितैः

తెల్లని మేఘములను పోలి చిత్రములై పద్మ, స్వస్తికాది ఆకారంలో నిర్మితమై విస్తరించియున్న వర్ధమాన గృహములచే ఆ లంక అంతా అలంకృతమై ఉండెను – 4.7

ताम् चित्र माल्य आभरणाम् कपि राज हितम् करः
राघव अर्थम् चरन् श्रीमान् ददर्श च ननन्द च 		తాం చిత్ర మాల్యాభరణాం కపిరాజ హితం కరః
రాఘవార్థం చరన్ శ్రీమాన్ దదర్శ చ ననంద చ
తాం = ఆ లంకయందు; చిత్ర = చిత్రమైన ; మాల్యాభరణాం = మాలికలు, భూషణముల చేతను; కపిరాజు = సుగ్రీవుడు; హితం కరః = హితకరుడై; రాఘవార్థం= రామకార్యార్ధము; చరన్ = చరించు; శ్రీమాన్= హనుమ; దదర్శ= చూసెను; ననంద చ = ఆనందించెను
చిత్రమైన మాలికల చేతను, భూషణముల చేతను, రమ్యమైన ఆ లంకయందు సుగ్రీవ హితకరుడై, రామకార్యార్ధము చరించు హనుమంతుడు ఆ దృశ్యములను చూచుచు, తన కార్యము ఫలోన్ముఖమగుచుండెనని ఆనందించెను – 4.8

भवनाद्भवनं गच्छ्न् ददर्श पवनात्मजः

शुश्राव मधुरम् गीतम् त्रि स्थान स्वर भूषितम्

స్త్రీణాం మద సమృద్ధానాం దివి చాప్సరసామ్ ఇవ

స్వర్గమందలి అప్సరస స్త్రీలను పోలిన మదవతులైన స్త్రీలు (వక్షము,కంఠము,శిరము అను స్ధానములను బట్టి) మంద్ర మధ్యమ తార స్వరములతో మధురముగా పాడు గీతములను వినెను – 4.10

शुश्राव कान्ची निनदम् नूपुराणाम् च निह्स्वनम्

సోపాన నినదాం శ్చైవ భవనేషు మహాత్మనామ్
శుశ్రావ = శ్రావ్యమైన (వినెను) ; కాంచీ = నడుము చుట్టూ ధరించే ఆభరణాలు; నినాదం =ధ్వని; నూపురాణాం= కాళ్ళ గజ్జెలు; నిస్వనన్= ధ్వని; సోపన = సోపానములపై (మెట్లు) ; నినాదం చ ఇవ = అడుగుల చప్పుళ్ళు; భవనేషు = ఇళ్లలో; మహాత్మనామ్= సంపన్న;
గొప్ప దానవుల గృహములందు స్త్రీలు నడచునప్పుడు బంగారు మొలనూళ్ళ ధ్వనిని, నూపుర నినాదములను, సోపానములపై (మెట్లు) నడుచునప్పుడు స్వనములను మారుతి వినెను – 4.11

आस्फोटित निनादामः च क्ष्वेडितामः च ततः ततः
शुश्राव जपताम् तत्र मन्त्रन् रक्षोगृहेषु वै 		ఆస్ఫోటిత నినాదాంశ్చ క్ష్వేళితాంశ్చ తత: తత:
శుశ్రావ జపతాం తత్ర మంత్రాన్ రక్షో గృహేషు వై
ఆస్ఫోటిత= చేతులు చప్పట్లు; నినాదాంశ్చ=ధ్వని; క్ష్వేళితాంశ్చ = భటుల సింహనాదములను; తత: తత: = అక్కడక్కడా; శుశ్రావ = శ్రావ్యమైన (వినెను) ; జపతాం= జపించువారి; తత్ర = అక్కడ ; మంత్రాన్ = మంత్రము; రక్షో = రాక్షసుల ; గృహేషు వై = గృహములలో
అంతకంతకు కరములను దట్టుటవలన కలుగు ధ్వనులను, భటుల సింహనాదములను వినెను. ఆ దానవ మందిరములందు జపించువారి జపమంత్రమును కూడా వినెను – 4.12

स्वाध्याय निरतामः चैव यातु धानान् ददर्श सः
रावण स्तव सम्युक्तान् गर्जतः राक्षसान् अपि 		స్వాధ్యాయ నిరతాం శ్చైవ యాతుధానాన్ దదర్శ సః
రావణస్తవ సంయుక్తాన్ గర్జతో రాక్షసాన్ అపి
స్వాధ్యాయ= వేదాధ్యయన; నిరతాంశ్చైవ= నిరతులైన వారు; యాతుధానాన్= యాతుధానులను; దదర్శ= చూసెను; రావణస్తవ = రావణుని స్తుతించేవారు; సంయుక్తాన్= సంయుక్తముగా; గర్జతో= గర్జిస్తున్న; రాక్షసాన్= రాక్షసులను; అపి = ఇంకా/మరియు ;
వేదాధ్యయన నిరతులైన రాక్షసులను, రావణునిపై స్తోత్రపాఠములను సమ్యుక్తముగా చదువుచున్న యాతుధానులను, మరియు గర్జిస్తున్న రాక్షసులను హనుమ చూసెను – 4.13

राज मार्गम् समावृत्य स्थितम् रक्षः बलम् महत्
ददर्श मध्यमे गुल्मे राक्षसस्य चरान् बहून् 		రాజ మార్గం సమావృత్య స్థితం రక్షో బలం మహత్
దదర్శ మధ్యమే గుల్మే రాక్షసస్య చరాన్ బహూన్
రాజ మార్గం = రాజమార్గము; సమావృత్య = సమావేశమై; స్థితం = ఉన్నారు ; రక్షో = రాక్షస; బలం మహత్ = మహాసేనను; దదర్శ = చూసెను; మధ్యమే గుల్మే = మధ్య మార్గమున ; రాక్షసస్య = రాక్షసులు ; చరాన్ = చరిస్తున్నారు/నడుస్తున్నారు; బహూన్ = అనేక
రాజమార్గమును ఆవరించియున్న రాక్షస మహాసేనను, సేనారక్షక మధ్య మార్గమున చాలామంది చారులను అతడు చూసెను – 4.14

दीक्षितान् जटिलान् मुण्डान् गः अजिन अम्बर वाससः
दर्भ मुष्टि प्रहरणान् अग्नि कुण्ड आयुधामः तथा 		దీక్షితాన్ జటిలాన్ ముణ్డాన్ గోజినామ్బర వాససః
దర్భ ముష్టి ప్రహరణాన్ అగ్ని కుణ్డా యుధాం స్తథా
దీక్షితాన్ = దీక్షబూనిన వారిని; జటిలాన్ = జడధారలను; ముణ్డాన్ = (గుండు) ముండిత మస్తకులను; గోజినామ్బర= ఎద్దు చర్మము; వాససః = ధరించిన వారు; దర్భ ముష్టి= దర్భ ; ప్రహరణాన్= అరిష్ఠ నివృత్తి చేయు తాంత్రికులని; అగ్ని కుణ్డ= అగ్నికుండములను; ఆయుధాం = ఆయుధములుగా ; స్తథా = కలవారిని ;
ముందు ముందునకు పోవుచు ఆ మహానుభావుడు, దీక్షబూనిన వారిని, జడధారలు ఉన్న వారిని, ముండిత (గుండు) మస్తకులను, వృషభ చర్మధారులను, దర్భముష్ఠితో కొట్టి అరిష్ఠ నివృత్తి చేయు తాంత్రికులని, అగ్నికుండములను ఆయుధములుగా కలవారిని వీక్షించెను – 4.15

कूट मुद्गर पाणीमः च दण्ड आयुध धरान् अपि
एक अक्ष अनेक कर्णामः च चलल् लम्ब पयः धरान् 		కూట ముద్గర పాణీంశ్చ దణ్డాయుధ ధరాన్ అపి
ఏకాక్షానేక కర్ణాంశ్చ లమ్బోదర పయోధరాన్
కూట ముద్గర = సుత్తులు మరియు ఇనుము వస్తువులు ; పాణీంశ్చ= చేతబూనినవారు; దణ్డాయుధ = దండించే ఆయుధాలు ; ధరాన్ అపి = ధరించినవారు ; ఏకాక్షానేక=ఒక్కకన్ను కలవారు ; కర్ణాంశ్చ= ఒక చెవి కలవారు; లమ్బోదర=వ్రేలాడు ఉదరము; పయోధరాన్ = మరియు స్తనములు కలవారిని;
కూటముద్గరములను ఆయుధములుగా ధరించినవారిని, దండాయుధము దాల్చినవారిని, ఒక్కకన్ను కలవారిని, ఒక చెవి కలవారిని, వ్రేలాడు ఉదరము కలవారిని, స్తనములు కలవారిని చూచెను – 4.16

करालान् भुग्न वक्त्रामः च विकटान् वामनामः तथा
धन्विनः खड्गिनः चैव शतघ्नी मुसल आयुधान् 		కరాళాన్ భుగ్నవక్త్రాంశ్చ వికటాన్ వామనాం స్తథా
ధన్వినః ఖడ్గిన శ్చైవ శతఘ్నీ ముసలాయుధాన్
కరాళాన్ = భయంకరముగా కనిపించేవారు; భుగ్నవక్త్రాంశ్చ = వంకర ముఖము కలవారు; వికటాన్ = వికృతాకారులు; వామనాం స్తథా =మరుగుజ్జులు (పొట్టిగా) ; ధన్వినః = ధనస్సును; ఖడ్గిన శ్చైవ = కత్తులు ధరించినవారు; శతఘ్నీ = శతఘ్నులు ; ముసలాయుధాన్ = ముసల ఆయుధములను (దేనినైనా పొడి చేయగల ఆయుధము) ధరించినవారు;
భయంకర ఆకారులను, వంకర ముఖము కలవారిని, వికృతాకారులను, మరుగుజ్జులను, ధనస్సును, ఖడ్గము ధరించినవారిని, శతఘ్నీ, ముసల ఆయుధములను ధరించినవారిని మారుతి చూచెను – 4.17

परिघ उत्तम हस्तामः च विचित्र कवच उज्ज्वलान्
नातिस्थूलान् नातिकृशान् नातिदीर्घ अतिह्रस्वकान् 		పరిఘోత్తమ హస్తాంశ్చ విచిత్ర కవచోజ్జ్వలాన్
నాతి స్థూలాన్ నాతికృశాన్ నాతిదీర్ఘాతి హ్రస్వకాన్
పరిఘోత్తమ = పరిఘను; హస్తాంశ్చ = చేతబట్టిన వారిని; విచిత్ర = విచిత్రములైన; కవచోజ్జ్వలాన్ = కవచములచే మెరయు వారిని; నాతి స్థూలాన్ =అతిస్ధూల శరీరము లేనివారు; నాతికృశాన్ = మిక్కిలి కృశింపనివారిని; నాతిదీర్ఘాతి = అతి పొడుగులేని వారు, హ్రస్వకాన్ = అతి పొట్టివారు కాని;
పరిఘను చేతబట్టిన వారిని, విచిత్రములైన కవచములచే మెరయు వారిని, అతిస్ధూల శరీరములు లేనివారు, మిక్కిలి కృశింపనివారిని, మరియు అత్యున్నతులు కాని వారిని, అతివామనులు కాని వారిని చూచెను – 4.18

नातिगौरान्नातिकृष्णान्नातिकुब्जान्न वामनान्
विरूपान् बहु रूपामः च सुरूपामः च सुवर्चसः 		నాతి గౌరాన్ నాతి కృష్ణాన్ నాతికుబ్జాన్ న వామనాన్
విరూపాన్ బహురూపాంశ్చ సురూపాంశ్చ సువర్చసః
నాతి గౌరాన్ = గౌరవర్ణులు కారు; నాతి కృష్ణాన్ = నల్లనివారు కారు; నాతికుబ్జాన్ = అతికుబ్జులు కారు; న వామనాన్ = వామనులు (పొట్టి) వా కారు; విరూపాన్ = వికృతరూపులు; బహురూపాంశ్చ = బహురూపధారులు; సురూపాంశ్చ = మంచి రూపాన్ని కలిగి ఉన్నవారు; సువర్చసః = మంచి వర్చస్సు కలవారిని;
కొందరు ఆద్యంతము గౌరవర్ణులు కారు. కొందరు అత్యంత కృష్ణవర్ణులు (నల్లనివారు) కారు. మరికొందరు అతికుబ్జులు, అతివామనులు కాకయుండిరి. అంతేకాక వికృతరూపులను, బహురూపధారులను, సురూపులను, మంచి వర్చస్సు కలవారిని మారుతి చూసెను – 4.19

ध्वजीन् पताकिनश्चैव ददर्श विविधायुधान्
शक्ति वृक्ष आयुधामः चैव पट्टिश अशनि धारिणः 		ధ్వజీన్ పతాకిన శ్చైవ దదర్శ వివిధాయుధాన్
శక్తి వృక్షాయుధాం శ్చైవ పట్టిసాశని ధారిణః
ధ్వజీన్ = ధ్వజములను; పతాకిన శ్చైవ = పతాకములను పట్టియు; దదర్శ= చూసెను; వివిధాయుధాన్ = వివిధ రకములైన ఆయుధములను ; శక్తి= శక్త్యాయుధములను ; వృక్షాయుధాం శ్చైవ = వృక్షాయుధములను; పట్టిసాశని= పట్టాసము అశనిని; ధారిణః = ధరించిరి
