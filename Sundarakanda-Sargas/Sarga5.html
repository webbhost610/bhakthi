హనుమ ప్రతి గృహము నందు సీతామాతను వెదకుట - ఆమె కానక దు:ఖించుట

ततः स मध्यंगतमंशुमन्तं
ज्योत्स्नावितानं महदुद्वमन्तम्
ददर्श धीमान् दिवि भानुमन्तं
गोष्ठे वृषं मत्तमिव भ्रमन्तम् 		తత: స మధ్యం గతం అంశుమన్తం
జ్యోత్స్నా వితానం మహత్ ఉద్వమన్తమ్
దదర్శ ధీమాన్ దివి భానుమన్తం
గోష్ఠే వృషం మత్తమివ భ్రమంతం
తత: = అంత; మధ్యం గతం= ఆకాశంలో మధ్యలో ఉన్న;అంశుమన్తం =చంద్రుని; జ్యోత్స్నా వితానం = చంద్రబింబము; మహత్ = అధిక; ఉద్వమన్తమ్ = వ్యాప్తి; దదర్శ = చూచెను; ధీమాన్ = ధీమంతుడు; దివి = ఆకాశంలో; భానుమన్తం = కాంతివంతమై వెన్నెల; గోష్ఠే =ఆవులు; వృషం =మధ్య; మత్తం ఇవ = మదించిన ఆంబోతువలె ; భ్రమంతం = భ్రమించడం/చుట్టు తిరగడం
అంత హనుమంతుడు ఆకాశమున చంద్రుని చూచెను. చంద్రబింబము ఆకాశ మధ్యమునుండి అధిక కాంతివంతమై వెన్నెలను విస్తారముగా వ్యాపింప చేయుచుండెను. చంద్రుడు గోగణ (ఆవులు ) మధ్య భాగమున చరించు మదించిన ఆంబోతువలె ఉండెను – 5.1

लोकस्य पापानि विनाशयन्तं
महोदधिं चापि समेधयन्तम्
भूतानि सर्वाणि विराजयन्तं
ददर्श शीतांशुमथाभियान्तम् 		లోకస్య పాపాని వినాశయన్తం
మహోదధిం చాపి సమేధయన్తమ్
భూతాని సర్వాణి విరాజయన్తం
దదర్శ శీతాంశుమ్ అథాభియాంతం
లోకస్య= లోకుల యొక్క; పాపాని = పాపాలు; వినాశయన్తం = నాశనం; మహోదధిం = మహాసముద్రం; సమేధయన్తమ్ చ అపి = మరియు ఉప్పొంగచేయు; సర్వాణి భూతాని = పంచభూతముల నెల్ల ; విరాజయన్తం = ప్రకాశింపజేయు; దదర్శ = చూచెను; శీతాంశుమ్ = చంద్రుడు; అథ: = ఆ తరువాత; అభియాంతం = తిరుగుతున్న;
ఆ చంద్రుడు లోకుల యొక్క తాపరూపమైన పాపమును పోగొట్టుచు మహాసాగరము ఉప్పొంగచేయు పంచభూతముల నెల్ల ప్రకాశింపజేయు చుండెను. అట్లాకాశమున తిరుగుతున్న చంద్రుని మారుతి చూచెను – 5.2

या भाति लक्ष्मीर्भुवि मन्दरस्था
तथा प्रदोषेषु च सागरस्था
तथैव तोयेषु च पुष्करस्था
रराज सा चारुनिशाकरस्था 		యా భాతి లక్ష్మీర్భువి మందరస్థా
తథా ప్రదోషేషు చ సాగరస్థా
తథైవ తోయేషు చ పుష్కరస్థా
రరాజ సా చారు నిశాకరస్థా
యా లక్ష్మీ: = ఏ కాంతియుండునో ; భువి = భూమిపై; మందరస్థా = మందర పర్వతంపై; తథా = అదే విధంగా; ప్రదోషేషు = సాయంత్రం సమయంలో; సాగరస్థా = మహా సముద్రం; తథైవ = ఎట్టి తేజస్సుండునో; తోయేషు = నీటిలో ; పుష్కరస్థా = కమలములలో ; రరాజ = ఏ శోభయుండునో; సా = అదే శోభ; చారు నిశాకరస్థా =అందమైన చంద్రుని మీద;
ధరిత్రి యందు మందరగిరి నందు ఏ కాంతియుండునో, ప్రదోషకాలమున సముద్రమున ఎట్టి తేజస్సుండునో, అట్లే జలమందలి కమలములలో ఏ శోభయుండునో, అట్టి కాంతి చంద్రుని యందు మహోజ్జ్వలముగా గోచరించు చుండెను – 5.3

हंसो यथा राजतपञ्जरस्थः
सिंहो यथा मन्दरकन्दरस्थः
वीरो यथा गर्वितकुञ्जरस्थ
चन्द्रोऽपि बभ्राज तथाम्बरस्थः 		హంసో యథా రాజత ప౦జరస్థః
సింహో యథా మందరకందరస్ధ:
వీరో యథా గర్విత కుఞ్జరస్థ:
చన్ద్రోపి బభ్రాజ తథామ్బరస్థః
హంసో యథా =హంసవలె; రాజత = వెండి; ప౦జరస్థః = పంజరం; సింహో యథా =సింహమువలె; మందరకందరస్ధ: = మందర పర్వతం యొక్క గుహలో; వీరో యథా =మహావీరునివలె; గర్విత= గర్వం; కుఞ్జరస్థ: = ఏనుగు; తథా = అదే విధంగా; చన్ద్రోపి = చంద్రుడు; బభ్రాజ= ప్రకాశించెను; తథా అంబరస్థః = ఆ ఆకాశంలో;
వెండిపంజరమున ఉన్న తెల్లని హంసవలెను, మందర గుహయందలి సింహమువలెను, ఏనుగుపై ఉన్న మహావీరునివలెను చంద్రుడు ఆకాశమున మెరయుచుండెను – 5.4

स्थितः ककुद्मानिव तीक्ष्णशृङ्गो
महाचलः श्वेत इवोच्चशृङ्गः
हस्तीव जाम्बूनदबद्धशृङ्गो
रराज चन्द्रः परिपूर्णशृङ्गः 		స్థితః కకుద్మాన్ ఇవ తీక్ష్ణ శృఙ్గో
మహాచలః శ్వేత ఇవోచ్చ శృఙ్గః
హస్తీవ జామ్బూనద బద్ధ శృఙ్గో
రరాజ చన్ద్రః పరిపూర్ణ శృఙ్గః
చన్ద్రః = చంద్రుడు; పరిపూర్ణ శృఙ్గః = పరిపూర్ణ కళతో; రరాజ = ప్రకాశించడం; కకుద్మాన్ ఇవ = వృషభము (ఎద్దు) వలె; స్థితః = నిలబడి; తీక్ష్ణ శృఙ్గో = వాడికొమ్మలు; మహాచలః = గొప్ప పర్వతము; శ్వేత = తెలుపు రంగులో; ఇవ ఉచ్చ శృఙ్గః= పొడవాటి శిఖరాలతో; హస్తీవ = ఏనుగు వంటి; జామ్బూనద = బంగారు; బద్ధ శృఙ్గో= దంతాలతో
వాడికొమ్మలు కల వృషభము (ఎద్దు) వలెను, మహోన్నత శిఖరములు గల గొప్ప పర్వతము వలెను, బంగారు తొడుగు గల దంతావళము వలెను తెలుపు రంగులో చంద్రుడు పరిపూర్ణ కళతో ప్రకాశించుచుండెను – 5.5

विनष्टशीताम्बुतुषारपङ्को
महाग्रहग्राहविनष्टपङ्कः
प्रकाशलक्ष्म्याश्रयनिर्मलाङ्को
रराज चन्द्रो भगवान् शशाङ्कः 		వినష్ట శీతామ్బు తుషార పంకో
మహాగ్రహ గ్రాహ వినష్ట పంక:
ప్రకాశ లక్ష్మ్యాశ్రయ నిర్మలాంకో
రరాజ చంద్రో భగవాన్ శశాంక:
వినష్ట=నశింప; శీతామ్బు తుషార పంకో = మంచు బిందువులు ; మహాగ్రహ = గ్రహములన్నిటికన్నా; గ్రాహ = అగ్రగణ్యుడు (సూర్యుడు) వల్ల; వినష్ట పంక: = మాలిన్యములు నశింప; ప్రకాశ లక్ష్మీ: = ప్రకాశ లక్ష్మిచే ; ఆశ్రయ = ఆశ్రయము పొంది; నిర్మలాంకో = నిర్మలముగా ; రరాజ = ప్రకాశంగా; భగవన్ = భగవానుడగు; చంద్రో = చంద్రుడు/ శశాంకుడు;
పంకమువంటి తుషార వృష్టిని (చల్లని మంచు బిందువులు ) నశింపచేసి, బృహస్పత్యాదులగు ఇతర మహాగ్రహముల తేజస్సును అణచి దివ్యప్రకాశము తన అంకమునకు శోభను కలిగించుచుండగా భగవానుడగు శశాంకుడు వెలుగుచుండెను – 5.6

शीलातलम् प्राप्य यथा मृगेन्द्रो
महारणम् प्राप्य यथा गजेन्द्रह्
राज्यम् समासाद्य यथा नरेन्द्र
स्तथाप्रकाशो विरराज चन्द्रः 		శిలా తలం ప్రాప్య యథా మృగే౦ద్రో
మహా రణం ప్రాప్య యథా గజేంద్ర:
రాజ్యం సమాసాద్య యథా నరేంద్ర:
తథా ప్రకాశో విరరాజ చంద్ర:
శిలా తలం = సమున్నతమగు శిలను; ప్రాప్య = అధిరోహించి; యథా మృగే౦ద్ర: = సింహం వలె; ప్రాప్య = సంపాదించటం; మహా రణం = రణరంగమందలి; ప్రాప్య = గెలిచిన; యథా గజేంద్ర: = గజరాజు వలె; రాజ్యం = రాజ్యం; సమాసాద్య = పొందడం; యథా నరేంద్ర:= రాజు వలె; తథా = అదే విధంగా; ప్రకాశ: = ప్రకాశించు; విరరాజ = విరాజిల్లెను; చంద్ర: = చంద్రుడు;
సమున్నతమగు శిలను/కొండను అధిరోహించి మృగేంద్రము వలె, రణరంగమందలి గెలిచిన గజరాజు వలెను, రాజ్యమును బడసిన భూపాలుని వలెను చంద్రుడు ప్రకాశించుచుండెను – 5.7

प्रकाशचन्द्रोदयनष्टदोषः
प्रवृत्तरक्षः पिशिताशदोषः
रामाभिरामेरितचित्तदोषः
स्वर्गप्रकाशो भगवान् प्रदोषः 		ప్రకాశ చంద్రోదయ నష్ట దోషః
ప్రవృద్ధ రక్షః పిశితాశదోషః
రామాభి రామేరిత చిత్త దోషః
స్వర్గ ప్రకాశో భగవాన్ ప్రదోష:
ప్రకాశ = ప్రకాశముచే ; చంద్రోదయ = చంద్రోదయము; నష్ట దోషః = అంధకారము పోగొట్టి; ప్రవృద్ధ = ప్రవృద్ది ; రక్షః = రాక్షసులకు ; పిశితాశ = మాంస భక్షణ సమయము; దోషః= సమకూర్చి; రామాభి = కామినుల; రామేరిత = మనస్సుల యందు; చిత్త = కామమును; దోషః= ప్రజ్వలింపచేసి ; స్వర్గ = స్వర్గమును పోలి ;ప్రకాశో= ప్రకాశము; భగవాన్ = చంద్ర భగవానుడు; ప్రదోష: = సాయంసమయము
ఆనాటి ప్రదోష సమయమున సముజ్జ్వల చంద్ర ప్రకాశముచే అంధకారమును పోగొట్టి, రాక్షసులకు మాంస భక్షణ సమయమును సమకూర్చి, కామినుల మనస్సుల యందు కామమును ప్రజ్వలింపచేసి, స్వర్గమును పోలి ఆనందమును కలిగించెను –5. 8

विहर्तुमत्यद्भुतरौद्रवृत्ताः 		తన్త్రీ స్వనాః కర్ణ సుఖాః ప్రవృత్తాః
స్వపన్తి నార్యః పతిభి: సువృత్తాః
నక్తంచరాశ్చాపి తథా ప్రవృత్తా
విహర్తుమ్ అత్యద్భుత రౌద్ర వృత్తాః
తన్త్రీ స్వనాః = వీణాదుల తంత్రులు; కర్ణ సుఖాః = చెవులకు హాయిగా; ప్రవృత్తాః = వినపడెను; స్వపన్తి = నిద్రపోయారు; నార్యః = మహిళలు; పతిభి: = భర్తలతో; సువృత్తాః = మంచి మనస్సుతో; నక్తంచరాశ్చాపి = రాక్షసులు కూడా; ప్రవృత్తా= ప్రవర్తనులై; విహర్తుమ్ = సంచరింపసాగిరి; అత్యద్భుత = చాలా విచిత్రమైన ; రౌద్ర వృత్తాః = క్రూర రూపులై
అప్పుడు వీణాదుల తంత్రులు నిస్వనములు మనోజ్ఞముగా మ్రోగినవి. స్త్రీలు భర్తలతో కూడి సుఖముగా నిద్రించుచుండిరి. రాక్షసులు విచిత్రమైన క్రూర ప్రవర్తనులై సంచరింపసాగిరి – 5.9

मत्तप्रमत्तानि समाकुलानि
तथाश्वभद्रासनसम्कुलानि
वीरः श्रिया चापि समाकुलानि
ददर्श धीमान् स कपिः कुलानि 		మత్త ప్రమత్తాని సమాకులాని
రథాశ్వ భద్రాసన సంకులాని
వీరశ్రియా చాపి సమాకులాని
దదర్శ ధీమాన్ స కపిః కులాని
మత్త ప్రమత్తాని = మదించి మైమరచిన; సమాకులాని= జనులతో నిండి; రథాశ్వ = అశ్వరధములు; భద్రాసన =భద్రాసనములచేత; సంకులాని= కిక్కిరిసి; వీర శ్రియా = సంపద; చాపి = మరియు ; సమాకులాని = నిండిన; దదర్శ = చూచెను; ధీమాన్= ధీమంతుడైన; స కపిః = ఆ హనుమ; కులాని= భవనశ్రేణిని;
మదించి మైమరచిన జనులతో నిండి, రధముల చేతను తురగముల చేతను చక్కని భద్రాసనములచేతను కిక్కిరిసి, వీరలక్ష్మీచే నిండిన నివాస భవనశ్రేణిని హనుమంతుడు చూచెను – 5.10

परस्परं चाधिकमाक्षिपन्ति
भुआंश्च पीनानधिनिक्षिपन्ति
मत्तप्रलापानधिकम् क्षिपन्ति
मत्तानि चान्योन्यमधिक्षिपन्ति		 పరస్పరం చ అధికమ్ ఆక్షిపన్తి
భుజాంశ్చ పీనాన్ అధివిక్షిపన్తి
మత్త ప్రలాపాన్ అధివిక్షిపన్తి
మత్తాని చ అన్యోన్యమ్ అధిక్షిపన్తి
పరస్పరం = ఒకరినొకరు; చ అధికమ్= అధికులము అని; ఆక్షిపన్తి = ఆక్షేపించుకొనుచు ; భుజాంశ్చ = భుజాలు ;పీనాన్= ఒకరిపై ఒకరు ; అధివిక్షిపన్తి = వేసుకొనుచు ; మత్త ప్రలాపాన్ = అసంబద్ధమైన పదాలు/ప్రేలాపనలు ; అధివిక్షిపన్తి = ప్రేలుచు /చెప్పుచు ; మత్తాని = మత్తులో; చ అన్యోన్యమ్ = ఒకరినొకరు ; అధిక్షిపన్తి = నిందించుకొనుచు;
మదోద్రిక్తులై దానవులు ఒకరినొకరు ఆక్షేపించుకొనుచు, కొవ్విన తమ చేతులు ఒకరిపై ఒకరు వేసుకొనుచు, ఉన్నత్త ప్రలాపములను ప్రేలుచు, ఒకరినొకరు నిందించుకొనుచు తిరుగుచుండిరి – 5.11

रक्षांसि वक्षांसि च विक्षिपन्ति
गात्राणि कान्तासु च विक्षिपन्ति
रूपाअणि चित्राणि च विक्षिपन्ति
दृढानि चापानि च विक्षिपन्ति 		రక్షాంసి వక్షాంసి చ విక్షిపన్తి
గాత్రాణి కాన్తాసు చ విక్షిపన్తి
రూపాణి చిత్రాణి చ విక్షిపన్తి
దృఢాని చాపాని చ విక్షిపన్తి
రక్షాంసి = రాక్షసులు; వక్షాంసి = రొమ్ము; విక్షిపన్తి = కదల్చుచు; గాత్రాణి = శరీరములను; కాన్తాసు = మహిళలపై; విక్షిపన్తి = పడవేయుచు; రూపాణి =రూపాలు; చిత్రాణి = వింత; విక్షిపన్తి = తిరుగుచు; దృఢాని = వారి బలమైన ;చాపాని = విండ్లను; విక్షిపన్తి= కదల్చుచు;
రక్కసులు కొందరు వక్షములను అటు ఇటు కదల్చుచు తమ శరీరములను స్త్రీలపై పడవేయుచు, కామరూపులై పెక్కు అద్భుత రూపములను దాల్చుచు బలీయములైన విండ్లను కదల్చుచు తిరుగుచుండిరి – 5.12

ददर्श कान्ताश्च समालभन्त्य
स्तथा परास्तत्र पुनः स्वपन्त्यः
सुरूपवक्त्राश्च तथा हसन्त्यः
क्रुद्धाः पराश्चपि विनिःश्वसन्त्यः 		దదర్శ కాన్తాశ్చ సమాలభంత్య:
తథా పరాస్తత్ర పునః స్వప౦న్త్య
సురూప వక్త్రాశ్చ తథా హసంత్య:
కృద్ధా: పరాశ్చాపి వినిశ్శ్వసంత్య:
దదర్శ = చూసెను ; కాన్తాశ్చ = కొందరు స్త్రీలు; సమాలభంత్య: = వారి శరీరంపై గంధపు పూత పూసుకొని; పరాస్తత్ర = మరికొందరు మహిళలు; పునః = మళ్ళా ; స్వప౦న్త్య = నిద్రించుచుండిరి; తథా = అదే విధంగా; సురూప = అందమైన స్త్రీలు; వక్త్రాశ్చ= ముఖముకల; హసంత్య: = నవ్వుతూ ఉన్నారు; కృద్ధా: = కోపంతో; పరాశ్చాపి = కొందరు మహిళలు; వినిశ్శ్వసంత్య: = నిట్టూర్పులు విడుచుచు
కొందరు కాంతలు మైపూత పూసుకొని అలంకరించుకొనుచుండిరి. కొందరు అచ్చటచ్చట నిద్రించుచుండిరి. మరికొందరు అందమైన ముఖముకల మందయానలు నవ్వుచుండిరి. కొందరు కోపించి నిట్టూర్పులు విడుచుచు హనుమంతుని కంటబడిరి - 5.13

महागजैश्चापि तथा नदद्भिः
सुपूजितैश्चापि तथा सुसद्भिः
रराज वीरैश्च विनिःश्वसद्भि
र्ह्रदो भुजङ्गैरिव निःश्वसद्भिः 		మహాగజైశ్చాపి తథా నదద్భిః
సుపూజితైశ్చాపి తథా సుసద్భిః
రరాజ వీరైశ్చ వినిశ్శ్వసద్భి:
హ్రదో భుజఙ్గై: ఇవ నిశ్శ్వసద్భిః
మహాగజైశ్చాపి = మదగజములతో; నదద్భిః = ఘీంకారములు ఒనర్చుచు; తథా = మరియు; సుపూజితై శ్చాపి= సమ్మానితులైన; సుసద్భిః = విద్వాంసుల; రరాజ = ప్రకాశించింది; వీరైశ్చ = యోధులు; వినిశ్శ్వసద్భి: = నిట్టూర్పులు; హ్రదో భుజఙ్గై: ఇవ = పాములతో నిండియున్న సరస్సువలె; నిశ్శ్వసద్భిః = బుసకొట్టు;
ఘీంకారములు ఒనర్చుచు మదగజములతోడను సమ్మానితులైన విద్వాంసుల తోడను, యుద్ధము లేకపోయనే అని నిట్టూర్పులు విడుచు భటుల తోడను నిండి, బుసకొట్టు పాములతో నిండియున్న సరస్సువలె రావణ అంత:పురము హనుమకు గోచరించెను – 5.14

बुद्धिप्रधानान् रुचिराभिधानान्
संश्रद्दधानान् जगतः प्रधानान्
नानाविधानान् रुचिराभिधानान्
ददर्श तस्याम् पुरि यातुधानान् 		బుద్ధి ప్రధానాన్ రుచిరాభిధానాన్
సంశ్రద్దధానాన్ జగతః ప్రధానాన్
నానా విధానాన్ రుచిరాభిధానాన్
దదర్శ తస్యాం పురి యాతుధానాన్
బుద్దిప్రదయనాన్ = మేధావులు; రుచిరాభిధానాన్= చమత్కారముగ సంభాషించు వారిని; సంశ్రద్దధానాన్ = మంచి భక్తి కలిగినవారు; జగతః ప్రధానాన్ = లోకమున ముఖ్యులను; నానా విధానాన్ = పలువిధలైన ఆచారములు కలిగినవారు; రుచిరాభిధానాన్ = అందమైన పేర్లు ఉన్నవారు; దదర్శ = చూసెను ;తస్యాం పురి = ఆ నగరంలో; యాతుధానాన్= రాక్షసులు;
మారుతి అచట మహాధీమంతులని/ మేధావులు, చమత్కారముగ సంభాషించు వారిని, మంచి భక్తి కలవారిని, అందమైన పేర్లు కలవారిని, లోకమున ముఖ్యులను, పలువిధలైన ఆచారములు కల రాక్షసులను గాంచెను – 5.15

ननन्द दृष्ट्वा स च तान् सुरूपा
न्नानागुणानात्मगुणानुरूपान्
विद्योतमानान्स तदानुरूपान्
ददर्श कांश्चिच्च पुनर्विरूपान् 		ననన్ద దృష్ట్వా స చ తాన్ సురూపాన్
నానా గుణాన్ ఆత్మ గుణానురూపాన్
విద్యోతమానాన్ స తదానురూపాన్
దదర్శ కాంశ్చిచ్చ పునర్విరూపాన్
ననన్ద = ఆనందంగా ; దృష్ట్వా = చూసిన ; స: = ఆ హనుమ; తాన్ సురూపాన్ = సురూపము కలవారిని ; నానా గుణాన్ = వివిధ మంచి గుణములను కలవారు; ఆత్మగుణానురూపాన్ = మంచి రూపాన్ని కలవారు;విద్యోతమానాన్ = ప్రకాశవంతమైనవారు; తదానురూపాన్ = అనురూపమైన ప్రవర్తనము కలవారిని; దదర్శ = చూచెను ; కాంశ్చిచ్చ = ఇంకొందరు ; పున:= మరల ; విరూపాన్ = వికృతాకారులు;
అనేక విధములైన గుణములను కలిగి ఆయా గుణములకు అనురూపమైన ప్రవర్తనము కలవారిని, సురూపము కలవారిని చూచి హనుమ ఆనందించెను. వికృతాకారులై ఆ రూపములకు తగిన వికృతచేష్టలు ఒనర్చువారిని కూడా అతడు చూచెను – 5.16

ददर्श ताराइव सुप्रभावाः 		తతో వరార్హ: సువిశుద్ధ భావా:
తేషాం స్త్రియస్ తత్ర మహానుభావా:
ప్రియేషు పానేషు చ సక్తభావా
దదర్శ తారా ఇవ సుప్రభావాః
తత: = అనంతరం; వరార్హ: = శ్రేష్ఠ పురుషుల; సువిశుద్ధ భావా: = పరమ పతివ్రతలను ; తేషాం స్త్రియ: = రాక్షసకాంతలను; తత్ర = అక్కడ; మహానుభావా: = గొప్ప నైపుణ్యం ఉన్న వారు; ప్రియేషు = ప్రేమికులలో/ తమ ప్రియులందు; పానేషు = పానీయాలు; సక్తభావా =ఆసక్తి హృదయులు; దదర్శ = చూచెను ;తారాహీవ = నక్షత్రమువలె మెరయు; సుప్రభావాః = మంచి ప్రభావముతో
అనంతరము శ్రేష్ఠ పురుషులకు తగిన సుందరులను, పరమ పతివ్రతలను, నిర్మల చిత్తులను, తమ ప్రియులందు, మద్యమునందును ఆసక్తురాండ్రును, తమ ప్రభావమువలన నక్షత్రమువలె మెరయు రాక్షసకాంతలను చూచెను – 5.17

श्रिया ज्वलन्तीस्त्रपयोगूढा
निशीथकाले रमणोपगूढाः
ददर्श काश्चित्प्रमदोपगूढा
यथा विहङ्गाः कुसुमोपगूढाः 		శ్రియా జ్వలన్తీ: త్రపయోపగూఢా
నిశీథ కాలే రమణోపగూఢాః
దదర్శ కాశ్చిత్ ప్రమదోపగూఢా
యథా విహంగాః కుసుమోపగూడాః
శ్రియా = కాంతితో ; జ్వలన్తీ: = మెరయుచు; త్రపయా ఉపగూఢా = సిగ్గుతో నిండి; నిశీథ కాలే = రాత్రి వేళలో; రమణోపగూఢాః = ప్రేమికులతో కౌగలింపబడినవారై; దదర్శ = చూచెను ; కాశ్చిత్ = కొందరు మహిళలు; ప్రమదోపగూఢా = ఆనందపరిపూర్ణులై; యథా విహంగాః = పక్షులవలె; కుసుమోపగూడాః = పుష్పాలంకృతులై;
సౌందర్యముతో మెరయుచు సిగ్గుతో నిండి రమణులచే కౌగలింపబడినవారై ఆనందపరిపూర్ణులై పుష్పాలంకృతులై అందమైన పక్షులవలె ఉన్న కొందరు స్త్రీలను చూచెను – 5.18

अन्याः पुनर्हर्म्यतलोपविष्टा
स्तत्र प्रियाङ्केषु सुखोपविष्टाः
भर्तुः प्रिया धर्मपरा निविष्टा
ददर्श धीमान् मदनाभिविष्टाः 		అన్యాః పున : హర్మ్య తలోపవిష్టా
తత్ర ప్రియాంగేషు సుఖోపవిష్టాః
భర్తుః ప్రియా ధర్మపరా నివిష్టా
దదర్శ ధీమాన్ మదనాభివిష్టాః
అన్యాః = ఇతర మహిళలు; పున : = ఇంకా/మళ్ళా; హర్మ్య = వారి భవనాల; తలోపవిష్టా = పైభాగంలో కూర్చున్న వారిని; తత్ర = అక్కడ; ప్రియాంగేషు = ప్రేమికుల; సుఖోపవిష్టాః = తొడలపై సుఖముగా కూర్చున్నవారిని; భర్తుః ప్రియా = తమ భర్తలకు ప్రియమైనవారు; ధర్మపరా = ధర్మమునందు ఆసక్తి ; నివిష్టా = కూర్చోవడం; దదర్శ = చూచెను ; ధీమాన్ = ధీశాలైన హనుమ; మదనాభివిష్టాః = కామావేశము కలవారిని
ధీశాలైన హనుమ మేడల పైభాగమున కూర్చున్న వారిని, రమణుల తొడలపై సుఖముగా కూర్చున్నవారిని, పతులకు అత్యంత ప్రియమైనవారిని, కామావేశము కలవారిని, ధర్మమునందు ఆసక్తి కల స్త్రీలను చూచెను – 5.19

अप्रावृताः काञ्चनराजिवर्णाः
काश्चित्परार्थ्यास्तपनीयवर्णाः
पुनश्च काश्चिच्छशलक्ष्मवर्णाः
कान्तप्रहीणारुचिराङ्गवर्णाः 		అప్రావృతాః కాంచిన రాజి వర్ణాః
కాశ్చిత్ పరార్ధ్యా: తపనీయ వర్ణాః
పునశ్చ కాశ్చిత్ శలక్ష్మవర్ణాః
కాన్త ప్రహీణా రుచిరా౦గ వర్ణాః
అప్రావృతాః = బట్టలు లేని;కాంచిన = బంగారు ; రాజివర్ణాః = ఛాయతో ఉన్న స్త్రీలు; కాశ్చిత్ = ఇంకొందరు మహిళలు; పరార్ధ్యా: = సంభోగం కోసం; తపనీయ వర్ణాః = బంగారుఛాయ గల మేనితో; పునశ్చ = ఇంకా; కాశ్చిత్ = కొందరు మహిళలు; శలక్ష్మవర్ణాః = చంద్రుని రంగు/ తెల్లబారిన వన్నెగల సుందరులను; కాన్త ప్రహీణా = ప్రియ విరహమువలన ; రుచిరా౦గ వర్ణాః = ఉజ్జ్వల శరీరచ్ఛాయ కలవారు
తమపతులను గవయువేళ వస్త్రములను త్యజించుటవలన బంగారుఛాయ గల మేనితో ఒప్పువారును,లోకోత్తర సౌందర్యవతులు,కనకవర్ణ విరాజితులైన స్త్రీలను కూడచూచెను. ఉజ్జ్వల శరీరచ్ఛాయ కలవారు, ప్రియ విరహమువలన తెల్లబారిన వన్నెగల సుందరులను గాంచెను - 5.20

ततह् प्रियान् प्राप्य मनोभिरामान्
सुप्रीतियुक्ताः सुमनोभिरामाः
गृहेषु हृष्टाः परमाभिरामाः
हरिप्रवीरः स ददर्श रामाः 		తతః ప్రియాన్ ప్రాప్య మనోభిరామాన్
సుప్రీతి యుక్తాః ప్రసమీక్ష్య రామాః
గృహేషు హృష్టాః పరమాభిరామా
హరిప్రవీరః స దదర్శ రామాః
తతః = అనంతరం; ప్రియాన్ ప్రాప్య = ప్రేమికులు యొద్ద కేగి; మనోభిరామాన్ = స్వీకరించి / గవసి; సుప్రీతి యుక్తాః = ఆనందపరవశులై; ప్రసమీక్ష్య రామాః = కన్నుల పండువు చేయు; గృహేషు = వారి ఇళ్లలో ;హృష్టాః = సంతోషంతో ; పరమాభిరామా = గొప్ప ఆనందము పొంది; హరిప్రవీరః = వానరయోధుడు; దదర్శ = చూచెను ; రామాః = నేత్రపర్వమొనర్చు మోహనాంగులు;
అనంతరము తమ ప్రియతముల యొద్ద కేగి వారిని గవసి ఆనందపరవశులై కన్నుల పండువు చేయు సుందరాంగులను చూసెను. పిదప గృహములందే భర్తలను పొంది హర్షముతో నిండి ఇంకనూ అధికముగా నేత్రపర్వమొనర్చు మోహనాంగులను చూసెను - 5.21

चन्द्रप्रकाशाश्च हि वक्त्रमाला
वक्राक्षिपक्ष्माश्च सुनेत्रमालाः
विभूषाणानाम् च ददर्श मालाः
शतह्रदानामिव चारुमालाः 		చన్ద్ర ప్రకాశాశ్చ హి వక్త్ర మాలా:
వక్రాక్షి పక్ష్మాశ్చ సునేత్ర మాలాః
విభూషణానాం చ దదర్శ మాలాః
శతహ్రదానామ్ ఇవ చారు మాలాః
చన్ద్ర ప్రకాశాశ్చ = చంద్రునిపోలు సుందరాంగుల; వక్త్ర మాలా: = వరుస ముఖాలు; వక్రాక్షి = వక్రమైన; పక్ష్మాశ్చ = కనుబొమ్మలతో; సునేత్ర మాలాః = అందమైన కళ్ళు; విభూషణానాం చ = మరియు ఆభరణాలను; దదర్శ = చూచెను ; మాలాః = వరుసను ; శతహ్రదానామ్ = మెరుపులవలె; చారు మాలాః= మెరయువారిని;
సీతామాత సందర్శన లాలసుడైన మనస్సుతో ఆ మహానుభావుడు చంద్రునిపోలు సుందరాంగుల ముఖముల శ్రేణులను, మనోజ్ఞములగు వక్రమైన కనుబొమ్మలతో అందమైన నయనముల (కళ్ళు ) వరుసను, నానావిధ భూషణములను, మెరుపులవలె మెరయువారిని చూసెను – 5.22

न त्वेव सीताम् परमाभिजाताम्
पथि स्थिते राजकुले प्रजाताम्
लताम् प्रपुल्लामिव साधु जाताम्
ददर्श तन्वीम् मनसाभिजाताम्		 న త్వేవ సీతాం పరమాభిజాతాం
పథిస్థితే రాజకులే ప్రజాతామ్
లతాం ప్రఫుల్లామ్ ఇవ సాధు జాతాం
దదర్శ తన్వీం మనసాభిజాతామ్
న త్వేవ దదర్శ = చూడలేదు; సీతాం = సీతామాతను; పరమాభిజాతాం = పరమ సౌందర్యవతియు; పథిస్థితే = సదాచార సంపన్న; రాజాకులే = రాచరిక కుటుంబంలో; ప్రజాతామ్ = జననం; లతాం ఇవ = ఒక లత వంటి; ప్రఫుల్లామ్ = వికసించడం;సాధు జాతాం = సాధ్వీలాగా జనించిన (రావణ సంహార కార్యమునకై); తన్వీం = సున్నితమైన శరీరము గల; మనసాభిజాతామ్ = మన:పూర్వకముగా మేనుదాల్చిన
కాని ఆ మహాత్ముడు అనన్య సామాన్య పరమ సౌందర్యవతియు, సదాచార సంపన్న, క్షత్రియ వంశమున అవతరించి, రావణ సంహార కార్యమునకై మన:పూర్వకముగా మేనుదాల్చిన సీతామాతను మాత్రము సందర్శింపలేకపోయెను – 5.23

सनातने वर्त्मानि सम्निविष्टाम्
रामेक्षणां तां मदनाभिविष्टाम्
भर्तुर्मनः श्रीमदनुप्रविष्टाम्
स्त्रीभ्यो वराभ्यश्च सदा विशिष्टाम् 		సనాతనే వర్త్మాని సన్నివిష్టాం
రామేక్షణాం తాం మదనాభివిష్టామ్
భర్తుర్మనః శ్రీమద్ అనుప్రవిష్ఠాం
స్త్రీభ్యో వరాభ్యశ్చ సదావిశిష్టామ్
సనాతనే వర్త్మాని = సనాతన ధర్మ మార్గంలో; సన్నివిష్టాం = నిలబడడం; రామేక్షణాం తాం = శ్రీరామదర్శన లాలస; మదనాభివిష్టామ్ = శ్రీ రామ విరహముచే ఆర్తయు; భర్తు: = భర్త; మన: = మనస్సులో; శ్రీమద్ = పవిత్రమైన ;అనుప్రవిష్ఠాం = అనుష్ఠితమైన/స్థానం; స్త్రీభ్యో వరాభ్యశ్చ = ఉత్తమోత్తమ స్త్రీలకంటె; సదా విశిష్టత = ఎల్లప్పుడూ విశిష్టమైన;
సనాతన పతివ్రతా ధర్మమార్గమున నిరతురాలు, శ్రీ రామదర్శన లాలస, శ్రీ రామ విరహముచే ఆర్తయు, భర్త యొక్క నిండు మనస్సున ఉన్నదియు, ఉత్తమోత్తమ స్త్రీలకంటె విశిష్టమైన వైదేహి మాత్రము కంటడలేదు –5. 24

उष्णार्दिताम् सानुसृतास्रकण्ठीम्
पुरा वरार्होत्तमनिष्ककण्ठीम्
सुजातपक्ष्मामभिरक्तकण्ठीम्
वनेऽप्रनृत्तामिव नीलकण्ठीम् 		ఉష్ణార్దితాం సానుసృతాస్ర కణ్ఠీం
పురా వరార్హోత్తమ నిష్కకణ్ఠీం
సుజాత పక్ష్మామ్ అభిరక్త కణ్ఠీం
వనే ప్రనృత్తామ్ ఇవ నీలకణ్ఠీం
ఉష్ణార్దితాం = శ్రీ రామ విరముచే వేడి; సానుసృతాస్ర = నిరంతరము స్రవించు కన్నిటిచే ; కణ్ఠీం =గద్గద కంఠము కలదై; పూరా = పూర్వం; వరార్హోత్తమ = అమూల్యమైన; నిష్కకణ్ఠీం = ఆభరణాలు కంఠమున దాల్చి; సుజాత = సుందరమైన ; పక్ష్మామ్ = అందమైన కనుబొమ్మలతో; అభిరక్త కణ్ఠీం = శుశ్వర కంఠము; వనే = అడవిలో; ప్రనృత్తామ్ = సంచరించు; నీలకణ్ఠీం= మయూరి/ నెమలి
విరహోష్మముచే కాగినది, నిరంతరము స్రవించు కన్నిటిచే గద్గద కంఠియై, పూర్వము రామదత్తమగు పతకమును కంఠమున దాల్చి, సుందరమైన కనురెప్పలు కలిగి, శుశ్వర కంఠముతో సంచరించు వన మయూరి అయిన సీతామాత అతనికి కానరాదయ్యెను –5.25

अव्यक्तरेखामिव चन्द्ररेखां

वायुप्रभिन्नामिव मेघरेक्षाम् 		అవ్యక్త రేఖామ్ ఇవ చన్ద్రరేఖాం
పాంసు ప్రదిగ్ధామ్ ఇవ హేమరేఖాం
క్షత ప్రరూఢామ్ ఇవ బాణరేఖాం
వాయు ప్రభిన్నామ్ ఇవ మేఘరేఖాం
అవ్యక్త రేఖామ్ = మేఘముచే కప్పబడి కాంతిని కోల్పోయిన; చన్ద్రరేఖాం ఇవ = చంద్రరేఖవలె; పాంసు ప్రదిగ్ధామ్ = దుమ్ము, ధూళితో; హేమరేఖాం ఇవ = స్వర్ణ పతాకము వలె; క్షత= గాయమున; ప్రరూఢామ్ = గుచ్చుకున్న; ఇవ బాణరేఖాం = బాణపు ములికివలె; వాయు = వాయువుచే; ప్రభిన్నామ్ = చెదరగొట్టబడిన; ఇవ మేఘరేఖాం = మేఘమువలె
మేఘముచే ఆవృతమగుటవలన కాంతిని కోల్పోయిన చంద్రరేఖవలె, ధూళిధూసరితమైన స్వర్ణ పతాకము వలె, గాయమున గుచ్చుకున్న బాణపు ములికివలె, వాయువుచే చెదరగొట్టబడిన మేఘమువలె ఉండు సీతామాత అతనికి కనపడలేదు – 5.26

प्लवङ्गमो मन्द इवाचिरस्य 		సీతామ్ అపశ్యన్ మనుజేశ్వరస్య
రామస్య పత్నీం వదతాం వరస్య
బభూవ దుఃఖాభి హతశ్చిరస్య
ప్లవంగమో మన్ద ఇవా చిరస్య
సీతామ్ = సీతామాత; అపశ్యన్ = కనుగొనలేక; మనుజేశ్వరస్య = మనుజులకు/జనులకు రాజు ; రామస్య = శ్రీ రామ ; పత్నీం = భార్య; వదతాం = యుక్తముగా మాటలాడు; వరస్య = ఉత్తముడు; బభూవ = అయ్యెను ; దుఃఖాభి హత: = దు:ఖచకితుడై; చిరస్య = ఒక క్షణం; ప్లవంగమో = హనుమ; మన్ద ఇవ = మందబుద్ధి/అజ్ఞానివలె ; అచిరస్య = ఒక క్షణం;
ఈ విధమున హనుమంతుడు ఎందరినో చూచెను, కానీ ఉత్తముడు, యుక్తముగా మాటలాడు నేర్పుగల రామభూపాలుని అర్ధాంగి అయిన మైధిలిని వెదకియు కనుగొనలేక ఒక క్షణం దు:ఖచకితుడై చాలాసేపు మందబుద్ధితో చింతించెను – 5.27
