सीताय वचनं श्रुत्वा परुषं राक्षसाधिपः

రాక్షసరాజగు రావణుడు సీతాదేవి పలికిన పరుషములైన మాటలను విని, ప్రియదర్శన అయిన ఆమెను చూచి, కఠినముగా ఇట్లు సమాధానమిచ్చెను – 22.1

యథా యథా ప్రియం వక్తా - పరిభూత: తథా తథా
యథా యథా = ఏ విధంగా అయినా; సాంత్వయితా = సున్నితమైన భాష/ బుజ్జగించునో; వశ్యః = స్వాధీనుడగును; స్త్రీణాం = మహిళలు; తథా తథా = ఆ విధంగా; ప్రియం వక్తా = ప్రియమైన మంచి మాటలు; పరిభూత: = తిరస్కరించబడు; తథా తథా = ఆ విధంగా;
పురుషుడు స్త్రీనెంత బుజ్జగించునో, వారికంత స్వాధీనుడగును. వారికెంత మంచి మాటలు చెప్పునో, వారిచే అంత తిరస్కారమును పొందును. ఇది లోక సహజము - 22.2

ద్రవతోమార్గమ్ ఆసాద్య - హయానివ సుసారథిః
సన్నియచ్ఛతి = అణుచుచున్నది; మే = నాకు ; క్రోధం = కోపం; త్వయి = నీయెడల; కామః సముత్థితః = కోరిక కలిగిన; ద్రవతో = పరుగెత్తు; అమార్గమ్ = త్రోవ తప్పి ; ఆసాద్య = ఇచ్చవచ్చినట్లు/తీసుకొని; హయాన్ ఇవ = గుర్రములను; సుసారథిః = సమర్ధుడైన సారధి (లాగునట్లు)
ఓ సీతా! నీయెడల నాకు కలిగిన కామము, సమర్ధుడైన సారధి త్రోవ తప్పి ఇచ్చవచ్చినట్లు పరుగెత్తు గుర్రములను లాగునట్లు, నా కోపము అణుచుచున్నది - 22.3

జనే తస్మిం త్వనుక్రోశః - స్నేహశ్చ కిల జాయతే
వామః = చెడ్డది; కామో = కామో; మనుష్యాణాం = పురుషులు; యస్మిన్ = ఎవ్వరిలో; కిల నిబధ్యతే = నిబద్ధత; జనే = మనిషి; తస్మిం = ఆ మనిషి; త్వ = వారు; అనుక్రోశః = కరుణ/ దయను; స్నేహశ్చ = స్నేహమును; కిల జాయతే = శిక్షింప అర్హులైనను;
కామము మిక్కిలి చెడ్డది (పక్షపాతి), అవి ఎవ్వరి విషయమున దృఢముగా కలుగునో, వారు శిక్షింప అర్హులైనను, వారి యెడల దయను, స్నేహమును కలుగుచుండును - 22.4

వధార్హామ్ అవమానార్హాం - మిథ్యా ప్రవ్రజితే రతామ్

అందమైన ముఖముగల ఓ సీతా! నీ యెడల నాకు మోహము ఉండుట చేతనే నన్ను తిరస్కరించి నీవు కపట వానప్రస్ధుడగు శ్రీరాముని యందు ఆసక్తి కలిగియున్నను, చంపదగినదానవైనను, అవమానింప తగిన దానవైనను నిన్ను హింసింపకున్నాను/చంపడం లేదు - 22.5
