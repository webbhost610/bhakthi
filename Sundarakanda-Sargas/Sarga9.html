तस्य आलय वरिष्ठस्य मध्ये विपुलम् आयतम्
ददर्श भवन श्रेष्ठम् हनूमान् मारुत आत्मजः		 తస్య ఆలయ వరిష్ఠస్య మధ్యే విపులమ్ ఆయతమ్
దదర్శ భవన శ్రేష్ఠం హనూమాన్ మారుతాత్మజః
తస్య ఆలయ వరిష్ఠస్య = ఉత్తమమైన విమానమున; మధ్యే = మధ్యలో; విపులమ్ = విశాలమైన/విస్తృతము; ఆయతమ్ = పొడవైన; దదర్శ = చూసెను; భవన శ్రేష్ఠం = ఒక ఉత్తమ భవనం; హనూమాన్ = హనుమ; మారుతాత్మజః = వాయు కుమారుడు;
మారుతి ఆ ఉత్తమోత్తమ భవన మధ్యమున విశాలమగు భవనము అనదగిన ఆ విమానమును చూచెను – 9.1

अर्ध योजन विस्तीर्णम् आयतम् योजनम् हि तत्
भवनम् राक्षस इन्द्रस्य बहु प्रासाद सम्कुलम् 		అర్ధ యోజన విస్తీర్ణమ్ ఆయతం యోజనం హి తత్
భవనం రాక్షసేన్ద్రస్య బహు ప్రాసాద సంకులమ్
అర్ధ యోజన = అర్ధయోజన ;విస్తీర్ణమ్ = విస్తీర్ణము; ఆయతం = పొడవు ; యోజనం = ఒక యోజనం;తత్ భవనం = ఆ భవనం; రాక్షసేన్ద్రస్య = రావణుని యొక్క; బహు ప్రాసాద = అనేక ప్రాసాదములతో; సంకులమ్= నిండియుండెను;
ఆ రాక్షసేంద్రుడగు రావణుని భవనము అర్ధయోజన విస్తీర్ణము కలదై యోజనము పొడవు కలిగి అవాంతర ప్రాసాదములతో నిండియుండెను – 9.2

मार्गमाणः तु वैदेहीम् सीताम् आयत लोचनाम्
सर्वतः परिचक्राम हनूमान् अरि सूदनः 		మార్గమాణస్తు వైదేహీం సీతామ్ ఆయత లోచనామ్
సర్వతః పరిచక్రామ హనూమాన్ అరిసూదనః
మార్గమాణస్తు = మార్గ అన్వేషణ; వైదేహీం = విదేహ రాజు కుమార్తె ; సీతామ్ = సీతామాత; ఆయత లోచనామ్ = సువిశాలనేత్ర; సర్వతః = అన్ని దిశలలో; పరిచక్రామ = చుట్టూ తిరగడం ; హనుమాన్ = హనుమా; అరిసూదనః = శత్రుసంహారకుడు;
సువిశాలనేత్ర అయిన విదేహరాజు పుత్రికను వెదకువాడై శత్రుసంహారకుడగు ఆంజనేయుడు ఆ మందిరమంతటా అన్వేషించెను – 9.3

उत्तमम् राक्षसावासम् हनुमानवलोकयन्
आससादाथ लक्ष्मीवान् राक्षसेन्द्रनिवेशनम् 		ఉత్తమం రాక్షసావాసం హనుమాన్ అవలోకయన్
ఆససాద అథ లక్ష్మీవాన్ రాక్షసేంద్ర నివేశనం
ఉత్తమం = శ్రేష్ఠమగు; రాక్షసావాసం= రాక్షసుల నివాసం; హనుమాన్ = హనుమ; అవలోకయన్ = పరిశీలన; ఆససాద అథ = దగ్గరకి చేరెను; లక్ష్మీవాన్ = కాంతి దీపితుడైన; రాక్షసేంద్ర = రావణుని ; నివేశనం = గృహము
శ్రేష్ఠమగు ఆ రాక్షస భవనమును చూచుచు, కాంతి దీపితుడైన మారుతి రావణుని నివాస గృహమును చేరెను – 9.4

चतुर् विषाणैर् द्विरदैः त्रिविषाणैः तथैव च
परिक्षिप्तम् असम्बाधम् रक्ष्यमाणम् उदायुधैः 		చతుర్విషాణైర్ ద్విరదై: త్రివిషాణై: తథైవచ
పరిక్షిప్తమ్ అసంబాధం రక్ష్యమాణమ్ ఉదాయుధైః
చతుర్విషాణైర్ = నాలుగు దంతాలు ఏనుగులు; ద్విరదై: = రెండు దంతాలు ఏనుగులు; త్రివిషాణై: = మూడు దంతాల ఏనుగులు; తథైవచ = మరియు; పరిక్షిప్తమ్ = నిండియున్న; అసంబాధం = విశాలమైన; రక్ష్యమాణమ్ = రక్షింపబడు; ఉదాయుధైః = వివిధ ఆయుధములు పట్టి
నాలుగేసి దంతములు, మూడేసి దంతములు, రెండేసి దంతములు కల ఏనుగులచే పరివేష్టితమై బహుశస్త్ర సన్నద్ధులగు భటులచే ఆ గృహము రక్షింపబడుచుండెను - 9.5

राक्षसीभिः च पत्नीभी रावणस्य निवेशनम्
आहृताभिश्च विक्रम्य राज कन्याभिरावृतम् 		రాక్షసీభిశ్చ పత్నీభీ రావణస్య నివేశనమ్
ఆహృతాభిశ్చ విక్రమ్య రాజకన్యాభిరావృతమ్
రాక్షసీభిశ్చ = రాక్షసాంగనలచేతను; పత్నీభీ = భార్యలు;రావణస్య = రావణుని; నివేశనమ్ = ఆ భవనం; ఆహృతాభిశ్చ = కొనితేబడిన ;విక్రమ్య = బలంతో/ పరాక్రమముచే; రాజకన్యాభి= రాజకన్యల ; ఆవృతమ్ = నిండియుండెను
ఆ రావణ మందిరము రాక్షసాంగనలచేతను, పరాక్రమముచే కొనితేబడిన రాజకన్యల చేతను నిండియుండెను – 9.6

तन् नक्र मकर आकीर्णम् तिमिम्गिल झषाकुलम्
वायु वेग समाधूतम् पन्नगैर् इव सागरम् 		తన్ నక్ర మకర ఆకీర్ణం తిమింగిల ఝషాకులమ్
వాయు వేగ సమాధూతం పన్నగై: ఇవ సాగరమ్
తన్= అక్కడ ; నక్ర = వక్ర ; మకర = మకరములతో/ మొసళ్ళతో ; ఆకీర్ణం= నిండి; తిమింగిల = తిమింగళములతో; ఝషాకులమ్= సొరచేపలు మరియు ఇతర చేపలతో; వాయు వేగ = వాయు వేగంతో; సమాధూతం= చలించినదై; పన్నగై:= సర్పములతో; ఇవ సాగరమ్ = సాగరమువలె
అందువలన వక్ర మకరములతో నిండి, తిమింగల సొరచేపలు మరియు ఇతర చేపలతో సంకులమై, వాయువుచే చలించినదై, సర్పములతో కూడిన సాగరమువలె ఆ భవనము కానవచ్చుచుండెను – 9.7

या हि वैश्वरणे लक्ष्मीर् या च इन्द्रे हरि वाहने
सा रावण ग्ऱ्हे सर्वा नित्यम् एव अनपायिनी 		యా హి వైశ్రవణే లక్ష్మీర్యాచ ఇంద్రే హరివాహనే
సా రావణ గృహే సర్వా నిత్యమ్ ఏవ అనపాయినీ
వైశ్రవణే = కుబేరుని; లక్ష్మీర్యాచ = సర్వ సంపదలతో; ఇంద్రే = ఇంద్రుడు; హరివాహనే = గుర్రాలతో వాహనుములు; సా రావణ గృహే = రావనా ఇంటిలో ఉండేది; సర్వా = అన్ని; నిత్యమ్ ఏవ = ఎల్లప్పుడూ; అనపాయినీ = తక్కువ కాకుండా
కుబేరుని సర్వ సంపదలతో, ఇంద్రుని అష్టైశ్వర్యములు, వాహనుములు ఎక్కడా తక్కువ కాకుండా ఆ రావణ భవనమందు స్ధిరముగా తేజరిల్లుచుండెను– 9.8

या च राज्ञः कुबेरस्य यमस्य वरुणस्य च
तादृशी तद् विशिष्टा वा बुद्धी रक्षो गृहेष्व् इह 		యా చ రాజ్ఞః కుబేరస్య యమస్య వరుణస్య చ
తాదృశీ తద్ విశిష్టా వా ఋద్ధీ రక్షో గృహేష్వ: ఇహ
యా చ రాజ్ఞః = యక్షరాజైన; కుబేరస్య = కుబేరుడు; యమస్య = యముడు; వరుణస్య చ = మరియు వరుణుడు; తాదృశీ = వారి సంపదలు; తద్ విశిష్టా వా ఋద్ధీ = సమానము లేదా ఎక్కువ నిధి; రక్షో గృహేష్వ = రావణుని గృహములో; ఇహ =ఉంది
యక్షరాజైన కుబేరుని, యముని, వరుణుని గృహములలో ఎంత సంపద ఉన్నదో, దానితో సమానముగా లేదా దానికంటే ఎక్కువ సంపద రావణుని గృహములో శోభించుచుండెను - 9.9

तस्य हर्म्यस्य मध्यस्थम् वेश्म च अन्यत् सुनिर्मितम्
बहुनिर्यूह सम्कीर्णम् ददर्श पवन आत्मजः 		తస్య హర్మ్యస్య మధ్యస్థం వేశ్మచాన్యత్ సునిర్మితమ్
బహు నిర్యూహ సంకీర్ణం దదర్శ పవనాత్మజః
తస్య హర్మ్యస్య = ఆ ఇల్లు; మధ్యస్థం = మధ్యలో; వేశ్మ చ అన్యత్ = మరొక ఇల్లు; సునిర్మితమ్ = చక్కగా నిర్మింపబడి; బహు= అనేక; నిర్యూహ= స్థిరమైన ఏనుగులు ప్రతిమలతో ; సంకీర్ణం =కలిగి ఉన్నవి; దదర్శ = చూసెను; పవనాత్మజః = హనుమ;
ఆ భవనము యొక్క మధ్యభాగమున చక్కగా నిర్మింపబడి స్థిరమైన ఏనుగులు ప్రతిమలతో అద్భుతముగా నిర్మింపబడిన ఒక గృహమును చూచెను – 9.10

ब्रह्मणो अर्थे कृतम् दिव्यम् दिवि यद्विस्व कर्मणा

పరివృత్తే = సమయంలో; అర్ధరాత్రే = అర్ధరాత్రి ; పాన నిద్రా = మద్యం వలన; వశం గతమ్ = నిద్ర పొందడానికి; క్రీడిత్వో = ఆటపాట ఆడిన తర్వాత; ఉపరతం = విశ్రాంతి; రాత్రౌ = రాత్రి సమయంలో; సుష్వాప = (పారవశ్యము) నిద్రించిరి; బలవత్ = బలవంతంగా; తదా = అప్పుడు;
సగము రాత్రి గడచెను. ఆ సుందరులు మద్యపానము చేతను, నిద్ర చేతను పారవశ్యమును పొందుచుండిరి. ఆటపాటలందు తృప్తి గానక మరల ఆడి తుదకు అలసిపోయి నిద్రించిరి – 9.35

तत् प्रसुप्तम् विरुरुचे निश्शब्दान्तर भूषणम्
निःशब्द हंस भ्रमरम् यथा पद्म वनम् महत् 		తత్ ప్రసుప్తం విరురుచే నిశ్శబ్దాన్తర భూషణమ్
నిశ్శబ్ద హంస భ్రమరం యథా పద్మ వనం మహత్
తత్ ప్రసుప్తం = నిదురిస్తున్న మహిళల సమూహం; విరురుచే = ప్రకాశిస్తు; నిశ్శబ్దాన్తర= శబ్దం చేయని ఆభరణాలతో; భూషణమ్ = అలంకరించారు; నిశ్శబ్ద = నిశ్శబ్ధంగా; హంస భ్రమరం = హంసలు,తేనెటీగలు; యథా పద్మ వనం = కమల వనమువలె; మహత్ = గొప్ప ;

పునశ్చ = మళ్ళా; సోచిన్తయత్= ఆలోచన; ఆర్త రూపో= దిగులుగా మారింది; ధ్రువం =ఖచ్చితంగా; విశిష్టా = ఉత్తమ ; గుణతో =గుణములో ; హి సీతా = జానకీదేవి; అథాయమ్= నిర్దయగా; అస్యాం = ఆమె పట్ల; కృతవాన్ = ప్రవర్తించెను ;మహాత్మా = ఉత్తమ వంశములో పుట్టినవాడు; ల౦కేశ్వరః = రావణుడు ; కష్టమ్ = కష్ఠాలు ; అనార్య కర్మ= హీనముగా;
మరియు భిన్నుడై అతడిట్లు చింతించెను. జానకీదేవి ఎల్లవిధముల ఈ స్త్రీలకంటె ఉత్తమోత్తమురాలు. పాపాత్ముడగు రావణుడు ఆమె పట్ల ఎంతటి హీనముగా ప్రవర్తించెను? అని – 9.73
