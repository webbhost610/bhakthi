मुक्ता प्रवाल सिकता स्फाटिकान्तर कुट्टिमाः
कान्चनैः तरुभिः चित्रैः तीरजै: उपशोभिताः 		ముక్తా ప్రవాళ సికతా- స్ఫటికాన్తర కుట్టిమాః

అవి ముత్యములు, పగడముల చూర్ణములతో నిండిన ఇసుక దిబ్బలు, స్ఫటికమయములైన తిన్నెలు, ఒడ్డున అందముగా ఉన్న బంగారు వృక్షములు కలిగి ఉండెను – 14.23

फुल्ल पद्मोत्पल वनाः चक्रवाकोप कूजिताः
नत्यूह रुत सम्घुष्टा हंस सारस नादिताः 		ఫుల్ల పద్మోత్పల వనా: - చక్రవాకోప కూజితాః
నత్యూహ రుత సంఘుష్టా - హంస సారస నాదితాః
ఫుల్ల పద్మోత్పల = పద్మములతో, కలువలతో; వనా: = వనం ; చక్రవాక = చక్రవాకములు; ఉపకూజితాః= అరుపులతో/ కూతలతో; నత్యూహ రుత = నీటి పక్షులు(వానకోయిలలు); సంఘుష్టా = కలిసి; హంస = హంసలు; సారస = బెగ్గురు పక్షుల/కొంగలు; నాదితాః= అరుపులతో/కూతలతో;
ఆ వనము వికసించిన పద్మములతో, కలువలతో నిండి ఉన్నది. వానకోయిలలు, చక్రవాకములు, హంసలు, బెగ్గురు పక్షుల (కొంగలు) యొక్క అరుపులతో (కూతలతో ) ఇంపుగా ఉన్నది – 14.24

दीर्घाभि: द्रुम युक्ताभिः सरद्भिश्च समन्ततः
अमृतोपम तोयाभिः शिवाभि: उपसंस्कृताः 		దీర్ఘాభి: ద్రుమ యుక్తాభిః - సరిద్భిశ్చ సమన్తతః
అమృతోపమ తోయాభిః - శివాభి: ఉపసంస్కృతాః
దీర్ఘాభిర్= ఎత్తయిన/ పొడవైన; ద్రుమ= చెట్లతో; యుక్తాభిః= కలిసి; సరిద్భిశ్చ= నీటికాలువలు; సమన్తతః = అంతటా విస్తరించి;అమృతోపమ తోయాభిః = అమృతములు పోలి మధురమైన జలము; శివాభి: = పవిత్రమైన; ఉపసంస్కృతాః= వాగులతో నిండియున్నవి;
ఆ వాపికల/ వాగుల గట్టుపై ఎత్తయిన వృక్షములు కలవు. మంగళకరమై అమృతములు పోలి మధురమైన, పవిత్రమైన జలము గల వాగులతో అవి నిండియున్నవి – 14.25

लताशतै: अवतताः सन्तान कुसमावृताः
नाना गुल्मावृतघनाः करवीर कृतान्तराः 		లతాశతై: అవతతాః – సన్తాన కుసమా వృతాః
నానా గుల్మావృత ఘనాః - కరవీర కృతాన్తరాః
లతాశతై: = వందల తీగలచే/లతలచే; అవతతాః= వ్యాప్తి; సన్తానక = అశోక చెట్లతో ; సమావృతాః=నిండి ; నానా = పెక్కు విధముల ; గుల్మావృత ఘనాః = పొదలతో; కరవీర = గన్నేరు చెట్లతో ; కృతాన్తరాః = నడుమ
ఆ వనము వందల కొలది తీగలచే/లతలచే నిండి చుట్టును కల్పవృక్షములు (అశోక చెట్లతో) కలిగి, పెక్కు విధములగు పొదలతో కూడి నడుమ గన్నేరు చెట్లతో శోభించుచున్నవి - 14.26

ततोऽम्बुधर सङ्काशं प्रवृद्ध शिखरम् गिरिम्

తతో=అనంతరం; అమ్బుధర సంకాశం= మేఘములతో సమానమై; ప్రవృద్ధ= ఎత్తైన; శిఖరం గిరిమ్= శిఖరములు;విచిత్రకూటం = విచిత్ర శిఖరములు; కూటైశ్చ= పర్వతమును; సర్వతః = అన్ని వైపులను ;పరివారితమ్= వ్యాపించియుండెను;
హనమంతుడు ఆ పిదప మేఘములతో సమానమై ఎత్తైన శిఖరములు కల ఒక పర్వతమును చూచెను. దాని శిఖరములు విచిత్రములై అన్ని వైపులను వ్యాపించియుండెను – 14.27

शिला गृहै: अवततम् नाना वृक्षैः समावृतम्

శిలాగృహై:= శిలాగృహములతోను (గుహలు) ; అవతతం = అవతరించిన/ఉండిన; నానా = వివిధ ; వృక్షైః = వృక్షములతోను; సమావృతమ్= నిండి ఉండెను; దదర్శ = చూసెను; కపి శార్దూలో = హనుమ; రమ్యం = అందమైన/ రమ్యమై; జగతి = ప్రపంచంలో; పర్వతమ్ = పర్వతం;
ఆ పర్వతము శిలాగృహములతోను (గుహలు) నానా వివిధ వృక్షములతోను నిండి ఉండెను, ప్రపంచంలో ఒక అందమైన/ రమ్యమై పర్వతమును కపిశార్దూలుడైన హనుమ చూసెను – 14.28

అ౦కాదివ సముత్పత్య - ప్రియస్య పతితాం ప్రియామ్

ఆ పర్వతము నుండి ప్రవహించుచున్న ఒక నదిని హనుమ చూచెను. ఆ నది ప్రియుని తొడపై నుండి జారిపడు యువతివలె కావనచ్చెను– 14.29

जले निपतिताग्रैश्च पादपैर् उपशोभिताम्

पुनरावृत्त तोयाम् च ददर्श स महाकपिः

అందమైన కన్నులు కలదియు, శ్రీరాముని గురంచి విచారముచే పరితపించునదియు, వనసంచారమునకు అలవాటుపడినదియు అగు సీతాదేవి వనవాసమును ఎరుగును కనుక ఇక్కడికి రావచ్చును – 14.47
