సుందరకాండ - ఆరవ సర్గ

రావణ - రావణానుచర గృహములలో హనుమంతుడు అన్వేషించుట

स निकामम् विमानेषु विषण्णः काम रूपधृत्
विचचार पुनर्लङ्काम् लाघवेन समन्वितः 		స నికామం వినామేషు నిషణ్ణ కామరూప ధృత్
విచచార పునర్లంకాం లాఘవేన సమన్వితః
స: = హనుమా; నికామం = శ్వేచ్ఛగా ; వినామేషు = విమాన సమానమైన గృహాలలో (శోధించడం); నిషణ్ణ సమన్వితః = దు:ఖిం కలిగి ; కామరూప ధృత్ =కోరిన రూపం; విచచార = అన్వేషించి; పున: = మళ్ళీ; లంకాం= లంకా నగరం; లాఘవేన= గొప్ప వేగం;
అణిమాది సిద్ధులను కలిగి కామరూపమును దాల్చగల ఆ హనుమంతుడు విమానములందు సీతామాతను మిక్కిలి అన్వేషించియూ చూడలేక దు:ఖించి , మహావేగముతో మరల లంకయందు తిరుగసాగెను – 6.1

आससादाथ लक्ष्मीवान्राक्षसेन्द्रनिवेशनम्
प्राकारेणार्कवर्णेन भास्वरेणाभिसम्वृतम् 		ఆససాద అథ: లక్ష్మీవాన్ రాక్షసేన్ద్ర నివేశనమ్
ప్రాకారేణార్కవర్ణేన భాస్వరేణాభి సంవృతమ్
ఆససాద = చేరుకున్న; అథ: = ఆ తరువాత; లక్ష్మీవాన్ = కాంతిమంతుడైన హనుమ; రాక్షసేన్ద్ర =రావణ; నివేశనమ్ = మందిరమును; ప్రాకార = ప్రాకారము; ఆర్క వర్ణేన= సూర్యుని వర్ణముతో (ఎరుపు రంగు); భాస్వరేణ = మధ్యాహ్న సూర్యరశ్మి వలె; అభిసంవృతమ్= చుట్టియుండెను
అంత ఆ కాంతిమంతుడు రావణమందిరమును చేరెను. సూర్యుని వర్ణముతో వెలుగు ప్రాకారము దానిచుట్టియుండెను - 6.2

रक्षितम् राक्षसैर् भीमैः सिम्हैर् इव महद् वनम्
समीक्षमाणो भवनम् चकाशे कपि कुन्जरः 		రక్షితం రాక్షసైర్ భీమై: సింహైర్ ఇవ మహద్వనమ్
సమీక్షమాణో భవనం చకాశే కపి కుఞ్జరః
రక్షితం = సంరక్షింపబడుచున్న; రాక్షసైర్ =రాక్షసులచే; భీమై: = భయంకరమైన; సింహైర్ = సింహములచే ; మహద్వనమ్ = మహా అడవివలె; సమీక్షమాణో = సమీక్షించిన; భవనం = భవనం; చకాశే = ఆశ్చర్యము; కపి కుఞ్జరః = హనుమ;
సింహములచే అభిరక్షితమగు మహా అడవివలె ఘోరరాక్షసులచే సంరక్షింపబడుచున్న ఆ రావణమందిరమున చూచుచు ఆంజనేయుడు ఆశ్చర్యపోయెను –6.3

रूप्य कोप हितैः चित्रैः तोरणैर् हेम भूषितैः
विचित्राभिः च कक्ष्याभिर् द्वारैः च रुचिरैर् वृतम् 		రూప్య కోపహితై: చిత్రై: తోరణైర్ హేమభూషితైః
విచిత్రాభిశ్చ కక్ష్యాభిర్ ద్వారైశ్చ రుచిరైర్ వృతమ్
రూప్య కోపహితై: = వెండితో అలంకరింపబడి; చిత్రై: = చిత్రములై; తోరణై: = తోరణము; హేమభూషితైః = బంగారుతో అలంకరించబడిన; విచిత్రాభిశ్చ = వింత/విచిత్రము; కక్ష్యాభి: = కక్ష్యలతో /ప్రవేశాలు; ద్వారైశ్చ = ద్వారములతో /తలుపులు; రుచిరైర్ = అందమైన ; వృతమ్= చుట్టుతా కూడియుండెను
వెండితో చేయబడి బంగారముతో అలంకరింపబడి చిత్రములై గోచరించు తోరణములతో, విచిత్రములైన కక్ష్యలతో, మెరిసే ద్వారములతో ఆ మందిరము కూడియుండెను – 6.4

गज आस्थितैर् महा मात्रैः शूरैः च विगत श्रमैः
उपस्थितम् असम्हार्यैर् हयैः स्यन्दन यायिभिः 		గజాస్థితైర్ మహామాత్రైః శూరైశ్చ విగతశ్రమైః
ఉపస్థితమ్ అసంహార్యైర్ హయైః స్యందన యాయిభిః
గజాస్థితై: = ఏనుగుల మీద పురుషులు; మహామాత్రైః = మహామాత్రులచే; శూరై: = యోధులు; విగతశ్రమైః = శ్రమ ఎరుగని; ఉపస్థితమ్ = అక్కడ ఉండెను/ వేచి ఉన్నారు ; అసంహార్యైర్ = అజేయమైన; హయైః = గుర్రాల ద్వారా; స్యందన యాయిభిః = రధములు
గజారూఢులై శౌర్యమును ప్రదర్శించుచు, శ్రమ ఎరుగని మహామాత్రులచే (ఏనుగులను నియంత్రించువారు) ఆ మందిరము కాపాడబడుచుండెను. రధములకు అజేయములైన గుర్రములను అచట నుండెను – 6.5

सिम्ह व्याघ्र तनु त्राणैर् दान्त कान्चन राजतैः
घोषवद्भिर् विचित्रैः च सदा विचरितम् रथैः 		సింహ వ్యాఘ్ర తను త్రాణైర్ దంతకాంచన రాజతైః
ఘోషవద్భిర్ విచిత్రైశ్చ సదా విచరితం రథైః
సింహ = సింహం; వ్యాఘ్ర = పులి; తను త్రాణైర్ = చర్మములచే చేయబడిన కవచములను; దంత = దంతము; కాంచన = స్వర్ణ; రాజతైః = రజతతో (వెండితో) అలంకరించి; ఘోషవద్భిర్ = మహాధ్వని చేయగల ; విచిత్రైశ్చ = వింత; సదా = ఎల్లప్పుడూ; విచరితం = తిరగగల; రథైః =రథాలు
సింహశార్దూల చర్మములచే చేయబడిన కవచములను కలిగి స్వర్ణ రజత దంతములచే నిర్మింపబడి, ఎల్లప్పుడూ మహాధ్వని చేయగల విచిత్ర రధములు అచట ఉండెను – 6.6

बहु रत्न समाकीर्णम् पर अर्ध्य आसन भाजनम्
महा रथ समावासम् महा रथ महा आसनम् 		బహురత్న సమాకీర్ణం పరార్ధ్యా సనభాజనమ్
మహారథ సమావాసం మహారథ మహాసనమ్
బహురత్న = చాలా వజ్రాలతో; సమాకీర్ణం= నింపబడి; పరార్ధ్యా = విలువైన పీఠములు ; సనభాజనమ్ =సామానులు కలిగి; మహారథ = గొప్ప యోధుల ; సమావాసం = నివాసం; మహారథ = గొప్ప రథాలు ; మహాసనమ్ = ధ్వనిచేయుచుండెను
ఆ భవనము నానామణిసంభరితమై మహోదాత్తములైన పీఠములను, సామానులు కలిగి మహారధములతో సంకులమై యుండెను. ఆ రధములు ధ్వనిచేయుచుండెను – 6.7

दृश्यैः च परम उदारैः तैः तैः च मृग पक्षिभिः
विविधैर् बहु साहस्रैः परिपूर्णम् समन्ततः 		దృశ్యైశ్చ పరమోదారై: తై: తై: మృగపక్షిభిః
వివిధై: ర్బహు సాహస్రైః పరిపూర్ణం సమంతత:
దృశ్యైశ్చ = చూడటానికి సుందరమైన; పరమోదారై: = చాలా అందమైన వాటిని ; తై: తై: = పలువిధముల; మృగపక్షిభిః = జంతువులు మరియు పక్షులు; వివిధై: = వివిధ రకాల; బహు సాహస్రైః = బహుసహస్ర (వేలల్లో); పరిపూర్ణం = పరిపూర్ణంగా; సమంతత: = నిండియుండెను
మిక్కిలి ప్రేక్షణీయము, గొప్పవియు, పలువిధముల బహుసహస్ర సంఖ్యాకములైన మృగపక్షులతో ఆ సౌధము అంతా నిండియుండెను - 6.8

विनीतैर् अन्त पालैः च रक्षोभिः च सुरक्षितम्
मुख्याभिः च वर स्त्रीभिः परिपूर्णम् समन्ततः 		వినీతై: అన్త పాలైశ్చ రక్షోభిశ్చ సురక్షితమ్
ముఖ్యాభిశ్చ వరస్త్రీభిః పరిపూర్ణం సమంతత:
వినీతై: = బాగా శిక్షణ పొంది వినయ సంపత్తి కలిగి; అన్త పాలైశ్చ = అంతర్గత ప్రాంతమును కాపాడుట; రక్షోభిశ్చ = రాక్షసులు; సురక్షితమ్ = సురక్షితంగా; ముఖ్యాభిశ్చ = చక్కని ముఖము; వరస్త్రీభిః = ఉత్తమ మహిళలు; పరిపూర్ణం = పరిపూర్ణం; సమంతత:= నిండియుండెను;
సుశిక్షితులై (బాగా శిక్షణ పొంది) వినయ సంపత్తి కలిగి బహిర్భాగ సంరక్షణార్ధము నియమింపబడిన రాక్షసులు, దానిని అప్రమత్తులై కాపాడుచుండిరి. చక్కని సుందరాంగులతో ఆ భవనము నిండియుండెను – 6.9

मुदित प्रमदा रत्नम् राक्षस इन्द्र निवेशनम्
वराभरण संह्रादैः समुद्र स्वनन्निस्वनम् 		ముదిత ప్రమదారత్నం రాక్షసేన్ద్ర నివేశనమ్
వరాభరణ సంహ్రాదైః సముద్ర స్వన నిస్వనం
ముదిత = సంతోషంగా; ప్రమదారత్నం= స్త్రీరత్నములతో; రాక్షసేన్ద్ర = రావణ; నివేశనమ్ =నివాసం; వరాభరణ = ఉత్తమ ఆభరణాల ; సంహ్రాదైః = ధ్వనులు; సముద్ర స్వన = మహాసముద్రం వంటి; నిహస్వనం =ధ్వని కలిగి ఉంది
సంతోషమున ఓలలాడు స్త్రీరత్నములతో నిండి, దానవ నాయకులు కునికిపట్టె స్త్రీపురుషుల ఆభరణధ్వనులు, సముద్రమువలె ఆ భవనము ఘోషించుచుండెను – 6.10

तद् राज गुण सम्पन्नम् मुख्यैः च वर चन्दनैः
महाजनैः समाकीर्णां सिंहैरिव महद्वनम् 		తద్రాజ గుణ సంపన్నం ముఖ్యైశ్చ వరచన్దనైః
మహాజనై: సమాకీర్ణం సింహైరివ మహద్వనం
తత్ రాజ = ఆ రాజు (రావణుని) యొక్క; గుణ సంపన్నం = రాజచిహ్నము; ముఖ్యైశ్చ = శ్రేష్ఠములైన ; వరచన్దనైః = ఉత్తమ సువాసనలు/ అగరు చందన పరిమళములతో; మహా జనై: = పెద్ద సంఖ్యలో ప్రజలు; సమాకీర్ణం = కిక్కిరిసి/ నిండిన; సింహై: ఇవ = సింహాలతో; మహద్వనం = మహారణ్యమువలె

भेरीमृदङ्गाभिरुतम् शङ्खघोषनिनादितम्

महा जन समाकीर्णम् ददर्श स महा कपिः

మహా రత్న= గొప్ప వజ్రాలతో; సమాకీర్ణం = నిండినది; దదర్శ= చూచెను; స మహా కపిః = ఆ హనుమ; దదర్శ = చూచెను; విరాజమానం = (ఇల్లు/సౌధము) విరాజిల్లుచుండెను; వపుషా = రూపంతో; గజాశ్వ = ఏనుగులు, గుర్రములతో ; రథ = రధములతో; సంకులమ్ = నిండిన
రత్నభరితమైన రావణుని మహాభవనమును మారుతి చూచెను. గజతురగ (ఏనుగులు, గుర్రములతో) రధములతో కిక్కిరిసి మిగుల పెద్దదై ఆ సౌధము విరాజిల్లుచుండెను - 6.14

लन्का आभरणम् इति एव सो अमन्यत महा कपिः

ల౦కా = లంకా నగరమునకు ; ఆభరణమ్ = ఆభరణమై; ఇతి = ఉన్నట్లు ; ఏవ = ఆ భవనము; అమన్యత = తలంచెను; స: మహా కపిః= మారుతి; చచార = తిరగసాగెను ; హనుమా౦ = హనుమ; తత్ర = అక్కడ; రావణస్య = రావణుని; సమీపత: = సమీపము నందే;

गृहाद् गृहम् राक्षसानाम् उद्यानानि च वानरः

तथा च इन्द्रजितो वेश्म जगाम हरि यूथपः
