దేవరాజ = ఇంద్రుడు; సేవ్యమాన = సేవించే; పావన = పావనమైన; అంఙ్ఘ్రి = పాదాలు ; పఙ్కజమ్ = తామరపువ్వు; వ్యాల= సర్పము; యజ్ఞసూత్రమ్ = యజ్ఞోపవీతం; ఇన్దుశేఖరం = చంద్రుని తలపై అలంకరించుకుని; కృపాకరమ్= కృపాకరుడు; నారద = నారదుడు; ఆది= ఇతర; యోగి వృన్ద= యోగి జనసమూహం; వన్దితం = స్తుతించడం; దిగంబరమ్ (దిక్ + అంబరం > దిగంబరం) = దిక్కులను అంబరమును (బట్టగా) చేసుకున్నవాడు; కాశికాపురాధినాథ =కాశీ నగరానికి అధినాయకుడైన; కాలభైరవం = శ్రీ కాలభైరవుడికి; భజే =నమస్కారాలు/ భజేహం

దేవతల రాజు ఇంద్రుడు తామరపువ్వులవంటి పావన పాదాలు సేవిస్తాడో; దిక్కులను అంబరముగా (బట్టగా) చేసుకున్న ఆ పరమాత్ముడిని నారదుడు మరియు ఇంతర యోగి బృందాలు స్తుతిస్తుంటారో. సర్పాన్ని యజ్ఞోపవీతముగా ధరించి, చంద్రుడిని తలపై ధరించిన కృపాకరుడై, కాశీ నగరానికి అధినాయకుడైన శ్రీ కాలభైరవకు నమస్కారాలు/భజేహం

భాను =సూర్యుడు ; కోటి = కోటి; భాస్వరం = ప్రకాశం ; భవాబ్ధి =ప్రాపంచిక సముద్రము నుండి ; తారకం = రక్షించు ; పరమ్ =పరమాత్ముడు ; నీలకణ్ఠమ్ = నీలకంఠుడు ; ఈప్సిత = కోరికలు ; అర్థ = సంపద; దాయకం = ఇచ్చువాడు/తీర్చువాడు ; త్రిలోచనమ్ =మూడు నేత్రాలు కలవాడు ; కాల = మరణం ; కాలమ్ = యొక్క మరణం/ కాలము (మరణానికి మించి); అంబుజ = తామర ; అక్షమ్ = కళ్ళు; అక్ష = మూడు కళ్ళు ; శూలమ్ = త్రిశూలం (మూడు శూలాలు); అక్షరమ్ = క్షరముకానిది/నశించనిది ; కాశికాపురాధినాథ =కాశీ నగరానికి అధినాయకుడైన; కాలభైరవం = శ్రీ కాలభైరవుడికి; భజే =నమస్కారాలు/ భజేహం

కోటి సూర్యుల ప్రకాశముతో, ప్రాపంచిక సముద్రము నుండి మమ్మల్ని రక్షించు ఆ పరమాత్ముడు, నీలకంఠుడు, మూడు నేత్రాలు కలవాడు, భక్తుల కోరికలను సంపదలను ఇచ్చువాడు/తీర్చువాడు. కాలము యొక్క ముగింపును శాసించగలరో (పునర్జన్మ చక్రం యొక్క సముద్రాన్ని నిర్మూలించగల శక్తికలిగినవాడు), కమలం లాంటి కళ్ళు కలిగి, త్రిశూలముతో మూడు ప్రపంచాలని నిర్వహించగలడో, క్షరములేనివాడు (నాశనములేనివాడు-అమరుడు). కాశీ నగరానికి అధినాయకుడైన శ్రీ కాలభైరవకు నమస్కారాలు/భజేహం
