ఆదిత్య హృదయం

యుద్దకాండమునందు సప్తోత్తర శతతమ (నూట ఏడవ) సర్గ

ततो युद्धपरिश्रान्तं समरे चिन्तया स्थितम्
रावणं चाग्रतो दृष्ट्वा युद्धाय समुपस्थितम् 		తతో యుద్ధ పరిశ్రాంతం సమరే చింతయా స్థితమ్
రావణం చాగ్రతో దృష్ట్వా యుద్ధాయ సముపస్థితమ్
తతో = అప్పుడు; యుద్ధ పరిశ్రాంతం = పోరాటంలో అలసిపోయిన; సమరే = యుద్ధ రంగంలో ; చింతయా స్థితమ్ = చింతాక్రాంతుడై ఉండెను; రావణం చ = రావణుడు; అగ్రతో = ఎదురుగా; దృష్ట్వా = చూసిన; యుద్ధాయ = యుద్ధం కోసం; సముపస్థితమ్ = సమీపంలో నిలిచెను
యుద్ధము చేసిచేసి మిక్కిలి అలసియున్న శ్రీరాముడు యుద్ధ (సమర) రంగమున చింతాక్రాంతుడై ఉండెను. ఆ తరువాత రావణుడు యుద్ధ సన్నద్ధుడై, శ్రీరాముని ఎదుట నిలిచెను – 1

दैवतैश्च समागम्य द्रष्टुमभ्यागतो रणम्
उपगम्याब्रवीद्राममगस्त्यो भगवान् ऋषिः 		దైవతైశ్చ సమాగమ్య ద్రష్టుమభ్యాగతో రణమ్
ఉపగమ్యాబ్రవీద్రామమ్ అగస్త్యో భగవాన్ ఋషిః
దైవతైశ్చ = దేవతలతో; సమాగమ్య = కలిసి వచ్చిన; ద్రష్టుం = చూచుటకు; అభ్యాగతో = అక్కడికి వచ్చిన; రణమ్ = యుద్ధం; ఉపగమ్య = సమీపించి; అబ్రవీత్ = ఇట్లు పలికెను; రామమ్ = శ్రీరామునితో; అగస్త్యో భగవాన్ ఋషిః = అగస్త్య భగవాన్ ఋషి
యుద్ధమును చూచుటకు దేవతలతో కలిసి అక్కడికి వచ్చిన అగస్త్య భగవాన్ ఋషి, శ్రీరాముని సమీపించి, ఇట్లు పలికెను – 2

राम राम महाबाहो शृणु गुह्यं सनातनम्
येन सर्वानरीन्वत्स समरे विजयिष्यसे 		రామ రామ మహాబాహో శృణు గుహ్యం సనాతనమ్
యేన సర్వానరీన్ వత్స సమరే విజయిష్యసి
రామ రామ = ఓ శ్రీరామా; మహాబాహో = మహానుభావుడవైన; శృణు = వినుము ; గుహ్యం = రహస్యముగా ; సనాతనమ్ = పురాతన కాలం నుండి; యేన సర్వాన్ అరీన్ = శత్రువులందరిపై; వత్స = నాయనా (పుత్ర); సమరే = యుద్ధముననందు; విజయిష్యసి = విజయము సాధించగలవు
సకల ప్రాణుల హృదయములను రంజింపచేయు మహానుభావుడవైన ఓ శ్రీరామా, నాయనా (పుత్ర) పురాతన కాలం నుండి (సనాతనం) రహస్యముగా ఉన్న ఈ స్తోత్రమును గురించి తెలిపెదను, వినుము. దీనిని జపించినచో, ఆ మంత్ర ప్రభావము వలన యుద్ధముననందు (సమరము) నీవు శత్రువులందరిపై విజయము సాధించగలవు– 3

आदित्य हृदयम् पुण्यम् सर्वशत्रु विनाशनम्
जयावहं जपेन्नित्यमक्षय्यं परमं शिवम् 		ఆదిత్య హృదయం పుణ్యం సర్వశత్రు వినాశనమ్
జయావహం జపేన్నిత్యమ్ అక్షయ్యం పరమం శివమ్
ఆదిత్య హృదయం = ఆదిత్య హృదయము అను ఈ స్తోత్రము; పుణ్యం = పరమ పవిత్రమైనది ; సర్వశత్రు వినాశనమ్ = సమస్త శత్రువులను నశింపచేయునది; జయావహం = జయము లభించును; జపేన్నిత్యమ్ = అన్ని సమయాల్లో జపించినచో; అక్షయ్యం = (క్షయముకానిది) శ్రేయస్సునిస్తుంది; పరమం శివమ్ = మహోన్నతమైనది
ఆదిత్య హృదయము అను ఈ స్తోత్రము పరమ పవిత్రమైనది. సమస్త శత్రువులను నశింపచేయునది. సకల శ్రేయస్సులను సమకూర్చునది, మహోన్నతమైన ఈ స్తోత్రము నిత్యము జపించినచో అన్నిటా జయము లభించును – 4

सर्वमङ्गलमङ्गल्यं सर्वपापप्रणाशनम्
चिन्ताशोकप्रशमनमायुर्वर्धनमुत्तमम् 		సర్వమంగళ మాంగళ్యం సర్వ పాప ప్రణాశనమ్
చింతాశోక ప్రశమనమ్ ఆయుర్వర్ధనముత్తమమ్
సర్వమంగళ మాంగళ్యం =సర్వమంగళ ప్రదమైనది; సర్వ పాప ప్రణాశనమ్ = సర్వ పాపాలను నశింపచేయునది; చింతాశోక =సమస్త చింతలను, శోకములను; ప్రశమనమ్ = తొలగించునది ; ఆయు: =ఆయువును (జీవితమును); వర్ధనమ్ = వృద్ధిచేయునది; ఉముత్తమమ్ = అద్భుతమైన ఈ స్తోత్రమును జపించుట ఉత్తమమైనది
పరమ పావనమైనది, సమస్త పాపములను నశింపచేయునది,చింత మరియు శోకములను తొలగించునది, ఆయుష్షును వృద్ధిచేయునది, సర్వ జపములలో శ్రేష్ఠమైన ఈ అద్భుతమైన స్తోత్రమును జపించుట ఉత్తమమైనది – 5

रश्मिमन्तं समुद्यन्तं देवासुरनमस्कृतम्
पूजयस्व विवस्वन्तं भास्करं भुवनेश्वरम् 		రశ్మిమంతం సముద్యంతం దేవాసుర నమస్కృతమ్
పూజయస్వ వివస్వంతం భాస్కరం భువనేశ్వరమ్
రశ్మిమంతం = అనంతమైన బంగారు కిరణములతో శోభిల్లు ; సముద్యంతం = అనుదినము ఉదయ కిరణములతో జగతిని జాగృతి చేయువాడు; దేవ = దేవతలు; అసుర =అసురులు; నమస్కృతమ్ = నమస్కరింతురు (సూర్యునకు); పూజయస్వ = పూజింపుము సూర్యభగవానుడికి; వివస్వంతం = అన్నీ తేజుస్సుల్లకంటే ఉన్నతమైన తేజస్సు కలవాడు; భాస్కరం = ప్రపంచానికి వెలుగును ఇచ్చువాడు (భాస్కరుడు); భువనేశ్వరమ్ = సమస్త భువనములను (లోకములను) నియంత్రించువాడు
అన్నీ తేజుస్సుల్లకంటే ఉన్నతమైన తేజస్సు కలవాడు, సమస్త భువనములను (లోకములను) నియంత్రించువాడు, అనంతమైన బంగారు కిరణములతో శోభిల్లుచు, ప్రపంచానికి వెలుగును ఇచ్చువాడు, అనుదినము తన ఉదయ కిరణములతో జగతిని జాగృతి చేయుచు, దేవ అసురులచే పూజలందుకుంటున్న ఆ సూర్యభవానుడిని పూజింపుము – 6

सर्वदेवात्मको ह्येष तेजस्वी रश्मिभावनः
एष देवासुरगणान् लोकान् पाति गभस्तिभिः 		సర్వదేవాత్మకో హ్యేష తేజస్వీ రశ్మిభావనః
ఏష దేవాసుర గణాన్ లోకాన్ పాతి గభస్తిభిః
సర్వ = సర్వ; దేవాత్మక: = దేవతలకు ఆత్మ స్వరూపుడు; హి ఏష: = అతనే; తేజస్వీ = తేజస్సుచే ; రశ్మిభావనః = శక్తి స్ఫూర్తి ప్రసాదించువాడు; ఏష = అతను; దేవాసుర గణాన్ = దేవ అసుర గణాలకు(సమూహాలకు); లోకాన్ = లోకాలను; పాతి = రక్షించువాడు; గభస్తిభిః = కిరణములతో
సర్వ దేవతలకు ఆత్మ స్వరూపుడు అతనే. తన తేజస్సుచే శక్తి, స్ఫూర్తి ప్రసాదించువాడు. అతను దేవ అసుర గణాలకు, లోకాలను తన కిరణములతో రక్షించువాడు– 7

एष ब्रह्माश्च विष्णुश्च शिवः स्कन्दः प्रजापतिः
महेन्द्रो धनदः कालो यमः सोमो ह्यपांपतिः 		ఏష బ్రహ్మాశ్చ విష్ణుశ్చ శివః స్కందః ప్రజాపతిః
మహేంద్రో ధనదః కాలో యమః సోమో హ్యపాం పతిః
ఏష = అతను; బ్రహ్మా చ = బ్రహ్మ; విష్ణుశ్చ = విష్ణు; శివః = శివుడు ; స్కందః = స్కందుడు (కార్తికేయుడు); ప్రజాపతిః =ప్రజాపతులు (మరీచి, అత్రి, అంగీరసుడు, పౌలస్త్యుడు/పులస్త్యుడు, పులహుడు, క్రతువు , వశిష్ట; దక్ష; భృగు; నారద); మహేంద్రో = దేవతల పాలకుడు ఇంద్రుడు; ధనదః = కుబేరుడు; కాలో = కాల స్వరూపుడు; యమః =యముడు; సోమో =చంద్రుడు; హి అపాం పతిః = వరుణుడు
బ్రహ్మ, విష్ణు, శివుడు, స్కందుడు (కార్తికేయుడు), ప్రజాపతులు (మరీచి, అత్రి, అంగీరసుడు, పౌలస్త్యుడు /పులస్త్యుడు, పులహుడు, క్రతువు , వశిష్ట; దక్ష; భృగు; నారద), దేవతల పాలకుడు ఇంద్రుడు, కుబేరుడు, కాల స్వరూపుడు, యముడు, చంద్రుడు, వరుణుడు మొదలగు వారి స్వరూపములన్నియూ ఇతడే– 8

पितरो वसवः साध्या अश्विनौ मरुतो मनुः
वायुर्वह्निः प्रजाः प्राण ऋतुकर्ता प्रभाकरः 		పితరో వసవః సాధ్యా హ్యశ్వినౌ మరుతో మనుః
వాయుర్వహ్నిః ప్రజాప్రాణః ఋతుకర్తా ప్రభాకరః
పితరో =పితృదేవతలు; వసవః = 8 వసువులు (ఫృథ్వి/ధరా, ఆగ్ని/అనల,వాయు/అనిల, వారుణ/ఆప, ఆదిత్య/ప్రత్యూష,ద్యౌస/ప్రభాస (ఆకాశం), చంద్రమస/ సోమ, నక్షత్రాణి/ధృవ); సాధ్యా = 12 సాధ్యులు (మన, అనుమంత్ర, ప్రాణ, నర, అపాన, వితి, నయ, హయ, హరిన్స, నారాయణ, విభు, ప్రభు) ; హి అశ్వినౌ = అశ్వినీ దేవతలు ; మరుతో = మరుత్తులు; మనుః = మనువు; వాయు: = వాయువు; వహ్నిః = అగ్ని; ప్రజా = ప్రజలు ; ప్రాణః = సమస్త ప్రాణులు ; ఋతుకర్తా = ఆరు ఋతువుల కారకుడు; ప్రభాకరః = ఈ ప్రభాకరుడే
పితృదేవతలు, వసువులు, సాధ్యులు, అశ్వినీ దేవతలు, మరుత్తులు, మనువు, వాయువు, అగ్ని, ప్రజలు, సమస్త ప్రాణులు అన్నియూ ఇతడే. ఆరు ఋతువుల కారకుడు ఈ ప్రభాకరుడే – 9

आदित्यः सविता सूर्यः खगः पूषा गभस्तिमान्
सुवर्णसदृशो भानुर्हिरण्यरेता दिवाकरः 		ఆదిత్యః సవితా సూర్యః ఖగః పూషా గభస్తిమాన్
సువర్ణసదృశో భానుః హిరణ్యరేతా దివాకరః
ఆదిత్యః = అదితి పుత్రుడు ; సవితా = జగత్ సృష్టికి కారకుడు ; సూర్యః = జగతికి విధులను నిర్వర్తించుటకు ప్రేరణ ఇచ్చువాడు ; ఖగః = లోకోపకారమునకు ఆకాశమున సంచరించువాడు ; పూషా = వర్షముల ద్వారా సృష్టిని పోషించువాడు; గభస్తిమాన్ = తన కిరణములచే లోకమునందు ప్రకాశించువాడు; సువర్ణ = బంగారు వన్నెతో ; సదృశో = కనుపించువాడు ; భానుః = ప్రకాశించువాడు ; హిరణ్యరేతా = బ్రహ్మండ ఉత్పత్తికి బీజమైనవాడు ; దివాకరః = చీకటిని తొలగిస్తూ దివా సమయమున జీవులకు ఆకార సాకారములు చేయువాడు (దివాకరుడు)
అదితి పుత్రుడు, జగత్ సృష్టికి కారకుడు జగతికి విధులను నిర్వర్తించుటకు ప్రేరణ ఇచ్చువాడు. లోకోపకారమునకు ఆకాశమున సంచరించువాడు. వర్షముల ద్వారా సృష్టిని పోషించువాడు. బంగారు వన్నెతో తన కిరణములచే లోకమునందు ప్రకాశించువాడు. బ్రహ్మండ ఉత్పత్తికి బీజమైనవాడు. చీకటిని తొలగిస్తూ దివా సమయమున జీవులకు ఆకార సాకారములు చేయువాడు (దివాకరుడు) – 10

हरिदश्वः सहस्रार्चिः सप्तसप्तिर्मरीचिमान्
तिमिरोन्मधनः शम्भुस्त्वष्टा मार्ताण्डम्शुमन् 		హరిదశ్వః సహస్రార్చిః సప్తసప్తిర్మరీచిమాన్
తిమిరోన్మథనః శంభుః త్వష్టా మార్తాండ అంశుమాన్
హరిద్ అశ్వ: = శ్యామలవర్ణము గల అశ్వములు కలవాడు; సహస్రార్చిః =అసంఖ్యకములైన కిరణములు కలవాడు; సప్తసప్తి: =ఒకే రధమునకు ఏడు గుర్రములు కలవాడు; మారీచిమాన్ =తేజోమయ కిరణములు కలవాడు; తిమిరోన్మథనః = చీకట్లను పారద్రోలువాడు; శంభుః = సుఖములను ఆనందమును కలిగించువాడు; త్వష్టా =సర్వ సంహారకుడు ; మార్తాండ =జగతికి (బ్రహ్మాండనికి) చైతన్యమును ఇచ్చువాడు; అంశుమాన్ =ఎల్లప్పుడు తన కిరణములతో ప్రకాశించువాడు
శ్యామలవర్ణము గల అశ్వములు కలవాడు, అసంఖ్యకములైన కిరణములు కలవాడు, ఒకే రధమునకు ఏడు గుర్రములు కలవాడు. తేజోమయ కిరణములు కలవాడు. చీకట్లను పారద్రోలువాడు, సుఖములను ఆనందమును కలిగించువాడు. సర్వ సంహారకుడు. జగతికి (బ్రహ్మాండనికి) చైతన్యమును ఇచ్చువాడు. ఎల్లప్పుడు తన కిరణములతో ప్రకాశించువాడు – 11

हिरण्यगर्भः शिशिरस्तपनोऽहस्करो रविः
अग्निगर्भोऽदितेः पुत्रः शङ्खः शिशिरनाशनः 		హిరణ్యగర్భః శిశిరః తపనో భాస్కరో రవిః
అగ్నిగర్భోஉదితేః పుత్రః శంఖః శిశిరనాశనః
హిరణ్యగర్భః = బ్రహ్మాండాన్ని తన ఉదరమున ధరించిన వాడు; శిశిరః = తాపత్రయములతో/దు:ఖముతో (ఆది భౌతిక, ఆది దైవిక, ఆధ్యాత్మికము) బాధపడు వారికి ఆశ్రయమిచ్చి శాంతిని (చల్లదనాన్ని) కలిగించువాడు ; తపనో = తపింపచేయువాడు ; భాస్కరో = దివ్యమైన వెలుగునిచ్చువాడు; రవిః = సకల లోకములకు స్తుతి/ప్రీతి పాత్రుడు; అగ్నిగర్భ: = దివా (ప్రొద్దున్న) కాలమున అగ్నిని గర్భమున దాల్చినవాడు; అదితేః పుత్రః = అదితి కుమారుడు; శంఖః = సాయం సమయమున తనకుతానుగా శాంతించువాడు; శిశిరనాశనః = మంచును తొలగించువాడు
బ్రహ్మాండాన్ని తన ఉదరమున ధరించిన వాడు. దు:ఖముతో బాధపడు వారికి శాంతిని కలిగించువాడు. దివ్యమైన వెలుగునిచ్చువాడు, తపింపచేయువాడు. సకల లోకములకు స్తుతి/ప్రీతి పాత్రుడు. దివా కాలమున అగ్నిని గర్భమున దాల్చినవాడు. అదితి కుమారుడు, సాయం సమయమున తనకుతానుగా శాంతించువాడు, మంచును తొలగించువాడు – 12

व्योमनाथस्तमोभेदी बुग्यजुः सामपारगः
घनवृष्टिरपां मित्रो विन्ध्यवीथीप्लवंगमः 		వ్యోమనాథస్తమోభేదీ ఋగ్యజుఃసామ పారగః
ఘనవృష్టిరపాం మిత్రో వింధ్యవీథీ ప్లవంగమః
వ్యోమనాథ: =ఆకాశమునకు అధిపతి అయినవాడు; తమోభేదీ =అజ్ఞానమును/చీకటిని ఛేదించువాడు; ఋగ్ యజుః సామ = మూడు వేదములలో ఆరితేరినవాడు (ఋగ్, యజుర్, సామ); పారగః = అలలారుచుండెడివాడు (ప్రొద్దున్న-ఋగ్, మద్యాహ్నము-యజుర్, సాయం-సామ వేదం); ఘనవృష్టి: =ఘనముగా వర్షములు కురుపించువాడు; అపాం మిత్రో = వరుణునకు (నీటికి) మిత్రుడు; వింధ్యవీథీ =వింధ్యగిరి మార్గమున; ప్లవంగమః = అతివేగముగా సంచరించువాడు
ఆకాశమునకు అధిపతి, అజ్ఞానమును/చీకటిని ఛేదించువాడు. మూడు వేదములలో ఆరితేరినవాడు, అలలారుచుండెడివాడు. ఘనముగా వర్షములు కురుపించువాడు, వరుణునకు (నీటికి) మిత్రుడు. వింధ్యగిరి మార్గమున అతివేగముగా సంచరించువాడు – 13

आतपी मण्डली मृत्युः पिङ्गलः सर्वतापनः
कविर्विश्वो महातेजा रक्तः सर्वभवोद्भवः 		ఆతపీ మండలీ మృత్యుః పింగళః సర్వతాపనః
కవిర్విశ్వో మహాతేజా రక్తః సర్వభవోద్భవః

नक्षत्रग्रहताराणामधिपो विश्वभावनः
तेजसामपि तेजस्वी द्वादशात्मन्नमोस्तु ते 		నక్షత్ర గ్రహ తారాణామ్ అధిపో విశ్వభావనః

నమః = నమస్కారములు; పూర్వాయ = తూర్పు; గిరయే = పర్వత; పశ్చిమాయాద్రయే = పశ్చిమ పర్వతం వరకు; నమః = నమస్కారములు; జ్యోతిర్గణానాం = గ్రహములకు, నక్షత్రములకు; పతయే = అధిపతివి; దినాధిపతయే = దివా రాత్రులుకు అధిపతివి ; నమః = నమస్కారములు
స్వామీ నీవు పూర్వ (తూర్పు) గిరియందును, పశ్చిమ గిరియందును విలసిల్లువాడవు. జ్యోతిర్గణములకు, నక్షత్రములకు అధిపతివి. దినమునకు (పగలు, రాత్రి) అధిపతివి. నీకు నమస్కారములు – 16

जयाय जयभद्राय हर्यश्वाय नमो नाम:
नमो नाम: सहस्राम्षो आदित्याय नमो नाम: 		జయాయ జయభద్రాయ హర్యశ్వాయ నమో నమః
నమో నమః సహస్రాంశో ఆదిత్యాయ నమో నమః
జయాయ = విజయం ఇచ్చేవాడవు; జయభద్రాయ = ఉపాసకులకు జయములను శుభములను చేకూర్చువాడవు; హర్యశ్వాయ =శ్యామలవర్ణ ఆశ్వారూఢుడవు ; నమో నమః = పదే పదే నమస్కారములు; నమో నమః = పదే పదే నమస్కారములు; సహస్రాంశో = సహస్ర కిరణములు కలవాడవు; ఆదిత్యాయ = అదితి కుమారుడవు ; నమో నమః = పదే పదే నమస్కారములు
ఉపాసకులకు జయమును అనుగ్రహించువాడవు, జయములను, శుభములను చేకూర్చువాడవు. శ్యామలవర్ణ రథారూఢుడవు, వేలకొలది కిరణములు కలవాడవు, అదితి పుత్రుడవైన సూర్యభగవానుడా నీకు వందనములు/నమస్కారములు – 17

नम उग्राय वीराय सारङ्गाय नमो नमः
नमः पद्मप्रबोधाय प्रचण्डाय नमोऽस्तु ते 		నమ ఉగ్రాయ వీరాయ సారంగాయ నమో నమః
నమః పద్మప్రబోధాయ మార్తాండాయ నమో నమః
నమ = నమస్కారములు; ఉగ్రాయ =ఉగ్రుడవు/భయంకరుడవు; వీరాయ = వీరుడు; సారంగాయ =అమిత వేగముగా ప్రయాణించువాడవు; నమో నమః = పదే పదే నమస్కారములు; నమః = నమస్కారములు; పద్మప్రబోధాయ = (ఉదయ కిరణములతో) పద్మములను వికసింపచేయువాడవు; మార్తాండాయ =తీక్షణమైన తేజము కలవాడవు ; నమో నమః = పదే పదే నమస్కారములు
ఉగ్రుడు, వీరుడు, అమిత వేగముగా ప్రయాణించే సూర్య భగవానునికి నమస్కారములు. ఉదయ కిరణములతో పద్మములను వికసింపచేయువాడవు, మార్తాండుడు (తీక్షణమైన తేజము కలవాడు) అయిన ఆదిత్యునికి పదే పదే నమస్కారములు – 18

ब्रह्मेशानाच्युतेशाय सूर्यायादित्यवर्चसे
भास्वते सर्वभक्षाय रौद्राय वपुषे नमः 		బ్రహ్మేశానాచ్యుతేశాయ సూర్యాయాదిత్య-వర్చసే
భాస్వతే సర్వభక్షాయ రౌద్రాయ వపుషే నమః
బ్రహ్మేశానాచ్యుతేశాయ = బ్రహ్మ, విష్ణు మహేశ్వరుల ప్రతీకవు; సూర్యాయ = సూర్యదేవుడు; ఆదిత్య వర్చసే = దివ్య తేజస్సంపన్నుడవు; భాస్వతే = ప్రకాశించేవాడు ; సర్వభక్షాయ = అన్నిటినీ దాహించేవాడు; రౌద్రాయ వపుషే = తీక్షణమైన రుద్ర రూపము కల; నమః = నమస్కారములు
సృష్టి, స్థితి, లయ కారకులైన బ్రహ్మ, విష్ణు మహేశ్వరుల ప్రతీకవు. వర్చస్సు కలవాడు అయిన ఆ సూర్యునికి నమస్కారములు. ప్రకాశించేవాడు, శక్తిమంతుడు, అన్నిటినీ దాహించేవాడు, తీక్షణమైన రుద్ర రూపము కల ఆదిత్యునికి నమస్కారములు – 19

तमोघ्नाय हिमग्नाय शत्रुघ्नायामितात्मने
कृतघ्नघ्नाय देवाय ज्योतिषां पतये नमः 		తమోఘ్నాయ హిమఘ్నాయ శత్రుఘ్నాయామితాత్మనే
కృతఘ్నఘ్నాయ దేవాయ జ్యోతిషాం పతయే నమః
తమోఘ్నాయ = అంధకారాన్ని తొలగించే వాడు; హిమఘ్నాయ = భయమును తొలగించే వాడు ; శత్రుఘ్నాయా = శత్రు నాశనము చేసేవాడు; అమితాత్మనే = సర్వ వ్యాప్తమైన ఆత్మ స్వరూపుడు; కృతఘ్నఘ్నాయ = కృతఘ్నులను నాశనము చేసేవాడు; దేవాయ = దేవుడు; జ్యోతిషాం పతయే = నక్షత్ర గ్రహ కూటమికి అధిపతి అయిన ఆ సూర్యునికి; నమః = నమస్కారములు
సూర్య భగవానుడు అంధకారాన్ని తొలగించే వాడు, భయమును (మంచు పొర) తొలగించే వాడు, శత్రు నాశనము చేసేవాడు, సర్వ వ్యాప్తమైన ఆత్మ స్వరూపుడు. క్రుతఘ్నులను నాశనము చేసేవాడు, దేవుడు, నక్షత్ర గ్రహ కూటమికి అధిపతి అయిన ఆ సూర్యునికి నమస్కారములు – 20

तप्तचामीकराभाय हरये विश्वकर्मणे
नमस्तमोभिनिघ्नाय रुचये लोकसाक्षिणे 		తప్త చామీకరాభాయ వహ్నయే విశ్వకర్మణే
నమస్తమోஉభి నిఘ్నాయ రుచయే లోకసాక్షిణే
తప్త చామీకరాభాయ = కరిగించిన బంగారము కాంతి కలవాడు; వహ్నయే = అగ్ని రూపుడు; విశ్వకర్మణే = విశ్వ కర్మ; నమ: = నమస్కారములు; తమోభినిఘ్నాయ = అంధకారమును తొలగించేవాడు ; రుచయే = సర్వ జ్ఞాన ప్రకాశకుడు; లోకసాక్షిణే = లోకానికి సాక్షి అయిన సూర్యునికి;
కరిగించిన బంగారము వంటి కాంతి కలవాడవు, అగ్ని రూపుడవు, సర్వ జ్ఞాన ప్రకాశకుడు, విశ్వ కర్మ (సర్వ జగత్కర్తవు), అంధకారమును తొలగించేవాడవు, సర్వ జ్ఞాన ప్రకాశకుడవు, లోకానికి (సర్వ జనుల కర్మలకు) సాక్షి అయిన సూర్యునికి నమస్కారములు – 21

नाशयत्येष वै भूतं तदेव सृजति प्रभुः
पायत्येष तपत्येष वर्षत्येष गभस्तिभिः 		నాశయత్యేష వై భూతం తదేవ సృజతి ప్రభుః
పాయత్యేష తపత్యేష వర్షత్యేష గభస్తిభిః
నాశయతి = నాశనము చేసి; భూతం = అన్ని జీవుల; తదేవ సృజతి = మరల సృష్టించేవాడు; ఏష వై ప్రభుః = ఈ సూర్యుడే ప్రభువు; పాయత్యేష తపత్యేష = వేడిమిచే నీటిని ఆవిరి చేసి; వర్షత్యేష = వర్షరూపములో ; గభస్తిభిః = తన కిరణాల ద్వారా
సమస్త సృష్టిని లయమొనర్చి/నాశనము చేసి మరల సృష్టించేవాడు, నీటిని ఆవిరి చేసి, మరల వర్షరూపములో మనకు ఇచ్చే ఆ గగన మండల అధిపతి అయిన సూర్యునికి నమస్కారములు – 22

एष सुप्तेषु जागर्ति भूतेषु परिनिष्ठितः
एष चैवाग्निहोत्रं च फलं चैवाग्निहोत्रिणाम् 		ఏష సుప్తేషు జాగర్తి భూతేషు పరినిష్ఠితః
ఏష ఏవాగ్నిహోత్రం చ ఫలం చైవాగ్ని హోత్రిణామ్
ఏష = సూర్య భగవానుడు; సుప్తేషు = నిద్రా సమయములో; జాగర్తి = మేలుకొని ఉన్నాడు; భూతేషు = అన్ని జీవుల; పరినిష్ఠితః =అంతర్యామిగా ప్రతిష్టితమై ఉన్నాడు; ఏష = అతను; అగ్నిహోత్రం చ = అగ్ని హోత్రములోని అగ్ని ; ఫలం చ ఏవ = ఫలము తానైయున్నాడు ; అగ్ని హోత్రిణామ్ = ఆ అగ్ని హోత్రము
సూర్య భగవానుడు సుషుప్తావస్థలో (నిద్రా సమయములో) యున్న జీవరాశి హృదయములో మేలుకొని అంతర్యామిగా ప్రతిష్టితమై ఉన్నాడు. అగ్నిహోత్రములోని అగ్ని మరియు ఆ అగ్నిహోత్ర ఫలము (హవిస్సు యొక్క స్వరూపము) తానైయున్నాడు – 23

देवाश्च क्रतवश्चैव क्रतूनां फलमेव च
यानि कृत्यानि लोकेषु सर्वेषु परमप्रभुः 		వేదాశ్చ క్రతవశ్చైవ క్రతూనాం ఫలమేవ చ
యాని కృత్యాని లోకేషు సర్వ ఏష రవిః ప్రభుః
వేదాశ్చ = వేదవేద్యుడు; క్రతవశ్చైవ = అన్నీ క్రతువులకు; క్రతూనాం ఫలమేవ చ = వాటి ఫలము తానెయైన వాడు; యాని కృత్యాని = అన్ని క్రియలకు; లోకేషు సర్వ = ఈ సమస్త జగత్తులో ; ఏష = అతను; రవిః ప్రభుః = కారణభూతుడు, ప్రభువు
సూర్య భగవానుడు వేదవేద్యుడు, క్రతువులు (యజ్ఞ స్వరూపుడు), వాటి ఫలము (యజ్ఞ ఫల స్వరూపుడు) తానెయైన వాడు, ఈ సమస్త జగత్తులో అన్ని క్రియలకు కారణభూతుడు, ప్రభువు – 24

कीर्तयन् पुरुषः कश्चिन्नावसीदति राघव 		ఏన మాపత్సు కృచ్ఛ్రేషు కాంతారేషు భయేషు చ

ఏనమ్ = ఈ సూర్య-దేవుడు; ఆపత్సు = ఆపద సమయములలో ; కృచ్ఛ్రేషు = బాధలు, కష్టములు కలిగిన సమయములో; కాంతారేషు = దిక్కుతోచక యున్నప్పుడు; భయేషు చ = భీతితో/ ప్రాణాంతక కాలంలో యున్నప్పుడు; కీర్తయన్ = పఠించుట వలన; పురుషః = ఏ వ్యక్తి ; కశ్చిన్-న అవశీదతి = నాశనము ఉండదు; రాఘవ = ఓ రాఘవా
ఓ రాఘవా! ఈ స్తోత్రమును ఆపద సమయములలో, బాధలు, కష్టములు కలిగిన సమయములో, దిక్కుతోచక యున్నప్పుడు, భీతితో యున్నప్పుడు పఠించుట వలన ధైర్యము, స్థైర్యము కలుగును – 25

पूजयस्वैनमेकाग्रो देवदेवं जगत्पतिम्
एतत्त्रिगुणितं जप्त्वा युद्धेषु विजयिष्यसि 		పూజయస్వైన మేకాగ్రో దేవదేవం జగత్పతిమ్
ఏతత్ త్రిగుణితం జప్త్వా యుద్ధేషు విజయిష్యసి
పూజయస్వ = పూజింపుము; ఏనమ్ = ఆ సూర్యభగవానుని; ఏకాగ్రో = ఏకాగ్రతతో; దేవదేవం = దేవదేవుడు; జగత్పతిమ్ = జగత్పతీయుడైన; ఏతత్ = ఈ ఆదిత్య హృదయమును; త్రిగుణితం = మూడు సార్లు; జప్త్వా = జపించినచో; యుద్ధేషు = మహాసంగ్రామము నందు; విజయిష్యసి = విజయము పొందగలవు
దేవదేవుడు జగత్పతీయుడైన ఆ సూర్యభగవానుని ఏకాగ్రతతో పూజింపుము. ఈ ఆదిత్య హృదయమును ముమ్మారు(మూడు సార్లు) జపించినచో నీవు ఈ మహాసంగ్రామము నందు విజయము పొందగలవు – 26

अस्मिन् क्षणे महाबाहो रावणं त्वं वधिष्यसि
एवमुक्त्वा ततोऽगस्त्यो जगाम स यथागतम् 		అస్మిన్ క్షణే మహాబాహో రావణం త్వం వధిష్యసి
ఏవముక్త్వా తదాగస్త్యో జగామ చ యథాగతమ్
అస్మిన్ క్షణే = ఈ క్షణంలో; మహాబాహో =మహాబాహుడవైన; రావణం = రావణుని; త్వం = నీవు (ఓ శ్రీరామా); వధిష్యసి =వధింపగలవు; ఏవమ్ ఉముక్త్వా = అని పలికి ; తథ = ఆపై; అగస్త్యో = అగస్త్య మహర్షి; జగామ చ = తన స్థానమునకు చేరెను ; యథాగతమ్ = అతను వచ్చిన విధంగానే
మహాబాహుడవైన ఓ శ్రీరామా నీవు ఈ క్షణముననే రావణుని వధింపగలవు అని పలికి అగస్త్య మహర్షి, అతను వచ్చిన విధంగానే తన స్థానమునకు చేరెను – 27

एतच्छ्रुत्वा महातेजा नष्टशोकोऽभवत्तदा
धारयामास सुप्रीतो राघवः प्रयतात्मवान् 		ఏతచ్ఛ్రుత్వా మహాతేజాః నష్టశోకోஉభవత్-తదా
ధారయామాస సుప్రీతో రాఘవః ప్రయతాత్మవాన్
ఏతత్ శ్రుత్వా = ఇది విన్న తరువాత; మహాతేజాః = మహాతేజస్వి అయిన; నష్టశోక: అభవత్ = చింతా రహితుడయ్యెను; తదా =అప్పుడు; ధారయామాస = గ్రహించి; సుప్రీతో = మిక్కిలి సంతృప్తి పొందెను; రాఘవః = శ్రీరాముడు; ప్రయతాత్మవాన్ = మనస్సునందు నిలుపుకొనెను
మహాతేజస్వి అయిన శ్రీరాముడు అగస్త్య మహాముని ద్వారా ఆదిత్య హృదయ మంత్రమును గ్రహించి/తెలుసుకుని ఏకాగ్రతతో ఆ మంత్రమును తన మనస్సునందు నిలుపుకొనెను., చింతా రహితుడయ్యెను (లేనివాడయ్యెను). శ్రీరాముడు మిక్కిలి సంతృప్తి పొందెను – 28

आदित्यं प्रेक्ष्य जप्त्वा तु परं हर्षमवाप्तवान्
त्रिराचम्य शुचिर्भूत्वा धनुरादाय वीर्यवान् 		ఆదిత్యం ప్రేక్ష్య జప్త్వాతు పరం హర్షమవాప్తవాన్
త్రిరాచమ్య శుచిర్భూత్వా ధనురాదాయ వీర్యవాన్
ఆదిత్యం = ఆ సూర్యభగవానుని; ప్రేక్ష్య = ఏకాగ్రతతో చూచుచు; జప్త్వా = ఈ మంత్రమును జపించిన; తు = అతను; పరం = పరమ; హర్షమ్ అవాప్తవాన్ = సంతుష్టుడాయెను; త్రిరాచమ్య = మూడుమార్లు ఆచనము చేసి; శుచి: = శుచిగా; భూత్వా = ఉండి; ధనుర్ = ధనుర్బాణములు; ఆదాయ =ధరించెను/తీసుకొనెను ; వీర్యవాన్ = వీర్యవంతుడై
మూడుమార్లు శుచిగా ఆ సూర్యభగవానుని ఏకాగ్రతతో చూచుచు ఈ మంత్రమును జపించి, పరమ సంతుష్టుడాయెను. ఆ తరువాత వీర్యవంతుడై ధనుర్బాణములు ధరించెను – 29

रावणं प्रेक्ष्य हृष्टात्मा युद्धार्थं समुपागमत्
सर्वयत्नेन महता वधे तस्य धृतोऽभवत् 		రావణం ప్రేక్ష్య హృష్టాత్మా యుద్ధాయ సముపాగమత్
సర్వయత్నేన మహతా వధే తస్య ధృతోஉభవత్
రావణం = రావణుని ; ప్రేక్ష్య = చూచి ; హృష్టాత్మా = మిగుల సంతుష్టుడైన ; యుద్ధాయ =యుద్ధమున ; సముపా గమత్ = పురోగమించెను/ సన్నద్ధుడయ్యెను; సర్వయత్నేన = అన్ని విధములుగా ; మహతా = గట్టి పూనికతో ; వధే = వధించుటకు ; తస్య = ఆ నిశాచరుని ; ధృతో అభవత్ = కృతనిశ్చయుడయ్యెను
మిగుల సంతుష్టుడైన శ్రీరాముడు రావణుని చూచి యుద్ధమునకై పురోగమిచెను. అన్ని విధములుగా గట్టి పూనికతో ఆ నిశాచరుని వధించుటకు కృతనిశ్చయుడయ్యెను – 30

अथ रविरवदन्निरीक्ष्य रामं
मुदितमनाः परमं प्रहृष्यमाणः
निशिचरपतिसंक्षयं विदित्वा
सुरगणमध्यगतो वचस्त्वरेति 		అథ రవిరవదన్ నిరీక్ష్య రామం
ముదితమనాః పరమం ప్రహృష్యమాణః
నిశిచరపతి సంక్షయం విదిత్వా
సురగణ మధ్యగతో వచస్త్వరేతి
అథ = ఆ తరువాత ; రవి: = సూర్యభగవానుడు; వదన్ = తనవైపు; నిరీక్ష్య = చూస్తున్న; రామం = శ్రీరాముని; ముదితమనాః = ఆనందకరమైన మనసుతో; పరమం = పరమ; ప్రహృష్యమాణః = సంతోషముతో; నిశిచరపతి = రావణుడు; సంక్షయం = నశించుట తధ్యము; విదిత్వా = తెలుసుకుని; సురగణ మధ్యగతో = దేవతల మధ్యన ఉన్న సూర్యడు; వచ: ఇతి = అని పలికెను; త్వర = త్వరపడుము ఓ రామా
ఆ తరువాత దేవతా సమూహముతో యుద్ధము తిలకించుచున్న సూర్యభగవానుడు "రావణుడు (నిశాచరులకు అధిపతి) నశించుట తధ్యము" అని తెలుసుకుని, తనవైపు చూస్తున్న శ్రీరామునిపై మనోల్లాసము పొందినవాడై, పరమ సంతోషముతో "ఓ రామా! త్వరపడుము" అని పలికెను – 31
