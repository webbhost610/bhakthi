లలితా సహస్రనామ తాత్పర్యం

సింధూరారుణ విగ్రహాం త్రినయనాం మాణిక్య మౌళి స్ఫురత్
తారానాయక శేఖరాం స్మితముఖీ మాపీన వక్షోరుహామ్
పాణిభ్యా మలిపూర్ణ రత్న చషకం రక్తోత్పలం బిభ్రతీం
సౌమ్యాం రత్నఘటస్థ రక్త చరణాం ధ్యాయేత్పరామంబికామ్
సింధూరారుణ విగ్రహాం = సింధూరము-ఎరుపు రంగు శరీరము; త్రినయనాం = మూడు నేత్రములుు; మాణిక్య మౌళి స్ఫురత్ = మాణిక్యాలతో నిండిన అందమైన కిరీటము ధరించిన ; తారానాయక శేఖరాం = నక్షత్రాలకు నాయకుడైన చంద్రునితో అలకరించిన (నెలవంక చంద్రునితో) ; స్మితముఖీ = కరుణను సూచించే అందమైన నవ్వుగల ముఖము; మాపీన వక్షోరుహామ్ = అందమైన అవయవాలు (వక్షోజాలు); పాణిభ్యామ్ = చేతులలో; అలి = తేనెతో; పూర్ణ = నిండిన; రత్న చషకం = మాణిక్యాలతో; రక్తోత్పలం బిభ్రతీం = ఎరుపు రంగు పువ్వు ; సౌమ్యాం రత్న = మాణిక్యాలతో కూడిన ; ఘటస్థ = కుండ ; రక్త = ఎరుపు రంగు; చరణాం = పాదాన్ని ఈ కుండ మీద ఉంచుతోంది ; ధ్యాయేత్ = ఆ రూపాన్ని ధ్యానిస్తూ ; పరామ్ అంబికామ్ = పరాత్పర రూపమైన తల్లిని ;
సింధూరము-ఎరుపు రంగు శరీరము కలిగి, మూడు కళ్ళు (జ్ఞాన నేత్రము), మాణిక్యాలతో నిండిన అందమైన కిరీటాన్ని అర్ధచంద్రాకారంతో అలంకరించబడి, కరుణను సూచించే అందమైన నవ్వు ముఖం కలిగి, అందమైన అవయవ సౌష్టవము, ఒక చేతిలో తేనెతో మరియు ఆభరణాలతో నిండిన బంగారు పాత్రను కలిగి, మరియు మరొక చేతిలో ఎర్రటి తామర పువ్వుతో ఉన్న ఆ పరాత్పర రూపమైన తల్లిని ధ్యానం చేద్దాం.

అరుణాం కరుణా తరంగితాక్షీం
ధృతపాశాంకుశ పుష్పబాణచాపామ్
అణిమాదిభిరావృతాం మయూఖైః
అహమిత్యేవ విభావయే భవానీమ్
అరుణాం = ఉదయించే సూర్యుడిలా ఎరుపు రంగులో ఉంది ; కరుణా = కరుణ ; తరంగితాక్షీం = కళ్ళ నుండి వెలువడే కాంతి కిరణాలు/ తరంగాలు ; ధృత = కలిగి/ పట్టుకుని ; పాశ = పాశము ; అంకుశ = అంకుశము ; పుష్ప = పుష్పము ; బాణ = బాణము ; చాపామ్ = ధనస్సు ; అణిమాదిభి: = 8 శక్తులు (అణిమా, మహిమా, గరిమా, లఘిమా, ప్రాప్తిః, ప్రాకామ్య, ఈశత్వ, వశిత్వ ) ; ఆవృతాం = అవృతమై ; = ; మయూఖైః = కాంతి కిరణం/కాంతి పుంజం; అహమ్ = నేను ; ఇత్యేవ = ఇలా ; విభావయే = గొప్ప ఆనంద స్థితిని ; భవానీమ్ = భవానీ మాతా ;
భవానీ మాతా అని పిలువబడే ఆమె రూపాన్ని ధ్యానిస్తున్నాను. ఆ తల్లి రంగు తెల్లవారుజామున సూర్యుడిలా (ఎరుపు రంగు) ఉంది, ఆమె కళ్ళ నుండి కాంతి కిరణాలు వెలువడుతున్నాయి ( ఆమె కరుణ, కళ్ళ నుండి సముద్రపు తరంగాల వలె వస్తుంది) . చేతుల్లో పాశము, అంకుశము, చెరకుతో చేసిన విల్లు, పువ్వులతో చేసిన బాణాలను పట్టుకుంది. ఆమె చుట్టూ అణిమా మరియు ఇతర శక్తులు ఆవృతమై/చుట్టిముట్టి ఉన్నాయి.

ధ్యాయేత్ పద్మాసనస్థాం వికసితవదనాం పద్మ పత్రాయతాక్షీం
హేమాభాం పీతవస్త్రాం కరకలిత లసద్ధేమపద్మాం వరాంగీమ్
సర్వాలంకారయుక్తాం సకలమభయదాం భక్తనమ్రాం భవానీం
శ్రీ విద్యాం శాంతమూర్తిం సకల సురనుతాం సర్వసంపత్ప్రదాత్రీమ్
ధ్యాయేత్ పద్మాసనస్థాం = తామర పువ్వుపై పద్మాసనంలో కూర్చున్న ; వికసిత = చంద్రునివలె వికసించిన ; వదనాం = ముఖము ; పద్మ = తామర పువ్వు/ కమలము ; పత్రాయ = పత్రము/రేకు; తాక్షీం = కళ్ళు ; హేమాభాం = బంగారు రంగు ; పీతవస్త్రాం = బంగారు వస్త్రము ; కరకలిత = చేతిలో పట్టుకొని ; లసద్ = మెరుస్తూ ; హేమ = బంగారు ; పద్మాం = కమలం ; వర = అందమైన ; అంగీమ్ = శరీర భాగాలు ; సర్వాలంకారయుక్తాం = అన్ని రకాల ఆభరణాలతో అలంకరించడి ; సతతమ్ /సకలమ్ = ఎల్లవేళలా ; అభయదాం = అభయమిస్తూ ; భక్త నమ్రాం = భక్తులు నమస్కరించే ; భవానీం = భవానీ ; శ్రీ విద్యాం శాంతమూర్తిం = శాంతి మరియు జ్ఞానం యొక్క స్వరూపం ; సకల సుర = దేవతలందరు ; నుతాం = పూజిస్తారు ; సర్వసంపత్ = అన్ని సంపదలు ; ప్రదాత్రీమ్ = ప్రసాదించే ;
తామర పువ్వుపై పద్మాసనంలో కూర్చుని, చంద్రునివలె వికసించిన ముఖము కలిగి, తామర రేకుల వంటి కళ్ళతో, బంగారు రంగు, బంగారు వస్త్రము ధరించి, చేతిలో బంగారు తామర పువ్వులు పట్టుకుని, ఆమె ముందు నమస్కరించే భక్తులకు ఎల్లవేళలా అభయమిస్తూ, దేవతలచే ప్రశంసించబడి, కోరుకున్న ప్రతి సంపదను ప్రసాదించే శాంతి, జ్ఞానం (విద్యా) యొక్క స్వరూపమైన ఆ తల్లిని ధ్యానిస్తున్నాను.

సకుంకుమ విలేపన మళికచుంబి కస్తూరికాం
సమంద హసితేక్షణాం సశరచాప పాశాంకుశామ్
అశేష జనమోహినీ మరుణమాల్య భూషోజ్జ్వలాం
జపాకుసుమ భాసురాం జపవిధౌ స్మరేదంబికామ్
సకుంకుమ =కుంకుమతో ; విలేపనమ్ = అభిషేకం/ లేపనము పూసి ; అళిక = నుదుటిపై ; చుంబి = అలకరించబడి ; కస్తూరికాం = సింధూరం; సమంద హసిత ఈక్షణాం = చక్కటి నవ్వుతున్న కళ్ళు ; సశర =బాణము ; చాప = ధనస్సు; పాశ = పాశము; అంకుశామ్ = అంకుశము; అశేష జన మోహినీమ్ = అశేష భక్తకోటిని ఆకర్షించే ; అరుణ = ఎర్రటి ; మాల్య =మాలను ; భూషోజ్జ్వలాం = ప్రకాశించే ఆభరణాలు; జపా కుసుమ = ఎర్ర మందార పువ్వు; భాసురాం = అలంకరించబడింది ; జపవిధౌ = జపవిధానుసారం ; స్మరేత్ అంబికామ్ = అమ్మని ధ్యానిస్తున్నాను ;
నేను జపవిధానుసారం తల్లిని ధ్యానిస్తున్నాను. ఆమె కుంకుమ మరియు సింధూరాలతో నుదుటిపై అభిషేకం చేయబడుతుంది. ఆమె కళ్ళు, భక్తులను చిరునవ్వుతో చూస్తున్నాయి, ఆమె చేతుల్లో బాణం, విల్లు, పాశము మరియు అంకుశము ఉన్నవి. అశేష భక్తకోటిని ఆకర్షిస్తుంది. ఆమె ఎర్ర మందార పువ్వు మరియు ఆభరణాలతో అలంకరించబడింది ప్రకాశిస్తున్నది
