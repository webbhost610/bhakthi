तपस्स्वाध्यायनिरतं तपस्वी वाग्विदां वरम्

నారదం పరిపప్రచ్ఛ వాల్మీకిర్ముని పుంగవమ్
తపః స్వాధ్యాయ = వేదాధ్యాయన; నిరతం = నిరతుడు; తపస్వీ = తపశ్శీలుడును; వాగ్విదాం వరమ్ = వాక్చతురులలో అగ్రుడు; నారదం = నారదుని; పరిపప్రచ్ఛ = ప్రశ్నించెను; వాల్మీకి: ముని = వాల్మీకి మహర్షి; పుంగవమ్ = గొప్ప తపస్వి
తపశ్శీలుడును, వేదాధ్యాయన నిరతుడు, వాక్చతురులలో అగ్రుడు అయిన నారద మునిని, గొప్ప తపస్వి అయిన వాల్మీకి మహర్షి ఇట్లు ప్రశ్నించెను – 1

कोन्वस्मिन्साम्प्रतं लोके गुणवान्कश्च वीर्यवान्
धर्मज्ञश्च कृतज्ञश्च सत्यवाक्यो दृढव्रत: 		కోన్వస్మిన్ సాంప్రతం లోకే గుణవాన్ కశ్చ వీర్యవాన్
ధర్మజ్ఞశ్చ కృతజ్ఞశ్చ సత్య వాక్యో దృఢవ్రతః
క: ను =నిజానికి; అస్మిన్ లోకే = ఈ భూమండలములో; సాంప్రతం = ఇప్పుడు; గుణవాన్ = సకల సద్గుణ సంపన్నుడును ; క: చ = ఎక్కడ ఉన్నాడు; వీర్యవాన్ = ఎట్టి విపత్కర పరిస్థితులలోను తొణకని వాడు ; ధర్మజ్ఞశ్చ = సామాన్య విశేష ధర్మములు తెలిసినవాడు; కృతజ్ఞశ్చ = శరణాగత వత్సలుడు; సత్య వాక్యో = ఎట్టి క్లిష్టపరిష్ఠితులలో కూడా చెప్పిన మాట తప్పనివాడు; దృఢవ్రతః = నిశ్చలమైన సంకల్పముగల
ఓ మహర్షీ సకల సద్గుణ సంపన్నుడును, ఎట్టి విపత్కర పరిస్థితులలోను తొణకని వాడు, సామాన్య విశేష ధర్మములు తెలిసినవాడు, శరణాగత వత్సలుడు ఎట్టి క్లిష్టపరిష్ఠితులలో కూడా చెప్పిన మాట తప్పనివాడు, నిశ్చలమైన సంకల్పముగల పురుషుని ఈ భూమండలములో నిజానికి ఇప్పుడు ఎక్కడ ఉన్నాడు – 2

चारित्रेण च को युक्तस्सर्वभूतेषु को हित:

విద్వాన్ కః కస్సమర్థశ్చ కశ్చైక ప్రియ దర్శనః
చారిత్రేణ చ = సదాచార సంపన్నుడు; క: = ఎవరు; యుక్తః = యుక్తము తెలిసిన మహానుభావుడు ; సర్వ భూతేషు హితః = సక ప్రాణులకు హితము కూర్చువాడు; కః = ఎవరు; విద్వాన్ = సకల శాస్త్రములు ఎరిగినవాడు; కః = ఎవరు; సమర్థశ్చ = సర్వకార్య ధురంధరుడు ; కః = ఎవరు; కః = ఎవరు; ప్రియ దర్శనః = తన దర్శనముచే సంతోషాన్ని ఇచ్చువాడు
సదాచార సంపన్నుడు, సకల ప్రాణులకు హితము కూర్చువాడు, సకల శాస్త్రములు ఎరిగినవాడు, సర్వకార్య ధురంధరుడు, తన దర్శనముచే సంతోషాన్ని ఇచ్చువాడు, యుక్తము తెలిసిన మహానుభావుడు ఎవరు ? – 3

आत्मवान्को जितक्रोधो तिमान्कोऽनसूयक:
कस्य बिभ्यति देवाश्च जातरोषस्य संयुगे 		ఆత్మవాన్ కో జిత క్రోధో ద్యుతిమాన్ కోనసూయకః
కస్య బిభ్యతి దేవాశ్చ జాత రోషస్య సంయుగే
ఆత్మవాన్ = స్వీయ నియంత్రణ/ ధైర్యశాలి; కః = ఎవరు; జిత క్రోధో = క్రోధము జయించినవాడు; ద్యుతిమాన్ = శోభతో విలసిల్లువాడు ; కః = ఎవరు; అనసూయకః = అసూయలేనివాడు; కస్య = ఎవరికి; బిభ్యతి = భయ కంపితము చేయువాడు; దేవాశ్చ = దేవాసురులను; జాత రోషస్య = కోపముతో; సంయుగే = రణరంగమున
ధైర్యశాలి, క్రోధము జయించినవాడు, శోభతో విలసిల్లువాడు, అసూయలేనివాడు, రణరంగమున కోపముతో దేవాసురులను భయకంపితము చేయువాడు ఎవరు – 4

एतदिच्छाम्यहं श्रोतुं परं कौतूहलं हि मे
महर्षे त्वं समर्थोऽसि ज्ञातुमेवंविधं नरम् 		ఏతత్ ఇచ్ఛామ్యహం శ్రోతుం పరం కౌతూహలం హి మే
మహర్షే త్వం సమర్థోసి జ్ఞాతుమేవం విధం నరమ్
ఏతత్ = ఈ విషయములను గురించి ; ఇచ్ఛామి = తెలుసుకోవాలని; అహం =నేను; శ్రోతుం = వినడానికి; పరం = మిక్కిలి ; కౌతూహలం హి = కుతూహలముతో ; మే = నాకు; మహర్షే = ఓ మహర్షీ ; త్వం = మీరు ; సమర్థో అసి = సమర్ధులు ; జ్ఞాతుమ్ = తెలుపగల; ఏవం విధం = ఇట్టి; నరమ్ = మహాపురుషుని గురించి
ఈ విషయములను గురించి తెలుసుకోవాలని నేను మిక్కిలి కుతూహలముతో ఉన్నాను. ఓ మహర్షీ! మీరు సర్వజ్ఞులు, ఇట్టి మహాపురుషుని గురించి తెలుపగల సమర్ధులు మీరే – 5

श्रुत्वा चैतत् त्रिलोकज्ञो वाल्मीकेर्नारदो वच:
श्रूयतामिति चामन्त्त्र्य प्रहृष्टो वाक्यमब्रवीत् 		శ్రుత్వా చైతత్ త్రిలోకజ్ఞో వాల్మీకే: నారదో వచః
శ్రూయతామ్ ఇతి చామన్త్ర్య ప్రహృష్టో వాక్యమ్ అబ్రవీత్
శ్రుత్వా చ = మాటలను విని; త్రిలోకజ్ఞో = త్రిలోకజ్ఞుడైన; వాల్మీకే: = వాల్మీకి; నారదో = నారదముని ; ఏతత్ వచః = ఈ పదాలు; శ్రూయతామ్ ఇతి చ = ఓ మహర్షీ! వినుము ; ఆమన్త్ర్య =ఆహ్వానించి కూర్చుండబెట్టి; ప్రహృష్టో = ఎంతో సంతోషించెను ; వాక్యమ్ అబ్రవీత్ = ఇట్లు పలికెను
త్రిలోకజ్ఞుడైన నారదముని వాల్మీకి మాటలను విని, ఎంతో సంతోషించెను. పిమ్మట ఆహ్వానించి కూర్చుండబెట్టి అతడు ఓ మహర్షీ! వినుము అని వాల్మికితో ఇట్లు పలికెను – 6

बहवो दुर्लभाश्चैव ये त्वया कीर्तिता गुणा:
मुने वक्ष्याम्यहं बुद्ध्वा तैर्युक्तश्श्रूयतान्नर: 		బహవో దుర్లభాశ్చైవ యే త్వయా కీర్తితా గుణాః
మునే వక్ష్యామ్యహం బుద్ధ్వా తైర్యుక్తః శ్రూయతాం నరః
బహవో = బహు విధములైన; దుర్లభాశ్చైవ = అసాధ్యము; త్వయా = నీవు; కీర్తితా = ప్రస్తుతించిన ; యే గుణాః = అట్టి ఉదాత్త గుణములు; మునే = ఓ మునీ! ; వక్ష్యామి = తెలిపెదను; అహం = నేను; బుద్ధ్వా తై: యుక్తః = అట్టి గుణములున్న; శ్రూయతాం = వినుము; నరః = ఉత్తమ పురుషుని
ఓ మునీ! నీవు ప్రస్తుతించిన బహు విధములైన ఉదాత్త గుణములు ఒక్కరియందే ఉండుట అసాధ్యము. నేను స్మృతి తెచ్చుకుని, అట్టి గుణములున్న ఉత్తమ పురుషుని గురించి తెలిపెదను వినుము – 7

इक्ष्वाकुवंशप्रभवो रामो नाम जनैश्श्रुत:
नियतात्मा महावीर्यो द्युतिमान्धृतिमान् वशी 		ఇక్ష్వాకు వంశ ప్రభవో రామో నామ జనైః శ్రుతః
నియతాత్మా మహావీర్యో ద్యుతిమాన్ ధృతిమాన్ వశీ
ఇక్ష్వాకు వంశ ప్రభవో = ఇక్ష్వాకు వంశములో అవతరించి; రామో నామ = శ్రీ రాముడిగా (ఇతరులను ఆనంద పరిచేవాడు); జనైః శ్రుతః = జగత్ప్రసిద్ధుడాయెను; నియతాత్మా = మనోనిగ్రహము గలవాడు ; మహావీర్యో = పరాక్ర మవంతుడు ; ద్యుతిమాన్ = మహాతేజస్వీ; ధృతిమాన్ = ధైర్యశాలి; వశీ = జితేంద్రియుడు (నియంత్రణ గలవాడు)
ఇక్ష్వాకువంశము మిక్కిలి ప్రసిద్ధిగాంచినది. లోకోత్తర పురుషుడైన శ్రీరాముడు ఆ వంశములో అవతరించి, జగత్ప్రసిద్ధుడాయెను అతడు మనోనిగ్రహము గలవాడు, పరాక్రమవంతుడు, మహాతేజస్వీ, ధైర్యశాలి, జితేంద్రియుడు – 8

बुद्धिमान्नीतिमान्वाग्मी श्रीमान् शत्रुनिबर्हण:

సుగ్రీవుడు శ్రీరాముడు చెప్పిన వివరములన్నిటిని వినెను. శ్రీరామునకు సహాయము చేయుటవలన తనకు సహాయము/ప్రయోజనము కలుగునని భావించి, అతడు శ్రీరామునితో అగ్నిసాక్షిగా మైత్రి చేసెను – 60

పిమ్మట సుగ్రీవుడు దు:ఖితుడై ఉండుట గమనించి శ్రీరాముడు దు:ఖానికి కారణమేమి అని సుగ్రీవుని అడిగెను. అప్పుడు సుగ్రీవుడు తనపై మొదట వాలికి గల ప్రేమ విశ్వాసములు, పిదప ఏర్పడిన వైరమును, అంతా స్నేహభావముతో శ్రీరామునకు మిక్కిలి దు:ఖముతో విపులముగా తెలిపెను – 61

प्रतिज्ञातं च रामेण तदा वालिवधं प्रति
वालिनश्च बलं तत्र कथयामास वानर: 		ప్రతిజ్ఞాతం చ రామేణ తదా వాలి వధం ప్రతి
వాలినశ్చ బలం తత్ర కథయామాస వానరః
ప్రతిజ్ఞాతం చ = ప్రతిజ్ఞ చేసెను ; రామేణ = శ్రీరాముడు ; తదా = అనంతరము; వాలి వధం ప్రతి = వాలిని వధింతును; వాలినశ్చ = వాలియొక్క; బలం తత్ర = బలపరాక్రమములను గురించి ; కథయామాస = వివరించెను; వానరః = సుగ్రీవుడు
అనంతరము శ్రీరాముడు "వాలిని వధింతును" అని ప్రతిజ్ఞ చేసెను. పిమ్మట సుగ్రీవుడు వాలియొక్క అసాధారణ బలపరాక్రమములను గురించి శ్రీరామునికి వివరించెను – 62

सुग्रीवश्शङ्कितश्चासीन्नित्यं वीर्येण राघवे
राघवप्रत्ययार्थं तु दुन्दुभे: कायमुत्तमम्
दर्शयामास सुग्रीवो महापर्वतसन्निभम् 		సుగ్రీవః శంకితశ్చాసీన్ నిత్యం వీర్యేణ రాఘవే
రాఘవః ప్రత్యయార్థం తు దున్దుభేః కాయ ముత్తమమ్
దర్శయామాస సుగ్రీవో మహాపర్వత సన్నిభం
సుగ్రీవః = సుగ్రీవునికి ; శంకిత: చ ఆసీన్ = సందేహము మెదలుచుండెను ; నిత్యం = ఎల్లప్పుడూ ; వీర్యేణ = పరాక్రమ విషయమున; రాఘవే = శ్రీరామునికి గల; రాఘవః ప్రత్యయార్థం తు = శ్రీరాముని బలపరాక్రములను తెలుసు కొనుటకు; దున్దుభేః = దుందుభి అను రాక్షసుని; కాయ ముత్తమమ్ =కళేబరము; దర్శయామాస = చూపించెను; సుగ్రీవో =సుగ్రీవుడు; మహాపర్వత సన్నిభం = మహా పర్వత సదృశమైన
వాలిని హతమార్చుటకు శ్రీరామునికి గల పరాక్రమ విషయమున సుగ్రీవునికి సందేహము మెదలుచుండెను. శ్రీరాముని బలపరాక్రములను తెలుసుకొనుటకు వాలిచే హతుడైన "దుందుభి" అను రాక్షసుని మహా పర్వత సదృశమైన కళేబరమును చూపించెను – 63 & 64

బంగారు పింగళవర్ణము కలవాడు, కపిశ్రేష్టుడు అయిన సుగ్రీవుడు బిగ్గరగా గర్జించెను. ఆ మహానాదమును విని, వానర ప్రభువైన వాలి తన గృహము నుండి బయటకి వచ్చెను – 68

ततस्सुग्रीववचनाद्धत्वा वालिनमाहवे
सुग्रीवमेव तद्राज्ये राघव: प्रत्यपादयत् 		తతః సుగ్రీవ వచనాత్ హత్వా వాలినమాహవే

దిశః ప్రస్థాపయామాస దిదృక్షు: జనకాత్మజామ్

అనంతరం, వానరప్రభువైన సుగ్రీవుడు వివిధ పరదేశములనుండి వానరులందరిని రప్పించి, సీతాస్వేషణకై వారిని నలుదిశలకు పంపెను – 71

ततो गृध्रस्य वचनात्सम्पातेर्हनुमान्बली
शतयोजनविस्तीर्णं पुप्लुवे लवणार्णवम् 		తతో గృధ్రస్య వచనాత్ సంపాతేః హనుమాన్ బలీ
శత యోజన విస్తీర్ణం పుప్లువే లవణార్ణవమ్
తత: = పిమ్మట; గృధ్రస్య = గృధ్రరాజైన ; వచనాత్ = సూచన మేరకు; సంపాతేః = సంపాతి; హనుమాన్ = హనుమ; బలీ = మహాబల సంపన్నుడైన; శత యోజన = నూరు యోజముల ; విస్తీర్ణం = విస్తీర్ణము గల; పుప్లువే = దాటెను; లవణార్ణవమ్ = లవణ సముద్రమును
పిమ్మట మహాబల సంపన్నుడైన హనుమంతుడు గృధ్రరాజైన సంపాతి సూచన మేరకు నూరు యోజముల విస్తీర్ణము గల లవణ సముద్రమును ఒక్క గంతులో దాటెను – 72

तत्र लङ्कां समासाद्य पुरीं रावणपालिताम्
ददर्श सीतां ध्यायन्तीमशोकवनिकां गताम् 		తత్ర లఙ్కాం సమాసాద్య పురీం రావణ పాలితామ్
దదర్శ సీతాం ధ్యాయన్తీమ్ అశోక వనికాం గతామ్
తత్ర = అంతట; లఙ్కాం పురీం = లంకకు; సమాసాద్య = చేరెను; రావణ పాలితామ్ = రావణునిచే పాలించ బడుచున్న; దదర్శ = చూచెను ; సీతాం = జానికీమాతను; ధ్యాయన్తీమ్ = రామధ్యానమున నిమగ్నమైయున్న; అశోక వనికాం = అశోక వనమును; గతామ్ = చేరి
అంతట హనుమ రావణునిచే పాలించబడుచున్న లంకకు చేరెను. క్రమముగా సీతాదేవి కొరకు వెదకుచు అతడు అశోక వనమును చేరి, అచట రామధ్యానమున నిమగ్నమైయున్న జానికీమాతను చూచెను – 73

निवेदयित्वाऽऽभिज्ञानं प्रवृत्तिं च निवेद्य च
समाश्वास्य च वैदेहीं मर्दयामास तोरणम् 		నివేదయిత్వా అభిజ్ఞానం ప్రవృత్తిం చ నివేద్య చ
సమాశ్వాస్య చ వైదేహీం మర్దయామాస తోరణమ్
నివేదయిత్వా = సమర్పించెను; అభిజ్ఞానం = రామనామాంకితమైన ఉంగరమును; ప్రవృత్తిం చ = రామసుగ్రీవుల మైత్రిని; నివేద్య చ = తెలిపి; సమాశ్వాస్య చ = ఓదార్చెను; వైదేహీం =సీతాదేవిని; మర్దయామాస తోరణమ్ = అశోకవనమును ధ్వంసము చేసెను
పిదప హనుమ సీతాదేవికి రామసుగ్రీవుల మైత్రిని తెలిపి, రామనామాంకితమైన ఉంగరమును ఆమెకు ఆనవాలుగా సమర్పించెను. ఆమెకొరకు శ్రీరాముడు పరితపించుచున్న తీరును వివరించి ఆమెను ఓదార్చెను. పిమ్మట అశోకవనమును ధ్వంసము చేసెను – 74

पञ्च सेनाग्रगान्हत्वा सप्तमन्त्रिसुतानपि
शूरमक्षं च निष्पिष्य ग्रहणं समुपागमत् 		పఞ్చ సేనాగ్రగాన్ హత్వా సప్త మన్త్రి సుతానపి
శూర మక్షం చ నిష్పిష్య గ్రహణం సముపాగమత్

अस्त्रेणोन्मुक्तमात्मानं ज्ञात्वा पैतामहाद्वरात्
मर्षयन्राक्षसान्वीरो यन्त्रिणस्तान्यदृच्छया 		అస్త్రేణోన్ముక్త మాత్మానం జ్ఞాత్వా పైతామహాత్ వరాత్
మర్షయన్ రాక్షసాన్ వీరో యన్త్రిణస్తాన్ యదృచ్ఛయా
అస్త్రేణ = బ్రహ్మాస్త్రము నుండి; జ్ఞాత్వా ఉన్ముక్తమ్ = విముక్తుడైనట్లు తెలుసుకొనెను; అత్మానం = తాను; పైతామహాత్ = బ్రహ్మ వర; వరాత్ = ప్రభావమున; మర్షయన్ = పెట్టు బాధలను భరించెను; తాన్ రాక్షసాన్ = ఆ రాక్షసులు ; వీరో = పరాక్రమశైలి అయిన హనుమ; యన్త్రిణ: = బ్రహ్మాస్త్రమునకు; యదృచ్ఛయా = బద్ధుడైనట్లు నటించుచు
నిరుపమాన పరాక్రమశైలి అయిన హనుమ బ్రహ్మ వర ప్రభావమున బ్రహ్మాస్త్రము నుండి అప్రయత్నముగా విముక్తుడైనట్లు తెలుసుకొనెను. అయినప్పటికీ రామకార్యము సాధించుటకై బ్రహ్మాస్త్రమునకు బద్ధుడైనట్లు నటించుచు, ఆ రాక్షసులు పెట్టు బాధలను భరించెను – 76

ततो दग्ध्वा पुरीं लङ्कामृते सीतां च मैथिलीम्
रामाय प्रियमाख्यातुं पुनरायान्महाकपि: 		తతో దగ్ధ్వా పురీం లఙ్కామ్ ఋతే సీతాం చ మైథిలీమ్
రామాయ ప్రియమాఖ్యాతుం పునరాయాన్ మహాకపిః
తత: = తర్వాత; దగ్ధ్వా పురీం లఙ్కామ్ = లంక అంతా నిప్పుపెట్టి; ఋతే = స్థలము తప్ప/ వదిలి; సీతాం చ మైథిలీమ్ = సీతామాత; రామాయ = శ్రీరాముని; ప్రియమామ్ ఆఖ్యాతుం = కుశల వార్తను తెలుపుటకై; పున: = మరలి; ఆరాయాన్ = వచ్చెను; మహాకపిః = హనుమ
సీతామాత ఉన్న స్థలము తప్ప, హనుమ తన తోకతో లంక అంతా నిప్పుపెట్టి, సీతామాత కుశల వార్తను తెలుపుటకై అతి శీఘ్రముగా శ్రీరాముని వద్దకు మరలి వచ్చెను – 77

सोऽधिगम्य महात्मानं कृत्वा रामं प्रदक्षिणम्

ఇదం పవిత్రం = ఇది పవిత్రమైనది; పాపఘ్నం = పాపాలను నాశనం చేస్తుంది ; పుణ్యం = పుణ్య సాధనం; వేదైశ్చ సమ్మితమ్ = వేదాలకు సమానం; యః = ఎవరు; పఠేత్ =చదువుతారో; రామ చరితం = ఈ శ్రీరామ చరితాన్ని; సర్వ పాపైః = అన్ని పాపాల నుండి ; ప్రముచ్యతే = విముక్తి పొందుతారు
ఈ పవిత్రమైన శ్రీరామ చరితమ్ పవిత్రమైనది, పాపాలను నాశనం చేస్తుంది, పుణ్య సాధనం, వేదాలకు సమానం. దీనిని ఎవరు చదువుతారో అన్ని పాపాల నుండి విముక్తి పొందుతారు – 98

सपुत्रपौत्रस्सगण: प्रेत्य स्वर्गे महीयते 		ఏతత్ ఆఖ్యానమ్ ఆయుష్యం పఠన్ రామాయణం నరః
స పుత్ర పౌత్రః స గణః ప్రేత్య స్వర్గే మహీయతే
ఏతత్ రామాయణం = ఈ రామాయణము; ఆఖ్యానమ్ = నిజమైన అర్ధం; ఆయుష్యం = అయుష్యాభివృద్ధి ; పఠన్ = పఠించిన వారికి; నరః = పురుషులు; స పుత్ర పౌత్రః = పుత్ర-పౌత్రులకు ; స గణః = పరివారానికి ; ప్రేత్య = అంత్య కాలమున; స్వర్గే మహీయతే = మోక్షప్రాప్తి కలుగును
ఈ రామాయణము పఠించిన వారికి అయుష్యాభివృద్ధి కలుగును, పుత్ర-పౌత్రులకు, పరివారానికి క్షేమలాభములు ప్రాప్తించును. మరియు అంత్య కాలమున మోక్షప్రాప్తి కలుగును – 99

जनश्च शूद्रोऽपि महत्वमीयात् 		పఠన్ ద్విజో వాగృషభత్వమ్ ఈయాత్
స్యాత్ క్షత్రియో భూమి పతిత్వమ్ ఈయాత్
వణిగ్జనః పణ్య ఫలత్వమ్ ఈయాత్
జనశ్చ శూద్రోపి మహత్త్వమ్ ఈయాత్
పఠన్ జనశ్చ = పఠించిన జనులు; ద్విజో = ద్విజులు; వాగృషభత్వమ్ ఈయాత్ = వేదవేదాంగముల యందు, శాస్త్రముల యందు పండితులు అగుదురు; స్యాత్ క్షత్రియో = క్షత్రియులు; భూమి పతిత్వమ్ = రాజ్యాధికారమును; ఈయాత్ = పొందుదురు; వణిగ్జనః = వైశ్యులకు ; పణ్య ఫలత్వమ్ = వ్యాపారలాభములు ; ఈయాత్ = కలుగును; జనశ్చ శూద్రోపి = శూద్రులు ; మహత్త్వమ్ = తోటివారిలో శ్రేష్ఠులు; ఈయాత్ = అగుదురు
ఈ రామాయణము పఠించిన ద్విజులు వేదవేదాంగముల యందు, శాస్త్రముల యందు పండితులు అగుదురు. క్షత్రియులు రాజ్యాధికారమును పొందుదురు. వైశ్యులకు వ్యాపారలాభములు కలుగును, శూద్రులు తోటివారిలో శ్రేష్ఠులు అగుదురు. ఈ రామాయణము పఠించిన వారు, వినిన వారు పొందే ఫలము అనంతము, అద్వితీయము – 100
